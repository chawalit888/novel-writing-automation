## Font Pairing Recommendations

### Mafia Romance / NC Dark
```
Title: Kanit Bold + Playfair Display
Tagline: Sarabun Light
```

### Sweet Romance
```
Title: Prompt Medium + Great Vibes
Tagline: Sarabun Light Italic
```

### Fantasy
```
Title: Cinzel Decorative + Pridi
Tagline: Prompt Light
```

### BL/GL
```
Title: Prompt SemiBold + Montserrat
Tagline: Sarabun Light
```

### Horror/Thriller
```
Title: Bebas Neue + Kanit Bold
```

## Text Effects

### Shadow (for dark backgrounds)
```css
/* Light */
text-shadow: 2px 2px 4px rgba(0,0,0,0.3);

/* Medium */
text-shadow: 3px 3px 6px rgba(0,0,0,0.5);

/* Heavy */
text-shadow: 4px 4px 8px rgba(0,0,0,0.7);
```

### Glow (for emphasis)
```css
/* Gold glow */
text-shadow: 0 0 10px rgba(212,175,55,0.5);

/* White glow */
text-shadow: 0 0 10px rgba(255,255,255,0.3);
```

### Outline (for contrast)
```css
/* Black outline */
-webkit-text-stroke: 1px #000000;
```

## Tips

1. **อย่าใช้เกิน 2-3 fonts** ในปกเดียว
2. **Thai + English** ควรเลือก font ที่ style คล้ายกัน
3. **Title ต้องอ่านง่าย** แม้จะดูจาก thumbnail
4. **ใช้ contrast สูง** ระหว่าง text และ background
5. **Test บน mobile** เพราะคนส่วนใหญ่อ่านบนมือถือ
