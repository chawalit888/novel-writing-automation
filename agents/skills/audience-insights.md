# Audience Insights Skill

## Overview
Skill สำหรับวิเคราะห์พฤติกรรมและความต้องการของกลุ่มผู้อ่านนิยาย

---

## กลุ่มผู้อ่านหลัก

### 1. วัยรุ่น (13-17 ปี)

**ลักษณะ:**
- นักเรียน ม.ต้น - ม.ปลาย
- มีเวลาอ่านมาก
- งบประมาณจำกัด (ชอบอ่านฟรี)
- ใช้มือถือเป็นหลัก

**แนวที่ชอบ:**
| อันดับ | แนว | ทำไมชอบ |
|--------|-----|---------|
| 1 | วาย/BL | Romance + Fantasy + ความโรแมนติก |
| 2 | รักวัยรุ่น | เข้าถึงง่าย relate ได้ |
| 3 | แฟนตาซี | หนีความจำเจ |
| 4 | Isekai | ฝันอยากเป็นพระเอก/นางเอก |

**แพลตฟอร์มที่ใช้:**
- Dek-D (หลัก)
- Wattpad
- Joylada

**พฤติกรรมการอ่าน:**
- อ่านก่อนนอน
- อ่านตอนเบรคเรียน
- ชอบตอนสั้นๆ (10-15 นาที)
- คอมเมนต์และ interact กับนักเขียน

---

### 2. วัยมหาลัย (18-24 ปี)

**ลักษณะ:**
- นักศึกษา / เพิ่งเริ่มทำงาน
- มีเงินบ้าง ยินดีจ่ายถ้าชอบมาก
- ใช้ทั้งมือถือและคอมพิวเตอร์
- ติดตาม Social Media

**แนวที่ชอบ:**
| อันดับ | แนว | ทำไมชอบ |
|--------|-----|---------|
| 1 | โรแมนซ์ผู้ใหญ่ | เริ่มสนใจความสัมพันธ์จริงจัง |
| 2 | Office Romance | relate กับชีวิตการทำงาน |
| 3 | วาย/ยูริ | ยังคงชอบ + mature content |
| 4 | สยองขวัญ | ความตื่นเต้น |

**แพลตฟอร์มที่ใช้:**
- Readawrite
- Dek-D
- MEB

**พฤติกรรมการอ่าน:**
- อ่านช่วงเย็น-ดึก
- ชอบตอนยาวขึ้น (20-30 นาที)
- ซื้อนิยายที่ชอบมาก
- รีวิวและแนะนำเพื่อน

---

### 3. วัยทำงาน (25-34 ปี)

**ลักษณะ:**
- พนักงานออฟฟิศ / อาชีพต่างๆ
- มีกำลังซื้อ ยินดีจ่ายเพื่อคุณภาพ
- เวลาจำกัด ต้องการความคุ้มค่า
- ใช้ทั้งมือถือและ E-reader

**แนวที่ชอบ:**
| อันดับ | แนว | ทำไมชอบ |
|--------|-----|---------|
| 1 | CEO Romance | Fantasy เกี่ยวกับความสำเร็จ + รัก |
| 2 | โรแมนซ์ผู้ใหญ่ | เนื้อหาเข้มข้น mature |
| 3 | ลึกลับ/สืบสวน | ท้าทายสติปัญญา |
| 4 | แฟนตาซี | หนีจากความเครียด |

**แพลตฟอร์มที่ใช้:**
- Readawrite
- MEB
- หนังสือเล่ม

**พฤติกรรมการอ่าน:**
- อ่านช่วงพักเที่ยง
- อ่านก่อนนอน
- ซื้อ E-book / หนังสือเล่ม
- อ่านจบเป็นเรื่องๆ ไม่ชอบรอ

---

### 4. วัยกลางคน+ (35+ ปี)

**ลักษณะ:**
- มีครอบครัว / อาชีพมั่นคง
- มีกำลังซื้อสูง
- เวลาจำกัดมาก
- ชอบความสะดวก

**แนวที่ชอบ:**
| อันดับ | แนว | ทำไมชอบ |
|--------|-----|---------|
| 1 | โรแมนซ์ | ย้อนวัย ความโรแมนติก |
| 2 | ครอบครัว/ชีวิต | relate กับชีวิตจริง |
| 3 | ประวัติศาสตร์ | ความรู้ + เรื่องเล่า |
| 4 | ลึกลับ | ความบันเทิงที่ท้าทาย |

**แพลตฟอร์มที่ใช้:**
- MEB
- หนังสือเล่ม
- Readawrite

## Reader Personas

### Persona 1: "น้องฝัน" - นักอ่านวัยรุ่น

```
อายุ: 16 ปี (ม.4)
เพศ: หญิง
แพลตฟอร์มหลัก: Dek-D, Joylada
งบประมาณ: ฟรีเป็นหลัก
เวลาอ่าน: 2-3 ชั่วโมง/วัน

ชอบ:
✓ วายหวานๆ
✓ พระเอกเย็นชา
✓ ตัวละครน่ารัก
✓ Happy Ending

ไม่ชอบ:
✗ เนื้อหาหนักเกินไป
✗ ตอนยาวมาก
✗ จบแบบเศร้า
✗ ต้องจ่ายเงิน
```

### Persona 2: "พี่เบล" - สาวออฟฟิศ

```
อายุ: 28 ปี (พนักงานบริษัท)
เพศ: หญิง
แพลตฟอร์มหลัก: Readawrite, MEB
งบประมาณ: 500-1,000 บาท/เดือน
เวลาอ่าน: 1-2 ชั่วโมง/วัน

ชอบ:
✓ CEO Romance
✓ นางเอกเก่ง
✓ เนื้อหาเข้มข้น
✓ นิยายที่จบแล้ว

ไม่ชอบ:
✗ นางเอกอ่อนแอ
✗ Cliffhanger ทุกตอน
✗ รออัพเดทนาน
✗ คุณภาพต่ำ
```

### Persona 3: "คุณแม่พลอย" - แม่บ้านนักอ่าน

```
อายุ: 42 ปี (แม่บ้าน)
เพศ: หญิง
แพลตฟอร์มหลัก: MEB, หนังสือเล่ม
งบประมาณ: 1,000-2,000 บาท/เดือน
เวลาอ่าน: 30 นาที - 1 ชั่วโมง/วัน

ชอบ:
✓ โรแมนซ์คลาสสิก
✓ เรื่องครอบครัว
✓ นิยายที่จบแล้ว
✓ ซื้อเป็นเล่ม

ไม่ชอบ:
✗ เนื้อหา 18+ มากเกินไป
✗ ภาษาวัยรุ่น
✗ อ่านบนมือถือ
✗ นิยายยังไม่จบ
```

### Purchase Behavior

**ปัจจัยที่ทำให้ตัดสินใจซื้อ:**
1. **รีวิวดี** (85%) - คนอื่นแนะนำ
2. **อ่านตัวอย่างแล้วชอบ** (80%)

4. **โปรโมชัน/ลดราคา** (65%)
5. **หน้าปกสวย** (50%)

## Survey Questions Template

**สำหรับวิจัยผู้อ่าน:**

```
1. คุณอ่านนิยายบ่อยแค่ไหน?
   □ ทุกวัน  □ 3-5 ครั้ง/สัปดาห์  □ 1-2 ครั้ง/สัปดาห์  □ นานๆ ครั้ง

2. แนวนิยายที่ชอบที่สุด (เลือกได้ 3)

   (คำตอบเปิด)
```

## Insights for Content Creation

### สิ่งที่ผู้อ่านต้องการ:

**ด้านตัวละคร:**
- พระเอกที่มี Depth (ไม่ใช่แค่หล่อรวย)
- นางเอกที่เก่งและมีเอกลักษณ์
- Character Development ที่สมเหตุสมผล
- Chemistry ที่รู้สึกได้

**ด้านเนื้อเรื่อง:**
- Plot ที่ไม่คาดเดาได้
- Pacing ที่เหมาะสม
- ไม่มี Filler มากเกินไป
- จบแบบ Satisfying

**ด้านการเขียน:**
- ภาษาที่อ่านง่าย
- Dialog ที่สมจริง
- Show don't tell
- ไม่มี Typo / Error
