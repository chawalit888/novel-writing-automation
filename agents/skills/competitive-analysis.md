# Competitive Analysis Skill

## Overview
Skill สำหรับวิเคราะห์คู่แข่งและนิยายที่ประสบความสำเร็จในตลาด

---

## Framework การวิเคราะห์คู่แข่ง

### 1. ระบุคู่แข่ง

**ประเภทคู่แข่ง:**

| ประเภท | คำอธิบาย | ตัวอย่าง |
|--------|----------|----------|
| Direct | นิยายแนวเดียวกัน กลุ่มเป้าหมายเดียวกัน | นิยาย CEO Romance ใน Readawrite |
| Indirect | แนวต่างแต่แย่งเวลาผู้อ่าน | ซีรีส์, Webtoon, เกม |
| Potential | อาจเข้ามาแข่งในอนาคต | นักเขียนดังที่อาจเปลี่ยนแนว |

### 4. SWOT Analysis

**สำหรับแต่ละแนวที่เราจะแข่ง:**

```
┌─────────────────────┬─────────────────────┐
│     Strengths       │     Weaknesses      │
│     (จุดแข็ง)        │     (จุดอ่อน)        │
├─────────────────────┼─────────────────────┤
│ - คุณภาพการเขียน    │ - ยังไม่มีชื่อเสียง  │
│ - ใช้ AI ช่วย       │ - ขาด fanbase       │
│ - Output สูง        │ - ไม่มี track record │
│ - วิเคราะห์ตลาดได้   │ - ...               │
└─────────────────────┴─────────────────────┘
┌─────────────────────┬─────────────────────┐
│    Opportunities    │       Threats       │
│     (โอกาส)         │      (อุปสรรค)       │
├─────────────────────┼─────────────────────┤
│ - ตลาดโต            │ - คู่แข่งเยอะ        │
│ - แนวใหม่ยังมีที่ว่าง │ - Platform เปลี่ยน  │
│ - คนอ่านเพิ่มขึ้น    │ - เทรนด์เปลี่ยนเร็ว  │
│ - ...               │ - ...               │
└─────────────────────┴─────────────────────┘
```

### 5. Gap Analysis

**หาช่องว่างในตลาด:**

| หมวด | สิ่งที่ตลาดมี | สิ่งที่ตลาดขาด | โอกาสสำหรับเรา |
|------|-------------|---------------|----------------|
| แนว | CEO Romance เยอะ | CEO + Fantasy ผสม | ลองทำ CEO ที่มีพลังวิเศษ |
| ตัวละคร | นางเอกอ่อนโยน | นางเอกแกร่งจริงจัง | สร้างนางเอก Strong Female Lead |
| Setting | กรุงเทพฯ | ต่างจังหวัด/ต่างประเทศ | ลอง Setting ใหม่ |
| Format | ตอนยาว | ตอนสั้น อ่านเร็ว | ทำ Bite-sized chapters |

## Success Factors Checklist

**จากการวิเคราะห์นิยายที่ประสบความสำเร็จ:**

### ด้าน Concept
- [ ] มี Hook ที่ชัดเจน
- [ ] มีความแตกต่างจากคู่แข่ง
- [ ] ตรงกับความต้องการของตลาด

### ด้าน Execution
- [ ] คุณภาพการเขียนดี
- [ ] อัพเดทสม่ำเสมอ
- [ ] ไม่มี plot holes

### ด้าน Marketing
- [ ] ชื่อเรื่องจับใจ
- [ ] คำโปรยน่าสนใจ
- [ ] หน้าปกดึงดูด

### ด้าน Engagement
- [ ] ตอบคอมเมนต์ผู้อ่าน
- [ ] สร้าง Community
- [ ] มี Social Media presence
