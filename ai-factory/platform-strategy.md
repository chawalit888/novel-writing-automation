# Platform Strategy - กลยุทธ์แพลตฟอร์ม

## แพลตฟอร์มหลัก: ธัญวลัย (Tunwalai)

### ทำไมเลือกเป็นหลัก
- ผู้อ่าน 500,000+ เรื่อง → ฐานผู้อ่านใหญ่
- ระบบ coin ทำเงินได้จริง
- รับเนื้อหา NC
- คนอ่านพร้อมจ่ายเงินซื้อตอนต่อ
- มี algorithm แนะนำเรื่อง → เรื่องดีได้ push อัตโนมัติ

### กลยุทธ์บน Tunwalai
- ลงทุกเรื่องที่นี่ก่อนเสมอ
- ตอน 1-10 ฟรี / ตอน 11+ คิดเงิน / NC คิดเงินทุกตอน
- ลงสม่ำเสมอวันละ 1 ตอน (algorithm ชอบเรื่องที่อัปเดตบ่อย)
- ใช้แท็กที่ถูกต้อง (แนว, kink, rating) เพื่อเข้าถึงกลุ่มเป้าหมาย

### ข้อควรระวัง
- ปกต้องใช้ภาพ stock photo (ไม่ใช้ AI-generated)
- ใช้ cover-design-team workflow: Image Scout หา stock → Prompt Engineer ปรับ → Layout Designer จัดวาง → Cover QC ตรวจ

## แพลตฟอร์มรอง

### Fictionlog
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ลงอะไร | เรื่อง NC 25+ เข้มข้น |
| จุดเด่น | เฉพาะทาง NC ผู้ใหญ่ คนอ่านยอมจ่ายเงินดี |
| ระบบเงิน | ซื้อตอน (chapter purchase) |
| เวลาลง | หลัง Tunwalai 7-14 วัน |

### ReadAWrite
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ลงอะไร | เรื่อง BL/GL โดยเฉพาะ |
| จุดเด่น | ชุมชน BL/GL แข็งแรง, รับปก AI-generated |
| ระบบเงิน | ระบบ coin |
| เวลาลง | หลัง Tunwalai 7-14 วัน |

### Dek-D (ใช้เป็น Funnel)
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ลงอะไร | ฟรี 5-10 ตอนแรก แล้วลิงก์ไป Tunwalai |
| จุดเด่น | ฐานผู้อ่านใหญ่ที่สุด กว่า 1 ล้านเรื่อง |
| ระบบเงิน | ไม่ใช้ → ใช้เป็นช่องทางดึงคนไป Tunwalai |
| เวลาลง | พร้อมกับ Tunwalai ได้ (เพราะฟรี) |

### MEB Market (ใช้ Bundle เรื่องจบ)
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ลงอะไร | เรื่องจบแล้ว ทำเป็น e-book |
| จุดเด่น | ขายเป็นเล่ม ราคาสูงกว่า |
| ระบบเงิน | ขาย e-book |
| เวลาลง | หลังเรื่องจบครบ 120+ ตอน |

## กลยุทธ์ Cross-posting

### ลำดับการลง

```
Tunwalai (หลัก) → ลงก่อนเสมอ
    ↓ 7-14 วันหลัง
Fictionlog (NC เข้มข้น) / ReadAWrite (BL/GL)
    ↓ พร้อมกัน
Dek-D (ฟรี funnel)
    ↓ หลังเรื่องจบ
MEB Market (e-book bundle)
```

### ตารางตามแนวเรื่อง

| แนวเรื่อง | แพลตฟอร์ม 1 | แพลตฟอร์ม 2 | แพลตฟอร์ม 3 |
|-----------|------------|------------|------------|
| NC Romance (มาเฟีย/CEO) | Tunwalai | Fictionlog | Dek-D (funnel) |
| BL/GL | Tunwalai | ReadAWrite | Dek-D (funnel) |
| Fantasy Romance | Tunwalai | Dek-D (funnel) | - |
| Horror/Mystery + NC | Tunwalai | Fictionlog | - |
| เรื่องจบแล้ว (ทุกแนว) | MEB Market | - | - |

## ข้อมูลจากรายงานตลาด

(อ้างอิง: `รายงาน-10-นิยายอิโรติกยอดฮิตไทย-2025.md`)

### แนว/Kink ที่ตลาดต้องการ
- CEO/มหาเศรษฐี + เลขา/พนักงาน
- มาเฟีย + บังคับรัก + possessive
- ท้อง/พลาดท้อง
- พี่เลี้ยง/ครู + ลูกศิษย์ (ห้ามอายุน้อยกว่า 18)
- ย้อนยุค/พีเรียด
- วงการบันเทิง (ดารา/ไอดอล)
- Enemies to Lovers
- Slow Burn + NC เข้มข้น

### ระดับ NC ตาม Platform

| Platform | ระดับ NC แนะนำ | หมายเหตุ |
|----------|---------------|---------|
| Tunwalai | 7-8/10 | NC ได้ แต่ไม่ Hard NC สุดๆ |
| Fictionlog | 9-10/10 | NC 25+ เต็มที่ |
| ReadAWrite | 8-9/10 | NC ได้ เน้น BL/GL |
| Dek-D | 5-6/10 | ระวัง ไม่ควรเกิน 18+ ทั่วไป |

## เพิ่มช่องทางในอนาคต

เมื่อมีเรื่อง hit แล้ว สามารถขยายไปยัง:
- **LINE OA:** สร้างฐานแฟนโดยตรง broadcast เรื่องใหม่
- **เว็บไซต์ตัวเอง:** novel-promo-site (มีอยู่แล้ว) → ขายตรง ไม่เสียค่า platform
- **Facebook Group:** สร้าง community ผู้อ่าน
- **TikTok/Reels:** Teaser clip ดึงคนอ่าน
