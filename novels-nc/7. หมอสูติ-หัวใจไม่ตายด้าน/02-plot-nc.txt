│                      (ตอนที่ 1-6)                           │
└─────────────────────────────────────────────────────────────┘

[จุดเริ่มต้น]
• กัญญาวีร์ (วีร์) ลูกสาวเจ้าของบริษัทอสังหาฯ มาพบหมอสูติเป็นครั้งแรก
  เพราะประจำเดือนมาไม่ปกติ แม่บังคับให้มาตรวจ
• เจอ ดร.สิรวิชญ์ หมอสูติสุดหล่อ หน้าเย็น มืออาชีพ ตรวจร่างกายเธอ
  ด้วยสายตาที่ไม่มีอารมณ์ใดๆ ทั้งสิ้น - วีร์ช็อกที่หมอหล่อขนาดนี้ไม่มีปฏิกิริยา
• วีร์เริ่มท้าทาย พยายามยั่วยวนหมอตายด้าน แต่หมอเฉยเป็นหิน

[Tension Build-up - Level 1]
• วีร์แกล้งนัดตรวจบ่อยขึ้น อ้างอาการต่างๆ เพื่อมาเจอหมอ
• พยาบาลปุณณ์ (พี่ปุณณ์) เริ่มสังเกตว่าคนไข้คนนี้มาบ่อยผิดปกติ
  และเริ่มหึงหวงหมอที่ตัวเองแอบชอบ
• สิรวิชญ์เริ่มรู้สึก "ผิดปกติ" เวลาตรวจวีร์ ไม่เหมือนคนไข้คนอื่น
  แต่บอกตัวเองว่ามันเป็นไปไม่ได้ เขาเห็นร่างกายมาเป็นร้อย
• ดร.ณัฐภัทร (ณัฐ) หมอหนุ่มรุ่นน้อง เริ่มสนใจวีร์ จีบอย่างเปิดเผย

🔥 ฉาก NC ซ่อนเร้น (ตอนที่ 5):
   • Trigger: ตรวจภายในครั้งที่สาม วีร์ตั้งใจยั่วยวน หายใจหอบ ครางเบาๆ
   • ประเภท: [x] sexual tension / tease
   • ความเข้มข้น: 4/10
   • อารมณ์หลัก: หมอพยายามเฉย แต่มือเริ่มสั่น ร่างกายตอบสนองครั้งแรก

│              องก์ที่ 2: เส้นแบ่งที่เลือนราง                   │
│                      (ตอนที่ 7-12)                          │
└─────────────────────────────────────────────────────────────┘

[หมอ "ตื่น" ครั้งแรก]
• วีร์ไม่สบายกะทันหัน ปวดท้องกลางดึก โทรหาหมอสิรวิชญ์
  หมอมาดูแลที่คอนโดวีร์ (เพราะเป็นคนไข้ VIP)
• อยู่ด้วยกันสองคนในคอนโด วีร์ในชุดนอนบาง
  ร่างกายหมอตอบสนองอย่างควบคุมไม่ได้
• สิรวิชญ์ตระหนักว่าเขา "ตื่น" เป็นครั้งแรกในรอบหลายปี
  ตกใจกับตัวเอง พยายามหนี

[ความสัมพันธ์ข้ามเส้น]
• วีร์รู้ว่าหมอเริ่ม "ตื่น" เพิ่มแรงกดดัน
• พยาบาลปุณณ์แพร่ข่าวลือในโรงพยาบาลว่าคนไข้คนนี้ "จีบหมอ"
• ณัฐเดินหน้าจีบวีร์เต็มตัว ชวนไปดินเนอร์
  สิรวิชญ์เห็นแล้วอารมณ์ปั่นป่วน แต่ยังปฏิเสธว่าไม่ได้รู้สึกอะไร
• วีร์จัดงานการกุศลของบริษัทครอบครัว เชิญหมอไป
  คืนนั้นทั้งคู่เมา... จูบกันครั้งแรก

🔥 ฉาก NC สำคัญ (ตอนที่ 8):
   • Trigger: คอนโดวีร์ ดึก ชุดนอนบาง หมอมาดูแลคนไข้
   • ประเภท: [x] passionate / first awakening
   • ความเข้มข้น: 7/10
   • อารมณ์หลัก: ความตายด้านแตกสลาย หมอสูญเสียการควบคุมครั้งแรก
   • พัฒนาการ: จากหมอที่ "ไม่รู้สึก" กลายเป็นผู้ชายที่กำลังตื่นจากภวังค์

🔥 ฉาก NC ช่วงกลาง (ตอนที่ 12):
   • Trigger: ห้องพักเวรกลางดึก วีร์มาหาหมอที่โรงพยาบาล
   • ประเภท: [x] passionate / forbidden
   • ความเข้มข้น: 7/10
   • อารมณ์หลัก: ผิดกฎจรรยาบรรณ แต่หยุดไม่ได้ จูบแรงในห้องพักเวร

[จุดหักมุม / Paywall Hook (ตอนที่ 10)]
• ผอ.โรงพยาบาลเรียกสิรวิชญ์ไปตักเตือน มีคนร้องเรียนว่า
  มีความสัมพันธ์เกินเลยกับคนไข้
• วีร์รู้ข่าว บุกไปหาหมอที่โรงพยาบาลกลางดึก
• Cliffhanger: สิรวิชญ์พูดว่า "หยุดมาหาผม คุณกัญญาวีร์
  ผมเป็นหมอคุณ... แค่นั้น" แต่มือที่จับข้อมือเธอกลับไม่ยอมปล่อย

[ความสัมพันธ์ลับ]
• สิรวิชญ์โอนวีร์ให้หมอคนอื่นดูแล เพื่อไม่ให้ผิดจรรยาบรรณ
  แต่กลับทำให้ทั้งคู่เจอกันในฐานะ "ผู้ชาย-ผู้หญิง" ได้อย่างเต็มที่
• เริ่มคบกันลับๆ นัดเจอนอกโรงพยาบาล
• ณัฐรู้เรื่อง ท้าทายสิรวิชญ์ ว่าคบคนไข้ผิดจรรยาบรรณ
  ขู่จะรายงานแพทยสภา
• พ่อแม่วีร์ (คุณพ่อสุวิทย์ คุณแม่นภัสสร) รู้เรื่อง
  คุณพ่อโกรธไม่อยากให้ลูกสาวคบหมอธรรมดา
  คุณแม่เข้าใจแต่ห่วง

[วีร์เปิดเผยความลับ]
• วีร์เล่าให้หมอฟังว่าเธอเคยถูกอดีตแฟนทำร้ายจิตใจ
  ดูถูกว่า "อารมณ์ทางเพศสูงเหมือนผู้หญิงไม่ดี"
  ทำให้เธอเกลียดตัวเอง
• สิรวิชญ์ปลอบ ยอมรับเธอทั้งหมด "ไม่มีอะไรผิดปกติกับเธอ"
  คืนนั้นเขายอมจำนนอย่างเต็มตัว

🔥 ฉาก NC เข้มข้น (ตอนที่ 15):
   • Trigger: คืนหลังวีร์เปิดใจ สิรวิชญ์ยอมจำนนต่อความรู้สึก
   • ประเภท: [x] passionate / first time เต็มรูปแบบ
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: อ่อนโยนแต่เร่าร้อน หมอดูแลทุกรายละเอียด
   • จุดเด่น: หมอใช้ความรู้ทางการแพทย์ดูแลเธออย่างดี

🔥 ฉาก NC สุดเดือด (ตอนที่ 18):
   • Trigger: วีร์แกล้งยั่วหมอในห้องตรวจ (ไม่ใช่คนไข้แล้ว)
   • ประเภท: [x] rough / possessive
   • ความเข้มข้น: 9/10
   • อารมณ์หลัก: หมอหลุดจากความตายด้านสิ้นเชิง หึงหวง ครอบครอง
   • จุดเด่น: ในห้องตรวจที่เคยเฉยชา กลายเป็นสถานที่ร้อนแรง

[อุปสรรคถาโถม]
• ณัฐรายงานแพทยสภาเรื่องสิรวิชญ์คบคนไข้ (แม้จะโอนแล้ว)
  สิรวิชญ์ถูกสอบสวน เสี่ยงถูกพักใบอนุญาต
• พยาบาลปุณณ์สมคบกับณัฐ เอาภาพจากกล้องวงจรปิด
  ตอนทั้งคู่จูบกันในห้องพักเวร ไปเผยแพร่
• คุณพ่อสุวิทย์บังคับวีร์ให้ไปดูตัวกับลูกชายหุ้นส่วนธุรกิจ
  ขู่จะตัดมรดก
• สิรวิชญ์ตัดสินใจ "ถอย" เพื่อไม่ให้วีร์เดือดร้อน
  บอกเลิก ทำเป็นเฉยชาเหมือนเดิม

[จุดต่ำสุด]
• วีร์เสียใจ คิดว่าหมอกลับไปตายด้านเหมือนเดิม
  ไม่เคยรักเธอจริง
• สิรวิชญ์ทรมานกับตัวเอง ทำงานหนัก ไม่นอน
  คนไข้สังเกตว่าหมอเปลี่ยนไป
• วีร์ถูกบังคับไปงานดินเนอร์กับลูกชายหุ้นส่วน
  สิรวิชญ์มาเห็น ทนไม่ไหว

🔥 ฉาก NC ช่วงหึงหวง (ตอนที่ 21):
   • Trigger: สิรวิชญ์เห็นวีร์กับผู้ชายอื่น ฉุดเธอออกมา
   • ประเภท: [x] rough / jealous
   • ความเข้มข้น: 9/10
   • อารมณ์หลัก: หมอตายด้านกลายเป็นสัตว์ร้ายหึงหวง

🔥 ฉาก NC ช่วงปลอบใจ (ตอนที่ 24):
   • Trigger: คืนพายุ วีร์ร้องไห้ สิรวิชญ์มาหา
   • ประเภท: [x] tender / comforting
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: อ่อนโยน ดูแล รักจริง ไม่ใช่แค่ต้องการ

[วิกฤตใหญ่]
• คนไข้ของสิรวิชญ์คลอดฉุกเฉิน มีภาวะแทรกซ้อน
  หมอผ่าตัดช่วยชีวิตแม่ลูกได้สำเร็จ ชื่อเสียงกลับมา
• ณัฐพยายามทำลายชื่อเสียงสิรวิชญ์อีกครั้ง
  แต่คราวนี้ถูกจับได้ว่าปลอมแปลงหลักฐาน ถูกพักงาน
• พยาบาลปุณณ์สำนึกผิด สารภาพทุกอย่าง ขอโทษ
• สิรวิชญ์ไปหาคุณพ่อสุวิทย์ ยืนยันว่ารักวีร์จริง
  พร้อมพิสูจน์ตัวเอง ไม่ใช่แค่หมอธรรมดา

[คืนดี]
• วีร์รู้ความจริงทั้งหมด ว่าหมอถอยเพราะปกป้องเธอ
• แม่นภัสสรช่วยพูดให้พ่อยอมรับ
• สิรวิชญ์ไปหาวีร์ที่ริมทะเล (ที่ที่วีร์หนีไปพักใจ)
  ขอโอกาสกลับมาเริ่มต้นใหม่

🔥 ฉาก NC ช่วงหลังวิกฤต (ตอนที่ 27):
   • Trigger: หลังผ่าตัดสำเร็จ วีร์มาเยี่ยมที่ห้องพักหมอ
   • ประเภท: [x] passionate / relief
   • ความเข้มข้น: 7/10
   • อารมณ์หลัก: ดีใจ คิดถึง ปลดปล่อยความกังวล

🔥 ฉาก NC คืนดี (ตอนที่ 30):
   • Trigger: ริมทะเล คืนดี กลับมาหากัน
   • ประเภท: [x] passionate / reunion
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: รักที่กลับมา เต็มเปี่ยม ไม่มีอะไรขวางอีกแล้ว

[ตัวละครพิสูจน์ตัวเอง]
• คุณพ่อสุวิทย์ยอมรับสิรวิชญ์ หลังเห็นความทุ่มเทและฝีมือการแพทย์
• สิรวิชญ์ขอแต่งงานวีร์ที่โรงพยาบาล (สถานที่ที่เจอกันครั้งแรก)
• เตรียมงานแต่งงาน วีร์จัดให้ธีม "โรงพยาบาล" แกล้งหมอ

[ตอนจบ - Happy Ending]
• พิธีแต่งงาน เพื่อนหมอ พยาบาล ครอบครัวทั้งสองฝ่ายมาร่วมยินดี
• คืนแต่งงาน วีร์ใส่ชุดพยาบาลเซ็กซี่แกล้งหมอ
  "คุณหมอคะ... หนูไม่สบาย"
• บทส่งท้าย: 1 ปีต่อมา วีร์ตั้งครรภ์ สิรวิชญ์ดูแลเธอเอง
  "หมอสูติคนนี้... หัวใจไม่ตายด้านอีกต่อไป"

🔥 ฉาก NC ไคลแมกซ์ (ตอนที่ 33, 35, 36):
   • ตอน 33: คืนหลังขอแต่งงาน - หวาน เร่าร้อน (9/10)
   • ตอน 35: คืนแต่งงาน - ชุดพยาบาล สูงสุด! (10/10)
   • ตอน 36: บทส่งท้าย - ตั้งครรภ์ อ่อนโยน (7/10)
