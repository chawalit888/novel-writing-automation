[จุดเริ่มต้น - ตอนที่ 1-3]
• ตอนที่ 1: เลขาคนใหม่
  - พิมพ์มาดา (มายด์) เด็กจบใหม่ แว่นหนา ผมมวย แต่งตัวเรียบร้อย
  - วันแรกที่บริษัท Pinnacle Group ในตำแหน่งเลขาประธาน
  - เลขาคนก่อนลาออกกะทันหัน (ทนนิสัยประธานไม่ไหว)
  - เจอ ธนัท (ท่านประธาน) ครั้งแรก — หล่อจนใจหยุดเต้น แต่เย็นชาจนใจแข็ง
  - ธนัทมองเธอแล้วพูด: "HR ส่งเด็กฝึกงานมาเป็นเลขาผมเหรอ"
  - มายด์กัดฟันทน ตั้งใจพิสูจน์ตัวเอง

• ตอนที่ 2: ทดสอบสาวแว่น
  - ธนัทใช้งานหนัก ทดสอบว่าจะทนได้นานแค่ไหน
  - มายด์ไม่ยอมแพ้ ทำงานเก่งเกินคาด (เกียรตินิยมอันดับ 1)
  - พนักงานคนอื่นซุบซิบ "ไม่ถึงอาทิตย์ก็ลาออก"
  - ธนัทเริ่มสนใจ — สาวคนนี้ทำไมไม่กลัวเขา
  - ฉากเย็น: มายด์หลับคาโต๊ะทำงาน ธนัทเดินผ่านเห็น...แอบถอดแจ็คเก็ตคลุมให้

• ตอนที่ 3: ระยะใกล้อันตราย
  - งานเลี้ยงลูกค้า มายด์ต้องไปด้วย แต่แต่งตัวไม่เป็น
  - ธนัทเห็นเธอในชุดเดรสงานเลี้ยง — ตะลึง (สวยกว่าที่คิด แม้ยังใส่แว่น)
  - งานเลี้ยง: ลูกค้าเมาลวนลามมายด์ ธนัทเข้ามาช่วย — กอดเธอเข้าหาตัว
  - "เธอเป็นคนของผม อย่ายุ่ง" (หมายถึงพนักงาน แต่ฟังเหมือนอ้างเป็นเจ้าของ)
  - ขากลับในรถ: บรรยากาศอึดอัด หัวใจเต้นแรง ทั้งคู่ไม่พูดอะไร

[Tension Build-up - ตอนที่ 4-6]
• ตอนที่ 4: ความลับของสาวแว่น
  - มายด์ทำงานดี ธนัทเริ่มผ่อนปรน
  - ธนัทเรียกมายด์เข้าห้อง ถามเรื่องส่วนตัว ทำไมแต่งตัวแบบนี้
  - มายด์เล่า: "หนูไม่เคยมีแฟนค่ะ ไม่เคยจีบใคร ไม่เคย...อะไรเลย"
  - ธนัทช็อกเล็กน้อย จ้องมอง: "อายุ 23 แล้วนะ"
  - มายด์แดงจนหู: "หนูรู้ค่ะ มันผิดปกติ..."
  - ธนัท (เสียงทุ้ม): "ไม่ได้ผิดปกติ แค่...น่าเสียดาย"
  - สายตาที่จ้องกันนานเกินไป

• ตอนที่ 5: บทเรียนแรก — สอนจูบ 🔥(เบา)
  - หลังงานเลี้ยงอีกครั้ง มายด์ต้องแกล้งเป็นแฟนธนัทเพื่อไล่อดีตคนรัก
  - ธนัทพูด: "เธอจูบเป็นไหม"
  - มายด์: "ไ-ไม่เป็นค่ะ..."
  - ธนัท: "งั้นผมสอนให้"
  - จูบแรก — สั้น อ่อนโยน แต่ทำให้ทั้งคู่สั่น
  - ธนัทผลักตัวเองออก ปรับเนคไท: "พอแค่นี้ก่อน"
  - มายด์กลับบ้านนอนไม่หลับ แตะริมฝีปากตัวเองทั้งคืน
  - [Sexual tension: 4/10]

• ตอนที่ 6: เส้นบางๆ ระหว่างเจ้านายกับอะไรมากกว่า
  - ธนัทเริ่มเอาใจใส่มายด์มากผิดปกติ
  - สั่งชุดทำงานใหม่ให้ ส่งอาหารกลางวัน ขับไปส่งบ้าน
  - พนักงานเริ่มซุบซิบ มายด์อึดอัด
  - มายด์เข้าห้องประธาน: "ท่านประธานค่ะ หยุดทำแบบนี้ได้ไหม คนเขามอง"
  - ธนัท (ยกคิ้ว): "มองก็มอง ผมดูแลลูกน้องของผม"
  - มายด์: "แต่ท่านไม่ได้ดูแลคนอื่นแบบนี้"
  - ธนัทยืนขึ้น เดินเข้ามาใกล้จนจมูกเกือบชน: "เพราะคนอื่นเขาไม่ต้องการผมสอน"
  - มายด์ถอยหลังจนชนผนัง หัวใจจะระเบิด

[ข้ามเส้น - ตอนที่ 7-9]
• ตอนที่ 7: ข้อเสนอที่ปฏิเสธไม่ได้
  - มายด์โดนเพื่อนร่วมงาน (ณัฐพล) ชวนไปเดท แต่เธอไม่มีประสบการณ์
  - ธนัทได้ยิน อารมณ์หงุดหงิดแต่กดไว้
  - กลางคืนทำงานดึกสองคน ธนัทเสนอ:
  - "ถ้าอยากเรียนรู้เรื่องพวกนี้...ผมสอนให้ได้"
  - มายด์ตกใจ: "ท-ท่านประธานพูดจริงเหรอคะ"
  - ธนัท: "เงื่อนไข — ห้ามมีอารมณ์จริง ห้ามบอกใคร เป็นแค่บทเรียน"
  - มายด์ลังเล...แต่ตกลง (เพราะอยากรู้ + แอบชอบเขาอยู่แล้ว)

   • Trigger: ออฟฟิศเปล่า + บรรยากาศใกล้ชิด + สัญญา "บทเรียน" เริ่มต้น

   • รายละเอียด:

     - มายด์สั่นเทา เสียงหอบ ธนัทสะดุดจังหวะ

• ตอนที่ 9: เช้าวันรุ่งขึ้น
  - มายด์มาทำงานหน้าแดง ไม่กล้าสบตา
  - ธนัททำเป็นปกติ แต่สายตาจ้องเธอบ่อยขึ้น
  - มายด์ทำถ้วยกาแฟตกแตก ธนัทรีบมาดูแลแผล — นิ้วเขาสัมผัสมือเธอ

  - ณัฐพลชวนมายด์กินข้าว ธนัทเห็นจากห้องกระจก — กำมือแน่น

  - ธนัทบ้าคลั่ง ตามหามายด์ทุกทาง
  - พบว่ามายด์ไปทำงานบริษัทคู่แข่ง (บริษัทของ ภูวดล — CEO หนุ่มอีกคน)
  - ธนัทบุกไปหา เห็นภูวดลยืนข้างมายด์ — เลือดขึ้นหน้า

🔥 ฉาก NC (ตอนที่ 21): หึงหวง
   • Trigger: เจอมายด์กับ CEO คนใหม่ + ลากมายด์ออกมา + ไม่ยอมสูญเสีย
   • ประเภท: [ ] tender [x] passionate [x] rough
   • ความเข้มข้น: 9/10
   • อารมณ์หลัก: ครอบครอง + หึง + คิดถึง + ไม่ยอมปล่อย
   • รายละเอียด:
     - ธนัทลากมายด์ไปคอนโดเขา: "เธอเป็นของผม ห้ามใครแตะ"
     - "เธอคิดว่าคนอื่นจะดูแลเธอได้เหมือนผมเหรอ"
     - ดิบ ดุ หึง แต่ลึกๆ อ่อนโยน
     - หลังจบ: กอดแน่น "อย่าหนีผมอีก"
     - มายด์ร้องไห้: "หนูไม่ได้อยากไป แต่หนูกลัวท่านเสียทุกอย่าง"

• ตอนที่ 22: เลือกเธอ
  - ธนัทตัดสินใจ: เลือกมายด์ ไม่สนพ่อคัดค้าน
  - ประกาศต่อบอร์ด: "ผมจะคบเธอ ถ้าใครไม่พอใจ ผมลาออกเอง"
  - ปรีชายอม เพราะธนัทเก่งจริงไม่มีใครแทนได้
  - สารินถูกไล่ออก เพราะพฤติกรรมขัดจริยธรรมบริษัท

• ตอนที่ 23: เปิดตัว
  - ธนัทพามายด์ไปงานเลี้ยงในฐานะ "แฟน" อย่างเป็นทางการ
  - มายด์ถอดแว่น ใส่คอนแทคเลนส์ แต่งตัวสวย (Makeover)
  - ทุกคนตกตะลึง: สาวแว่นกลายเป็นสาวสวย
  - ธนัทกระซิบ: "ผมชอบเธอทั้งตอนใส่แว่นและไม่ใส่"

🔥 ฉาก NC (ตอนที่ 24): สาวแว่นเป็นฝ่ายรุก
   • Trigger: มายด์มั่นใจขึ้น + ถึงเวลาที่เธอจะ "สอน" บ้าง
   • ประเภท: [ ] tender [x] passionate [ ] rough
   • ความเข้มข้น: 9/10
   • อารมณ์หลัก: มั่นใจ + กล้า + เธอเป็นคนนำ
   • รายละเอียด:
     - มายด์ใส่แว่นกลับ + ใส่ชุดเลขา: "วันนี้หนูจะเป็นคนสอนบ้าง"
     - ธนัทตะลึง ไม่เคยเห็นเธอแบบนี้
     - มายด์ผลักเขานั่งเก้าอี้ประธาน ล็อคประตูห้อง
     - "ท่านประธานค่ะ วันนี้ห้ามขยับ"
     - ธนัทยิ้มมุมปาก: "ลูกศิษย์โตไวจัง"

• ตอนที่ 25: อดีตของธนัท
  - เปิดเผยว่าธนัทเคยหย่าร้าง ภรรยาเก่าทิ้งเพราะเขาทุ่มเทให้งาน
  - ปมลึก: ธนัทไม่เชื่อว่าตัวเองจะรักใครได้จริง
  - มายด์เริ่มเข้าใจว่าทำไมเขาถึงเสนอ "แค่บทเรียน" ตอนแรก

• ตอนที่ 26: ภูวดลกลับมา
  - ภูวดล (CEO คู่แข่ง) ยังไม่ยอมแพ้ ส่งดอกไม้ให้มายด์
  - ธนัทหึง แต่พยายามกดไว้
  - ภูวดลพูดกับมายด์: "ธนัทรักเธอจริง ผมรู้ แต่ถ้าวันไหนเขาทำเธอเสียใจ ผมรอ"

🔥 ฉาก NC (ตอนที่ 27): คืนดีหลังทะเลาะ
   • Trigger: ทะเลาะเรื่องภูวดล + มายด์โกรธ + ธนัทขอโทษ
   • ประเภท: [x] tender [x] passionate [ ] rough
   • ความเข้มข้น: 8/10
   • รายละเอียด:
     - ธนัทขอโทษครั้งแรกในชีวิต: "ผมเป็นคนเห็นแก่ตัว แต่ผมไม่อยากเสียเธอ"
     - อ่อนโยน ช้า ดูแล ทำให้เธอรู้ว่าเขาเปลี่ยนเพราะเธอ
     - "ผมไม่เคยรักใครแบบนี้ พิมพ์มาดา ไม่เคย"

• ตอนที่ 28: ครอบครัวมายด์
  - มายด์พาธนัทไปเจอแม่ (แม่ค้าขายอาหาร)
  - ธนัทอ่อนน้อม ช่วยขายของ ทำให้แม่มายด์ปลื้ม
  - มายด์เห็นธนัทอีกมุม — ไม่ได้มีแค่ด้าน CEO เย็นชา
  - แม่มายด์กระซิบ: "เขาดีนะลูก แต่ระวังคนรวยหน่อย"

🔥 ฉาก NC (ตอนที่ 29): ห้องประธาน ล็อคประตู
   • Trigger: มายด์กลับมาทำงานที่เดิม + ความสัมพันธ์เปิดเผย + เร่าร้อนกลางวัน
   • ประเภท: [x] passionate [ ] tender [ ] rough
   • ความเข้มข้น: 9/10
   • รายละเอียด:
     - มายด์กลับมาเป็นเลขาธนัทอย่างเป็นทางการ
     - ระหว่างประชุม ธนัทส่งข้อความ: "หลังประชุม ห้ามไปไหน"
     - ล็อคประตูห้องประธาน "เธอรู้ไหมว่าผมคิดถึงเธอแค่ไหนตอนเธอหายไป"

• ตอนที่ 30: สารินกลับมาแก้แค้น
  - สารินร่วมมือกับบริษัทคู่แข่ง วางแผนทำลายธนัท
  - ปล่อยข่าวลือว่ามายด์เป็น "สาวเอาใจเจ้านาย" เพื่อขึ้นตำแหน่ง
  - มายด์ถูก cyberbully หนัก ล้มป่วย

🔥 ฉาก NC (ตอนที่ 31): คืนหลังเกือบเสียเธอ
   • Trigger: มายด์ล้มป่วย + ธนัทดูแลทั้งคืน + กลัวเสียเธอ
   • ประเภท: [x] tender [ ] passionate [x] rough
   • ความเข้มข้น: 9/10
   • รายละเอียด:
     - ธนัทดูแลมายด์ที่ป่วย อยู่ข้างเตียงทั้งคืน
     - มายด์หายดี ธนัทกอดแน่น: "อย่าทำให้ผมกลัวอีก"
     - ดิบ ครอบครอง แต่ลึกๆ คืออ่อนแอ กลัวเสียเธอ

• ตอนที่ 32: จัดการศัตรู
  - ธนัทสืบพบว่าสารินอยู่เบื้องหลังทุกอย่าง
  - ใช้หลักฐานจัดการ สารินถูกดำเนินคดี
  - ภูวดลช่วยเหลือ ยอมรับว่ามายด์เลือกธนัท

🔥 ฉาก NC (ตอนที่ 33): บทเรียนสุดท้าย
   • Trigger: สารินถูกจัดการแล้ว + ทุกอย่างลงตัว + เธอเป็นผู้สอน
   • ประเภท: [x] passionate [x] tender [ ] rough
   • ความเข้มข้น: 10/10
   • รายละเอียด:
     - มายด์: "ท่านประธานค่ะ บทเรียนสุดท้ายแล้วนะคะ"
     - ธนัท: "บทเรียนอะไร"
     - มายด์: "สอนท่านประธานว่า...การรักใครสักคนไม่ใช่เรื่องน่ากลัว"
     - เร่าร้อน อ่อนโยน สมบูรณ์แบบ ทั้งคู่เป็นผู้ให้
     - "ผมรักเธอ พิมพ์มาดา ไม่ใช่บทเรียน ไม่ใช่เกม นี่คือเรื่องจริง"

• ตอนที่ 34: ขอแต่งงาน
  - ธนัทเซอร์ไพรส์ขอแต่งงานกลางออฟฟิศ
  - พนักงานทั้งบริษัทร้องยินดี
  - ปรีชา (พ่อธนัท) ยอมรับมายด์ในที่สุด
  - มายด์ร้องไห้ตอบว่า "ค่ะ"

🔥 ฉาก NC (ตอนที่ 35): คืนแต่งงาน
   • Trigger: คืนแต่งงาน + ชุดเจ้าสาว + ความสุขสมบูรณ์
   • ประเภท: [x] passionate [x] tender [ ] rough
   • ความเข้มข้น: 10/10
   • รายละเอียด:
     - "จำได้ไหม วันแรกที่เธอเป็นเลขาผม ผมบอกว่า HR ส่งเด็กฝึกงานมา"
     - "ตอนนี้เด็กฝึกงานคนนั้น กลายเป็นภรรยาผมแล้ว"
     - หวาน ช้า เต็มไปด้วยรัก ไม่รีบ ดูแล
     - "ขอบคุณที่สอนผมรักอีกครั้ง"

• ตอนที่ 36: บทส่งท้าย — เรียนจบแล้ว
  - 1 ปีต่อมา มายด์ตั้งครรภ์
  - มายด์ยังใส่แว่น ทำงานเป็นผู้จัดการฝ่ายใหม่ (เลื่อนตำแหน่ง)
  - ธนัทกลายเป็นสามีบ้ารัก ดูแลภรรยาท้อง
  - ฉากสุดท้าย: มายด์ใส่แว่น นั่งบนตักธนัทในห้องประธาน
  - "ท่านประธานค่ะ พร้อมเรียนเรื่องเป็นพ่อคนยังคะ"
  - ธนัท (ยิ้ม): "ครูมายด์สอนผมได้ทุกเรื่อง"
  - [ความเข้มข้น: 7/10 — หวาน อ่อนโยน อบอุ่น]
