# NC Translation Specialist Skill

## Overview
Skill เฉพาะสำหรับแปล NC scenes จากภาษาอังกฤษเป็นภาษาไทย โดยรักษา intensity, emotion, และความ natural ของภาษา

---

## 1. Translation Principles

### 1.1 Preserve Intensity
```
- รักษาระดับความ explicit ตามต้นฉบับ
- ไม่ลดทอน ไม่เพิ่มเติม
- ความร้อนแรงต้องเท่าเดิม
```

### 1.2 Natural Thai
```
- ใช้ภาษาไทยที่ลื่นไหล
- ไม่ใช่ภาษาแปล
- เหมือนเขียนเป็นไทยตั้งแต่แรก
```

### 1.3 Cultural Adaptation
```
- ปรับให้เหมาะกับผู้อ่านไทย
- ใช้สำนวนที่คุ้นเคย
- รักษา mood และ tone
```

## 3. Sentence Structure

### 3.1 Short = Intense
```
English: "He kissed her. Hard. Desperate."
Thai: "เขาจูบเธอ. แรง. กระหาย."

หรือ

"เขาจูบเธอ
แรง
เหมือนคนกระหาย"
```

### 3.2 Long = Tender
```
English: "His fingers traced slowly along her spine,
         memorizing every curve."
Thai: "นิ้วเขาลากผ่านแนวกระดูกสันหลังของเธอช้าๆ
       เหมือนต้องการจดจำทุกเส้นโค้ง"
```

### 3.3 Fragment for Impact
```
English: "Skin against skin. Finally."
Thai: "ผิวแนบผิว ในที่สุด"

หรือ

"ผิวกายแนบกัน
ในที่สุด"
```

## 4. Dialogue Translation

### 4.1 Keep It Short
```
❌ "I love you so much and I want you"
❌ "ฉันรักเธอมากและฉันต้องการเธอ"

✅ "I love you... want you..."
✅ "รักนะ...ต้องการ..."
```

### 4.2 Add Thai Particles
```
"Please" → "ได้ไหม" / "นะ"
"Yes" → "ค่ะ/ครับ" / "อืม" / "ช่าย"
"More" → "อีก" / "อีกนะ"
"Don't stop" → "อย่าหยุด" / "อย่าหยุดนะ"
```

### 4.3 Interjections
```
"Oh..." → "อา..." / "โอ้..."
"Mmm..." → "อืม..." / "หืม..."
"Ah..." → "อา..." / "อ๊ะ..."
"God..." → "พระเจ้า..." (formal) / ตัด (natural)
```

## 6. Platform-Specific Language

### 6.1 Dek-D / Joylada (Young readers)
```
Tone: Slightly softer, more romantic
Vocab: ไม่ explicit มาก, เน้น romantic
Example: "เขาจูบเธอ อ่อนโยน เหมือนกลีบดอกไม้แตะริมฝีปาก"
```

### 6.2 MEB / ReadAWrite (Adult)
```
Tone: Can be more explicit
Vocab: Direct but literary
Example: "ริมฝีปากเขาเรียกร้อง ไม่ให้เธอหลุดพ้น"
```

### 6.3 Jamsai / Published (Professional)
```
Tone: Literary, polished
Vocab: เลือกคำที่สละสลวย
Example: "เขาประทับจูบลงบนริมฝีปากเธอ ราวกับโหยหามานาน"
```

## 7. Common Mistakes to Avoid

### 7.1 Too Literal
```
❌ "เขาดูดริมฝีปากล่างของเธอเข้าไปในปากของเขา"
✅ "เขาดูดริมฝีปากล่างของเธอ"
```

### 7.2 Wrong Register
```
❌ "เขาทำการจูบเธอ" (formal)
✅ "เขาจูบเธอ"
```

### 7.3 Awkward Words
```
❌ "passion" → "ความเสน่หา" (old-fashioned)
✅ "passion" → "ความปรารถนา" / "ความต้องการ"
```

### 7.4 Over-Translation
```
❌ "My love, you are so beautiful tonight"
   "ที่รักของฉัน คุณช่างสวยมากในคืนนี้"

✅ "สวยมาก...คืนนี้"
```

## 8. Translation Workflow

### Step 1: First Pass - Meaning
```
- แปลความหมายก่อน
- ไม่ต้องสวย แค่ถูกต้อง
```

### Step 2: Second Pass - Natural Thai
```
- ปรับให้เป็นภาษาไทยธรรมชาติ
- ตัดคำที่ไม่จำเป็น
- เรียงประโยคใหม่ถ้าต้อง
```

### Step 3: Third Pass - Emotion
```
- เช็คว่า emotion ยังอยู่ครบ
- เพิ่ม particles ที่เหมาะสม
- ปรับ rhythm
```

### Step 4: Final Pass - Polish
```
- อ่านออกเสียง (ในใจ)
- ตรวจ flow
- แก้คำซ้ำ
```

## 9. Intensity Preservation Guide

### Low Intensity (1-4)
```
English: "He kissed her gently"
Thai: "เขาจูบเธอเบาๆ"

Focus: Tenderness, romance
Words: อ่อนโยน, เบาๆ, แผ่วเบา, ละมุน
```

### Medium Intensity (5-7)
```
English: "His hands explored her body"
Thai: "มือเขาสำรวจร่างกายเธอ"

Focus: Passion building
Words: ร้อนรน, เร่าร้อน, ปรารถนา
```

### High Intensity (8-10)
```
English: "He claimed her completely"
Thai: "เขาครอบครองเธอทั้งหมด"

Focus: Intensity, urgency
Words: กระหาย, ดิบ, รุนแรง, ครอบครอง
```

## 10. Quality Checklist

### Before Submitting:
```
✅ ความหมายถูกต้องครบถ้วน
✅ ภาษาไทยลื่นไหลธรรมชาติ
✅ Intensity เท่าต้นฉบับ
✅ Emotion ครบถ้วน
✅ ไม่มีคำแปลกๆ
✅ Dialogue natural
✅ ไม่ซ้ำคำมากเกินไป
✅ เหมาะกับ platform
```

## 12. Example Translation

### English (Original):
```
He pressed her against the wall, his breath hot against
her neck. "I've wanted this," he whispered, "for so long."

She shivered, her fingers gripping his shirt.
"Then don't wait anymore."

His lips found hers, hungry and desperate. His hands
traced down her sides, pulling her closer. There was
no space between them now. Just heat. Just need.
```

### Thai (Translated):
```
เขาดันเธอพิงกำแพง ลมหายใจร้อนรดต้นคอ

"อยากได้มานานแล้ว" เขากระซิบ "นานมาก"

เธอสั่น นิ้วเกาะเสื้อเขาแน่น
"งั้นก็อย่ารอ"

ริมฝีปากเขาพบริมฝีปากเธอ กระหาย เร่งร้อน
มือลากผ่านเรือนร่างเธอ ดึงเข้ามาชิด
ไม่มีช่องว่างระหว่างกันอีกแล้ว
มีแค่ความร้อน แค่ความต้องการ
```
