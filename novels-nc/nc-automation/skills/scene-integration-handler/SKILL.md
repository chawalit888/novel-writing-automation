# Scene Integration Handler Skill

## Overview
Skill สำหรับเชื่อม NC scenes เข้ากับ main story อย่างลื่นไหล รวมถึงการสร้าง transition เข้า/ออก และ aftermath ที่มีผลต่อเรื่อง

---

## 1. Integration Components

### 1.1 Lead-In (ก่อนเข้า NC)
```
การเตรียมผู้อ่านก่อน NC scene:
- Emotional buildup
- Situation setup
- Character state
- Anticipation building
```

### 1.2 The NC Scene
```
ตัว scene เอง (จาก nc-scene-polish):
- Buildup
- Rising action
- Peak
- Initial aftermath
```

### 1.3 Transition Out
```
การออกจาก NC scene:
- Immediate aftermath
- Return to reality
- Next scene setup
```

### 1.4 Story Integration
```
ผลกระทบต่อเรื่อง:
- Relationship change
- Character development
- Plot implications
- Future foreshadowing
```

## 2. Lead-In Techniques

### 2.1 Emotional Lead-In

**Building Desire:**
```
ก่อนถึง NC scene:
- แสดง tension สะสม
- Almost moments
- Character thoughts about desire
- Physical awareness เพิ่มขึ้น
```

**Example:**
```markdown
[Previous scene - dinner together]

เขามองเธอข้ามโต๊ะอาหาร
แสงเทียนทำให้ผิวเธอเหมือนทองคำ

'หยุดมองได้แล้ว' เขาบอกตัวเอง
แต่สายตาไม่ยอมเชื่อฟัง

เธอเงยหน้า จับได้ว่าเขาจ้องมา
แล้วยิ้ม ยิ้มที่ทำให้หัวใจเขาพลิกคว่ำ

"กลับกันเถอะ" เธอพูด
เสียงเรียบ แต่แววตาไม่เรียบเลย

[→ Transition to NC scene]
```

### 2.2 Situational Lead-In

**Setting the Stage:**
```
สร้างสถานการณ์ที่นำไปสู่ NC:

Physical:
- กลับถึงห้อง
- ฝนตก ต้องหลบ
- เมาเล็กน้อย (ยังมีสติ)
- อยู่สองต่อสอง

Emotional:
- หลังจากเหตุการณ์สำคัญ
- Confession ที่เพิ่งเกิด
- Reunion หลังจากกัน
- Crisis ที่ผ่านไป
```

### 2.3 Types of Transitions Into NC

**Type 1: Natural Flow**
```
จบ scene ก่อนหน้า → NC เริ่มต่อเนื่อง

"...ประตูปิดลง
และทุกอย่างที่ทั้งคู่กลั้นไว้ก็ทะลักออกมา"

[NC scene begins]
```

**Type 2: Time Skip**
```
จบ scene → [ข้าม] → NC เริ่ม

"กลับกันเถอะ" เธอพูด
เขาพยักหน้า

---

หลังจากนั้น ในห้องของเขา...

[NC scene begins]
```

**Type 3: Scene Cut**
```
จบกลาง tension → ตัดไป NC

"ถ้าเธอไม่หยุดมองแบบนั้น ฉันจะ—"
"จะทำอะไร"

[Scene break]

เขาไม่ตอบด้วยคำพูด

[NC scene begins]
```

## 3. Transition Out Techniques

### 3.1 Immediate Aftermath

**Right After NC:**
```
สิ่งที่เกิดทันทีหลัง NC scene:

Physical:
- นอนข้างกัน
- หายใจเริ่มสงบ
- ร่างกายยังติดกัน
- ความเงียบ

Emotional:
- ความรู้สึกหลังจาก
- Vulnerable moment
- First words after
```

**Example:**
```markdown
[After NC scene ends]

ความเงียบครอบคลุมห้อง
มีแค่เสียงหายใจของทั้งคู่

เขาดึงเธอเข้ามาใกล้ ริมฝีปากแตะหน้าผาก

"โอเคไหม" เขาถามเบาๆ

เธอพยักหน้า ซุกหน้าลงที่อกเขา
ได้ยินเสียงหัวใจที่ยังเต้นเร็ว

'เหมือนของเธอ' เธอคิด และยิ้ม
```

### 3.2 Pillow Talk

**Conversation After:**
```
บทสนทนาหลัง NC scene:

Topics:
- ความรู้สึกที่มี
- คำถามเกี่ยวกับอนาคต
- เรื่องส่วนตัวที่ไม่เคยเล่า
- แค่เรื่องเล็กๆ น้อยๆ

Effect: Deepens intimacy, shows comfort
```

**Example:**
```markdown
"ตอนเจอกันครั้งแรก..." เขาเริ่ม

"อืม?"

"คิดว่าเธอน่ารำคาญมาก"

เธอหัวเราะ "เหรอ"

"แต่ตอนนี้..." เขาหยุด "ตอนนี้คิดว่าเธอน่ารำคาญ...ในแบบที่รักมาก"

เธอตีไหล่เขาเบาๆ แต่กอดแน่นขึ้น
```

### 3.3 Return to Reality

**Leaving the Bubble:**
```
กลับสู่โลกความจริง:

Triggers:
- แสงเช้า
- โทรศัพท์ดัง
- ต้องไปทำงาน
- มีคนมาเคาะประตู

Handling:
- Reluctant parting
- Promise to continue
- Plans for next time
- Or... awkwardness
```

## 4. Story Impact Integration

### 4.1 Relationship Changes

**Positive Changes:**
```
✅ ใกล้ชิดกันมากขึ้น
✅ Trust เพิ่มขึ้น
✅ เปิดเผยความรู้สึกได้มากขึ้น
✅ Commitment ชัดเจนขึ้น
```

**Complex Changes:**
```
⚠️ สิ่งที่ต้อง address หลัง NC:
- "แล้วเราเป็นอะไรกัน"
- ความรู้สึกผิด (ถ้ามี)
- Expectations ที่ต่างกัน
- การรักษาความลับ
```

### 4.2 Character Development

**What NC Reveals:**
```
ตัวละครเปลี่ยนไปอย่างไร:

Char A:
- เปิดใจมากขึ้น
- Trust มากขึ้น
- Vulnerability ที่ไม่เคยแสดง

Char B:
- ตระหนักถึงความรู้สึกจริง
- กล้าขึ้น
- ความต้องการที่ซ่อนไว้
```

### 4.3 Plot Implications

**NC Can Lead To:**
```
Positive:
- ความสัมพันธ์ที่แน่นแฟ้น
- Motivation เพิ่ม (ปกป้องคนรัก)
- Secret bond (forbidden love)

Conflict:
- ถูกจับได้
- ความรู้สึกผิดต่อคนอื่น
- Power dynamic เปลี่ยน
- ความคาดหวังไม่ตรงกัน
```

### 4.4 Future Foreshadowing

**What to Plant:**
```
ใส่ไว้ใน aftermath เพื่อใช้ทีหลัง:

- คำพูดที่จะมีความหมายภายหลัง
- Physical mark (รอยจูบ, รอยช้ำ)
- Promise ที่จะต้องรักษา
- Secret ที่แชร์กัน
```

## 5. Integration Patterns

### Pattern 1: Sweet Integration
```
Lead-in: Romantic buildup, tender moments
NC: Gentle, loving
Aftermath: Cuddles, sweet conversation
Story impact: Relationship solidifies

Best for: Romance, BL sweet
```

### Pattern 2: Dramatic Integration
```
Lead-in: After crisis/emotional turmoil
NC: Passionate, desperate
Aftermath: Emotional revelation, tears
Story impact: Major relationship shift

Best for: Drama, angst
```

### Pattern 3: Tension Integration
```
Lead-in: Forbidden, shouldn't happen
NC: Intense, conflicted
Aftermath: "What have we done" / but no regret
Story impact: Complications begin

Best for: Forbidden love, affair stories
```

### Pattern 4: Comedic Integration
```
Lead-in: Funny circumstances
NC: Playful elements
Aftermath: Awkward/funny moments
Story impact: Lightens mood, bonds characters

Best for: Romantic comedy
```

### Pattern 5: Plot-Driven Integration
```
Lead-in: Plot necessity (fake dating becomes real, etc.)
NC: Changes everything
Aftermath: "Now what?"
Story impact: Major plot point

Best for: Plot-heavy stories
```

## 6. Handling Different Scenarios

### 6.1 First Time Together

**Lead-in needs:**
```
- Clear consent
- Some nervousness is realistic
- Anticipation building
```

**NC adjustments:**
```
- Some awkwardness okay
- Discovery element
- "Is this okay?" moments
```

**Aftermath should include:**
```
- Vulnerability
- "That was..."
- What this means for them
```

### 6.2 Established Couple

**Lead-in can be:**
```
- More casual
- Playful initiation
- Quickie setup or planned
```

**NC adjustments:**
```
- More comfortable
- Can include humor
- Familiar but not boring
```

**Aftermath might include:**
```
- Routine comfort
- Inside jokes
- Plans/future talk
```

### 6.3 Reunion After Separation

**Lead-in needs:**
```
- Longing established
- Tension from time apart
- "Finally" moment
```

**NC adjustments:**
```
- Desperate quality
- Relearning each other
- Emotion heavy
```

**Aftermath should include:**
```
- Never letting go
- Time apart reflection
- Promise not to be apart again
```

### 6.4 Angry/Make-up Intimacy

**Lead-in needs:**
```
- Argument/conflict before
- Tension explosion
- "Shut up" kiss moment
```

**NC adjustments:**
```
- More rough/passionate
- Still showing love underneath
- Anger to passion
```

**Aftermath should include:**
```
- Actually resolving the issue
- "I still love you even when..."
- Communication
```

## 8. Integration Checklist

### Before NC:
```
✅ Emotional state ของทั้งคู่ clear
✅ Situation setup สมเหตุสมผล
✅ Anticipation built
✅ Consent established or will be in NC
```

### NC Scene:
```
✅ Matches story tone
✅ Characters in-character
✅ Pacing appropriate
✅ Complete (buildup-peak-initial aftermath)
```

### After NC:
```
✅ Immediate aftermath written
✅ Return to reality handled
✅ What this means for relationship
✅ Plot implications addressed
✅ Next scene transition smooth
```

### Story Level:
```
✅ NC scene makes sense at this point
✅ Relationship development justified
✅ Character growth shown
✅ No plot holes created
✅ Future threads planted (if needed)
```

## 10. Common Mistakes

### Mistake 1: NC Out of Nowhere
```
❌ ไม่มี buildup เลย กระโดดเข้า NC
✅ มี lead-in ที่ build tension
```

### Mistake 2: No Aftermath
```
❌ NC จบแล้วตัดไปเรื่องอื่นเลย
✅ มี aftermath ที่แสดง impact
```

### Mistake 3: NC ไม่มีผลต่อเรื่อง
```
❌ หลัง NC ทุกอย่างเหมือนเดิม
✅ ความสัมพันธ์เปลี่ยน, มีผลต่อเรื่อง
```

### Mistake 4: Tone Mismatch
```
❌ เรื่อง comedy แต่ NC dramatic มาก
✅ NC tone ควรเข้ากับเรื่อง
```

### Mistake 5: Rushed Integration
```
❌ Lead-in 1 line, aftermath 1 line
✅ ใส่ใจทั้ง before และ after
```
