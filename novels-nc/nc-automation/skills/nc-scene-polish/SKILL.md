# NC Scene Polish Skill

## Overview
Skill สำหรับขัดเงาและปรับปรุง NC scenes ที่ generate จาก Ollama ให้มีคุณภาพสูงขึ้น

---

## 1. เป้าหมายหลัก

### 1.1 ปรับ Pacing
- Slow down ในจุดสำคัญ
- Speed up ในส่วนที่ไม่ต้อง detail มาก
- สร้าง rhythm ที่ลื่นไหล

### 1.2 เพิ่ม Sensory Details
- Touch (สัมผัส) - ความร้อน ความนุ่ม texture
- Sound (เสียง) - หายใจ เสียงครวญ เสียงกระซิบ
- Smell (กลิ่น) - น้ำหอม กลิ่นผิว
- Sight (ภาพ) - แสงเงา สีผิว การเคลื่อนไหว
- Taste (รส) - รสจูบ เหงื่อ

### 1.3 เพิ่ม Emotional Depth
- ความรู้สึกภายใน
- Internal monologue
- ความหมายของการกระทำ

---

## 2. Polish Checklist

### Before Polish
```markdown
- [ ] อ่าน scene ทั้งหมดก่อน
- [ ] ระบุ weak points
- [ ] ระบุ intensity level ที่ต้องการ
- [ ] ตรวจ character consistency
```

### During Polish
```markdown
- [ ] เพิ่ม sensory details ทุก paragraph
- [ ] ปรับ pacing - slow down at peaks
- [ ] เพิ่ม emotional reactions
- [ ] ตรวจ dialogue ให้ natural
- [ ] เพิ่ม physical reactions
```

### After Polish
```markdown
- [ ] อ่านทวนอีกครั้ง
- [ ] เช็ค flow ของ scene
- [ ] ตรวจว่าไม่มี repetition มากเกินไป
- [ ] ยืนยัน intensity ตรงตามต้องการ
```

## 3. Sensory Details Guide

### Touch (สำคัญที่สุดสำหรับ NC)

**Texture:**
```
- ผิวนุ่ม / ผิวหยาบ
- กล้ามเนื้อแข็งแรง
- ผมนุ่มลื่น
- ริมฝีปากอุ่น
```

**Temperature:**
```
- ความร้อนที่แผ่ออกมา
- เหงื่อเย็นชื้น
- ลมหายใจอุ่น
- มือเย็น/ร้อน
```

**Pressure:**
```
- สัมผัสแผ่วเบา
- กอดแน่น
- กดทับ
- ลูบไล้
```

### Sound

**Voice:**
```
- กระซิบแหบพร่า
- ครวญเบาๆ
- หายใจเร็วขึ้น
- เรียกชื่อ
- "อืม..." "อา..."
```

**Physical:**
```
- เสียงจูบ
- เสียงผ้าเสียดสี
- เตียงเคลื่อน
- หัวใจเต้น
```

### Smell
```
- น้ำหอมติดผิว
- กลิ่นตัวเฉพาะคน
- กลิ่นเหงื่อ
- กลิ่นผมหลังสระ
```

### Sight
```
- แสงจันทร์/แสงเทียน
- เงาบนผนัง
- ดวงตาที่มองจับตา
- ริมฝีปากแดงระเรื่อ
- กล้ามเนื้อที่เกร็ง
```

## 4. Pacing Techniques

### Slow Down (จุดสำคัญ)
```
ใช้เมื่อ:
- First touch
- First kiss ใน scene
- Climactic moments
- Emotional revelations

วิธีทำ:
- เพิ่ม sensory details
- แทรก internal thought
- บรรยายละเอียด
- ใช้ประโยคสั้นๆ ต่อกัน
```

### Speed Up (ส่วนที่ผ่านได้)
```
ใช้เมื่อ:
- Transition ระหว่าง positions
- ช่วงกลางของ action
- Build-up ที่ไม่ใช่ key moments

วิธีทำ:
- ใช้ประโยคยาวขึ้น
- Summary แทน detail
- Skip repetitive actions
```

### Rhythm Pattern
```
Buildup: Slow → Medium
Rising: Medium → Fast
Peak: SLOW (detail สูงสุด)
Aftermath: Slow → Settle
```

## 5. Emotional Layer Enhancement

### Adding Internal Monologue
```
❌ Before:
"เขาจูบเธอ"

✅ After:
"เขาจูบเธอ และในหัวว่างเปล่า
มีแค่รสของเธอ กลิ่นของเธอ
เขาไม่เคยต้องการใครขนาดนี้"
```

### Adding Meaning
```
❌ Before:
"เขาจับมือเธอ"

✅ After:
"เขาจับมือเธอ นิ้วสอดประสาน
เหมือนต้องการยึดเธอไว้ ไม่ยอมปล่อย
แม้จะรู้ว่าพรุ่งนี้ต้องจากกัน"
```

### Physical Reactions = Emotional Mirror
```
ความต้องการ → หายใจเร็ว, ตัวสั่น
ความรัก → สัมผัสอ่อนโยน, จูบนาน
ความหลงใหล → เร่งรีบ, แรง
ความอ่อนโยน → ช้า, ละเมียด
```

## 6. Dialogue Polish

### Natural Dialogue Rules
```
1. ไม่พูดเยอะเกินไประหว่าง action
2. ใช้คำสั้นๆ ในจังหวะเร่งด่วน
3. เรียกชื่อให้ intimate
4. ใส่ หายใจ/ครวญ ระหว่างคำพูด
```

### Examples

**❌ Unnatural:**
```
"ฉันรักเธอมาก และฉันต้องการเธอ
เธอสวยมากคืนนี้"
```

**✅ Natural:**
```
"รักนะ..." เขากระซิบ ริมฝีปากแนบหู
"รัก...มาก..."
```

**❌ Too Much Talking:**
```
"เธอรู้สึกยังไง ตอนนี้เธอต้องการอะไร
บอกฉันสิว่าชอบแบบไหน"
```

**✅ Show Instead:**
```
"ชอบไหม" แค่คำถามสั้นๆ
เธอตอบด้วยเสียงครวญแทนคำพูด
```

## 7. Common Problems & Fixes

### Problem 1: Mechanical Writing
```
❌ "เขาจูบเธอ แล้วลูบหลังเธอ แล้วจับเอวเธอ"

✅ "ริมฝีปากเขาค่อยๆ เคลื่อนลงมา
มืออีกข้างลูบไล้แผ่นหลังเธอ
ก่อนจะเลื่อนมาประคองเอว"
```

### Problem 2: Repetitive Words
```
❌ "เธอครวญ" ซ้ำ 10 ครั้ง

✅ Alternatives:
- เสียงหลุดออกมา
- ส่งเสียงเบาๆ
- ร้องครางเบาๆ
- หอบหายใจ
- กลั้นไม่อยู่
```

### Problem 3: Missing Transitions
```
❌ "พวกเขายืนจูบกัน" [ตัด] "อยู่บนเตียงแล้ว"

✅ "จูบลึกขึ้น ร้อนแรงขึ้น
เธอไม่รู้ตัวว่าถูกพาถอยหลัง
จนหลังแตะขอบเตียง"
```

### Problem 4: Over-explicit Without Emotion
```
❌ บรรยาย action อย่างเดียว ไม่มี feeling

✅ เพิ่ม:
- ความรู้สึกของทั้งคู่
- สายตาที่สบกัน
- ความหมายของช่วงเวลานี้
```

## 8. Polish Workflow

### Step 1: First Read
```
- อ่านทั้งหมดไม่หยุด
- จด weak points
- จด missing elements
```

### Step 2: Structure Check
```
- Buildup พอไหม?
- Peak ชัดเจนไหม?
- Aftermath มีไหม?
```

### Step 3: Sensory Pass
```
- เพิ่ม touch ในทุก paragraph
- เพิ่ม sound ที่เหมาะสม
- เพิ่ม smell/sight ตามจังหวะ
```

### Step 4: Emotional Pass
```
- เพิ่ม internal monologue
- เพิ่ม meaning/significance
- เพิ่ม eye contact moments
```

### Step 5: Pacing Pass
```
- Slow down at peaks
- Speed up transitions
- Check rhythm
```

### Step 6: Dialogue Pass
```
- Natural speech
- Not too much
- Add breaths/sounds
```

### Step 7: Final Read
```
- อ่านเหมือนผู้อ่าน
- ตรวจ flow
- ตรวจ repetition
```

## 9. Intensity Adjustment

### Making Scene Hotter (เพิ่ม Intensity)
```
1. เพิ่ม physical details
2. เพิ่ม reactions ที่ intense
3. ใช้คำที่ explicit มากขึ้น
4. เพิ่ม urgency/desperation
5. เพิ่ม dialogue ที่ bold
```

### Making Scene Softer (ลด Intensity)
```
1. เน้น emotional มากกว่า physical
2. ใช้ metaphor แทน explicit
3. Focus ที่ faces/eyes
4. เพิ่ม tenderness
5. ลด urgency, เพิ่ม gentleness
```

## 10. Quality Metrics

### Excellent NC Scene Should Have:
```
✅ Clear emotional arc
✅ 5 senses represented
✅ Natural dialogue
✅ Proper pacing (slow at peaks)
✅ Character-consistent reactions
✅ Meaningful connection
✅ Smooth transitions
✅ Satisfying aftermath
```

### Red Flags:
```
❌ All physical, no emotion
❌ Repetitive words/actions
❌ Mechanical listing of actions
❌ No character voice
❌ Rushed or dragged pacing
❌ Missing consent clarity
```


## 12. Before/After Example

### Before (Raw from Ollama):
```
He kissed her. She kissed back. They moved to the bed.
He touched her body. She moaned. They continued.
```

### After (Polished):
```
เขาจูบเธอ เบาๆ ก่อน เหมือนขออนุญาต
แล้วเธอก็ตอบ ดึงเขาเข้ามาใกล้ขึ้น

ลมหายใจผสมกัน ร้อนและเร่งรีบ
มือเขาลูบแก้มเธอ ลงมาที่คอ
เธอสั่น แต่ไม่ใช่เพราะกลัว

"ไหวไหม" เขาถามเบาๆ
เธอตอบด้วยการดึงเขาลงมา

หลังเธอแตะผ้าปูนุ่มๆ
ความหนักของเขาทับลงมา
ความอุ่นแผ่ไปทั้งตัว

เธอครวญเบาๆ เมื่อมือเขาเริ่มสำรวจ
สัมผัสทุกที่ที่เขาเคยจินตนาการ
ค่อยๆ ไม่รีบ เหมือนต้องการจดจำทุกรายละเอียด

"สวยมาก" เขากระซิบ
และเธอก็ละลายไปกับคำพูดนั้น
```
