┌─────────────────────────────────────────────────────────────┐
│                  องก์ที่ 1: พบกัน & Chemistry                │
│                      (ตอนที่ 1-6)                           │
└─────────────────────────────────────────────────────────────┘

[จุดเริ่มต้น]
• พิมพ์ลดา (พิม) เริ่มงานเลขาประธานบริษัทวันแรก
• เจอธนัท CEO หน้าเย็น ปากร้าย ทะเลาะกันตั้งแต่แรกเจอ
• ธนัทตั้งกฎเข้มงวด พิมไม่ยอม ปะทะกันดุเดือด

[Tension Build-up]
• ธนัทรู้เรื่องแม่พิมป่วยหนัก เริ่มสนใจเธอมากขึ้น
• เดินทางไปทริปธุรกิจเชียงใหม่ด้วยกัน booking ผิดพลาด มีห้องเดียว
• ความใกล้ชิดในคืนที่ฝนตก ไฟดับ...

🔥 ฉาก NC แรก (ตอนที่ 6):
   • Trigger: ห้องพักเดียว ฝนตกหนัก ไฟดับ ความใกล้ชิด
   • ประเภท: [x] passionate
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: ความต้องการที่สะสม + chemistry ที่ระเบิด

┌─────────────────────────────────────────────────────────────┐
│                องก์ที่ 2: สัญญาลับ                           │
│                      (ตอนที่ 7-12)                          │
└─────────────────────────────────────────────────────────────┘

[หลังคืนแรก]
• ตื่นมาอยู่ในอ้อมแขนกัน ทั้งคู่เขินแต่แกล้งลืม
• ธนัทเสนอสัญญา - คู่หมั้นปลอม แลกเงินค่ารักษาแม่
• กฎ 5 ข้อ: ห้ามมีความรู้สึก ห้ามบอกใคร ต้องหวานในที่สาธารณะ

[งานเลี้ยงเปิดตัว]
• เปิดตัวคู่หมั้น ดา (อดีตคู่หมั้น) ปรากฏตัวสร้างปัญหา
• ธนัทอ้าง "ซ้อมความหวาน" เริ่มข้ามเส้น

🔥 ฉาก NC ช่วงกลาง (ตอนที่ 11-12):
   • ตอน 11: ซ้อมให้สมจริง - จูบที่ไม่ใช่แค่ซ้อม (6/10)
   • ตอน 12: ในรถประธาน - กลับจากงานเลี้ยง ควบคุมไม่อยู่ (8/10)
   • พัฒนาการ: จากสัญญาเริ่มกลายเป็นความต้องการจริง

┌─────────────────────────────────────────────────────────────┐
│                องก์ที่ 3: ความรู้สึกก่อตัว                    │
│                      (ตอนที่ 13-18)                         │
└─────────────────────────────────────────────────────────────┘

[ความสัมพันธ์พัฒนา]
• พิมเริ่มสับสนในความรู้สึก แนนเตือนอย่าหลงจริง
• วันเกิดธนัท พิมเซอร์ไพรส์เค้กเล็กๆ ธนัทประทับใจ
• วินเข้ามาจีบพิม ธนัทหึงจนควบคุมไม่อยู่

[จุดหักมุม / Paywall Hook (ตอนที่ 10)]
• ดาเริ่มแผนทวงคืนธนัท ปล่อยข่าวลือในออฟฟิศ
• พิมถูกดูถูกว่าเป็นแค่ "ของเล่น" ของประธาน
• Cliffhanger: พิมจะทนอยู่ต่อหรือลาออก?

🔥 ฉาก NC สำคัญ (ตอนที่ 15, 18):
   • ตอน 15: ของขวัญวันเกิด - คืนโรแมนติก (8/10)
   • ตอน 18: ลงโทษ - ธนัท "สั่งสอน" เรื่องวิน มัดมือ (9/10)

┌─────────────────────────────────────────────────────────────┐
│                องก์ที่ 4: ความสัมพันธ์ซับซ้อน                 │
│                      (ตอนที่ 19-24)                         │
└─────────────────────────────────────────────────────────────┘

[ความขัดแย้งถึงจุดสูงสุด]
• พิมไปเยี่ยมพ่อธนัท ถูกดูถูกว่าไม่คู่ควร
• ดากลับมาเผยความลับ (โกหก) ว่าธนัทยังรักเธออยู่
• พิมเข้าใจผิด คิดว่าตัวเองเป็นแค่ตัวแทน

[ความลับแตก / All is Lost]
• พิมขอถอยออก ทำตัวห่างเหิน
• ธนัททนไม่ไหว บุกห้องพิม บังคับให้ฟัง

🔥 ฉาก NC ช่วงวิกฤต (ตอนที่ 20, 24):
   • ตอน 20: ปลอบใจ - หลังถูกพ่อต่อว่า อ่อนโยน (7/10)
   • ตอน 24: บังคับให้ยอมรับ - ดุเดือด ครอบครอง (9/10)

┌─────────────────────────────────────────────────────────────┐
│                องก์ที่ 5: วิกฤต                              │
│                      (ตอนที่ 25-30)                         │
└─────────────────────────────────────────────────────────────┘

[ความจริงถูกเปิดเผย]
• ธนัทรู้ว่าพ่อตัวเองทำให้บริษัทพ่อพิมล้มละลาย
• ธนัทพยายามปกปิด ปกป้องพิมจากความเจ็บปวด
• ดาแฉทุกอย่างต่อหน้าสื่อ - สัญญาปลอม + เรื่องพ่อ

[จุดต่ำสุด]
• พิมรู้ทุกอย่าง รู้สึกถูกหลอก ลาออกหนีไป
• ธนัทตามหาทุกที่ ยอมทิ้งศักดิ์ศรี
• แม่พิมเล่าความจริง - ธนัทช่วยค่ารักษาตลอดโดยไม่บอก

┌─────────────────────────────────────────────────────────────┐
│                  องก์ที่ 6: คืนดี & Happy Ending             │
│                      (ตอนที่ 31-36)                         │
└─────────────────────────────────────────────────────────────┘

[ตัวละครพิสูจน์ตัวเอง]
• ธนัทมาหาพิมที่โรงพยาบาล อธิบายทุกอย่าง ขอโอกาส
• ธนัทเผชิญหน้ากับพ่อ ประกาศเลือกพิมเหนือธุรกิจ
• ขอแต่งงานริมทะเล

[ตอนจบ - Happy Ending]
• พิธีแต่งงาน ทุกคนมาร่วมยินดี แม่หายป่วย
• ชีวิตครอบครัวที่มีความสุข

🔥 ฉาก NC ไคลแมกซ์ (ตอนที่ 32, 34, 36):
   • ตอน 32: เริ่มต้นใหม่ - คืนดี หวาน (8/10)
   • ตอน 34: หลังขอแต่งงาน - ฉลองริมทะเล (8/10)
   • ตอน 36: คืนแต่งงาน - ชุดเจ้าสาว (10/10) - สูงสุด!
