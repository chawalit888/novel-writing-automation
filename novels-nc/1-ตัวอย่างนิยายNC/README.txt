                   โครงสร้างโฟลเดอร์นิยาย NC
══════════════════════════════════════════════════════════════

โฟลเดอร์นี้ใช้สำหรับเก็บไฟล์นิยาย NC หนึ่งเรื่อง
โครงสร้างที่แนะนำ:

📁 [ลำดับ]-[ชื่อเรื่อง]/
├── 01-ชื่อเรื่อง-nc.txt        ← ข้อมูลพื้นฐานเรื่อง
├── 02-plot-nc.txt              ← โครงเรื่อง + ตำแหน่งฉาก NC
├── 03-ตัวละคร-nc.txt           ← ข้อมูลตัวละคร + NC traits
├── 04-เรื่องย่อ-nc.txt         ← เรื่องย่อ + คำโปรย
├── ตอนที่01-[ชื่อตอน].txt      ← เนื้อหาตอนที่ 1
├── ตอนที่02-[ชื่อตอน].txt      ← เนื้อหาตอนที่ 2
├── ...
├── ฉากNC/                       ← โฟลเดอร์เก็บฉาก NC
│   ├── ฉาก01-[ชื่อฉาก].txt     ← ฉาก NC พร้อม metadata
│   ├── ฉาก02-[ชื่อฉาก].txt
│   └── ...
└── README.txt                   ← ไฟล์นี้

══════════════════════════════════════════════════════════════
                       การใช้งาน
══════════════════════════════════════════════════════════════

1. Copy โฟลเดอร์นี้เป็น template
2. เปลี่ยนชื่อโฟลเดอร์เป็น "[ลำดับ]-[ชื่อเรื่อง]"
3. กรอกข้อมูลในแต่ละไฟล์ template
4. เขียนตอนปกติด้วย Claude
5. สร้างฉาก NC ด้วย Ollama → แปลด้วย Claude
6. รวมฉาก NC เข้าไปในตอน
