===============================================
NC Writing Server - คู่มือติดตั้ง
สำหรับ Mac Studio M1 Max 64GB
===============================================

## วิธีติดตั้ง (ทำที่ Mac Studio)

1. Copy โฟลเดอร์นี้ทั้งหมดไป Mac Studio
   (ใช้ AirDrop หรือ USB)

2. เปิด Terminal

3. ไปที่โฟลเดอร์:
   cd ~/Desktop/nc-server-setup

4. ให้สิทธิ์รัน:
   chmod +x *.sh

5. รัน script ทีละตัว:
   ./1-install-ollama.sh      (รอ 15-30 นาที)
   ./2-install-sillytavern.sh (รอ 5-10 นาที)

===============================================
## วิธีใช้งาน

1. ดับเบิลคลิก "NC-Server" บน Desktop
   หรือรัน: ~/start-nc-server.sh

2. เปิด Browser: http://localhost:8000

3. ตั้งค่า API ครั้งแรก:
   - Settings → API → Text Completion → Ollama
   - URL: http://localhost:11434

===============================================
## Models ที่ติดตั้ง

- dolphin-mistral:7b    = เร็ว (สำหรับทดสอบ)
- dolphin-mixtral:8x7b  = คุณภาพสูง (แนะนำ)
