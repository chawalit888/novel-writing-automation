[จุดเริ่มต้น — Rebirth — ตอนที่ 1-3]

• ตอนที่ 1: ตายแล้วเกิดใหม่
  - ณิชาภา (นิช) สาวออฟฟิศวัย 28 ถูกรถชนตาย
  - ตื่นขึ้นมาในร่างของ พิมลรัตน์ (มิลลี่) สาววัย 20
  - มิลลี่คือลูกสาวของ ป้าแก้ว แม่บ้านประจำคฤหาสน์ตระกูลมาเฟีย "วรเดช"
  - ค่อยๆ รับรู้สถานการณ์: เธออยู่ในโลกที่มาเฟียมีอำนาจเหนือกฎหมาย
  - **จุดพลิก:** ความทรงจำของมิลลี่ตัวจริงค่อยๆ ผุดขึ้น — เธอเคยเห็นเหตุการณ์
    ที่ เตชิน (เจ้าลอร์ด) ถูกทรยศและตายในอีก 2 ปีข้างหน้า

• ตอนที่ 2: คฤหาสน์แห่งอันตราย
  - มิลลี่เริ่มเรียนรู้กฎเหล็กของบ้านมาเฟีย
  - แนะนำตัวละคร: ป้าแก้ว (แม่ ห่วงลูก) / คนรับใช้คนอื่นที่กลัวตาย
  - เห็นเตชินเป็นครั้งแรก — เย็นชา อันตราย สายตาฆ่าคน
  - เตชินมองผ่านมิลลี่เหมือนเธอไม่มีตัวตน — เธอคือแค่ลูกคนรับใช้
  - มิลลี่ตัดสินใจ: "ชีวิตนี้ ฉันจะเปลี่ยนชะตา ฉันจะไม่ปล่อยให้เขาตาย"

• ตอนที่ 3: ก้าวแรกของแผน
  - มิลลี่ใช้ความรู้จากชีวิตก่อน (บัญชี/การเงิน/การตลาด) วิเคราะห์ธุรกิจตระกูล
  - สังเกตจุดอ่อนในระบบ — รู้ว่ามีคนยักยอกเงิน
  - แอบส่ง "คำใบ้" ไม่ระบุตัวตนให้ วรุตม์ (มือขวาเตชิน) เรื่องตัวเลขผิดปกติ
  - เตชินเริ่มสั่งสืบ — ใครส่งข้อมูลนี้มา?
  - มิลลี่ต้องระวัง: ถ้าถูกจับได้ว่าเป็นคนส่ง อาจตายก่อนเปลี่ยนชะตา

[ดึงดูดความสนใจ — ตอนที่ 4-6]

• ตอนที่ 4: คนรับใช้ที่ไม่ธรรมดา
  - เตชินสังเกตมิลลี่: เธอพูดจาไม่เหมือนลูกคนรับใช้ทั่วไป
  - ฉากมิลลี่ช่วยแก้ปัญหาเรื่องการขนส่ง (ใช้ความรู้โลจิสติกส์จากชีวิตก่อน)
  - เตชินเรียกมิลลี่ไปถาม "เธอเรียนรู้สิ่งนี้จากที่ไหน?"
  - มิลลี่ตอบอ้อมค้อม — เตชินจ้องตาเธอ "ร่างกายเธอสั่น แต่ตาไม่กลัว"
  - ป้าแก้วห่วง สั่งลูกสาวอยู่ให้ห่างเจ้านาย

• ตอนที่ 5: ทดสอบ
  - เตชินทดสอบมิลลี่ — สั่งให้ทำงานใกล้ชิดในห้องทำงาน
  - มิลลี่ต้องเสิร์ฟชา จัดเอกสาร อยู่ในวงประชุมมาเฟีย
  - เธอเห็นเตชิน "ลงโทษ" คนทรยศ — โหดเหี้ยมไม่กะพริบตา
  - แต่มิลลี่ไม่หนี ไม่ร้อง ไม่สั่น — เตชินสนใจขึ้นอีก
  - Tension เริ่มก่อตัว: สายตาที่จ้องนานเกินไป สัมผัสนิ้วที่ส่งถ้วยชา

• ตอนที่ 6: เบาะแสแรกของคนทรยศ
  - มิลลี่จำได้จากความทรงจำของมิลลี่ตัวจริง: ก่อนเตชินตาย มีคนในตระกูลทรยศ
  - เธอยังไม่รู้ว่าใคร แต่เริ่มสังเกตพฤติกรรมของคนใกล้ชิดเตชิน
  - แนะนำ ผู้เฒ่าวรพล (ลุงของเตชิน) — ดูน่าเชื่อถือ แต่มิลลี่รู้สึกไม่ดี
  - แนะนำ ธาริน (ลูกพี่ลูกน้องเตชิน) — มีเสน่ห์ ดูเป็นมิตร แต่ซ่อนอะไร?
  - **Cliffhanger:** มิลลี่บังเอิญได้ยินบทสนทนาลับระหว่างสองคนในตระกูล
    "ถ้าเจ้าลอร์ดตาย ทุกอย่างจะเป็นของเรา"

[Tension สูงขึ้น — ตอนที่ 7-9]

• ตอนที่ 7: เงาที่ตามติด
  - เตชินสั่งให้มิลลี่ประจำห้องทำงานถาวร — "ลูกน้องที่ฉลาดเกินไป ต้องอยู่ใกล้"
  - มิลลี่ตระหนักว่าเตชินจับตาดู — เธอกลัวแต่ก็ต้องใช้โอกาสนี้
  - ฉากใกล้ชิด: เตชินเช็ดเลือดที่แก้มมิลลี่ (ถูกคนอื่นทำร้าย)
  - "ใครกล้าแตะของฉัน" — เตชินพูดทั้งที่ไม่ได้ตั้งใจ
  - มิลลี่หัวใจเต้นแรง: ชายคนนี้อันตราย แต่เธอไม่ได้กลัว...เธอดึงดูด

• ตอนที่ 8: ราตรีในห้องลับ
  - เตชินเรียกมิลลี่ไป "สอบสวน" — เขาสงสัยว่าเธอเป็นสายลับ
  - ห้องปิดตาย สองต่อสอง มิลลี่ต้องพิสูจน์ความบริสุทธิ์
  - เตชินกดดัน จนมิลลี่ถึงจุดที่ต้องเลือก — พูดความจริงบางส่วน
  - "ฉันรู้ว่ามีคนจะทรยศท่าน ฉันอยากช่วย" — เตชินจ้องตาเธอ
  - ความตึงเครียดทางกายเริ่มปะทุ: ระยะห่างเหลือไม่กี่นิ้ว

  NC Teaser (ตอนที่ 8):
   • Trigger: การสอบสวน ใกล้ชิดในห้องปิด ความตึงเครียดปะทุ
   • ประเภท: [ ] tender [x] passionate [ ] rough
   • ความเข้มข้น: 5/10 (teaser — จูบ/สัมผัส แต่ยังไม่ข้ามเส้น)
   • อารมณ์หลัก: อำนาจ / ท้าทาย / ดึงดูดกันแม้ไม่ควร
   • รายละเอียด: เตชินจับคางมิลลี่บังคับสบตา จูบเธอกระแทกผนัง
     แต่หยุดก่อนข้ามเส้น — "เธอเล่นกับไฟ" / มิลลี่หายใจหอบ

• ตอนที่ 9: ผลที่ตามมา
  - เช้าหลังคืนนั้น — ทั้งสองทำเป็นไม่มีอะไรเกิดขึ้น
  - แต่สายตาทุกครั้งที่สบกัน...บอกตรงกันข้าม
  - มิลลี่ค้นพบหลักฐานสำคัญ: มีการโอนเงินผิดปกติจากภายในตระกูล
  - เธอตัดสินใจเสี่ยง — ส่งข้อมูลให้เตชินโดยตรง
  - เตชินเริ่มเชื่อใจมิลลี่: "เธอรู้มากเกินไปสำหรับลูกคนรับใช้"

[Paywall Zone — ตอนที่ 10-12]

• ตอนที่ 10: เปิดตัวคนทรยศ (PAYWALL HOOK)
  - มิลลี่เริ่มแกะปม — เชื่อมโยง ธาริน กับการยักยอกเงิน
  - แต่ยังไม่มีหลักฐานชัด ผู้เฒ่าวรพลดูเหมือนจะรู้เห็นด้วย
  - มิลลี่ถูกคนในตระกูลกลั่นแกล้ง (ลูกสาวคนรับใช้ไม่ควรใกล้ชิดเจ้านาย)
  - เตชินลงโทษคนที่กลั่นแกล้ง — ทุกคนตกใจ "ทำไมท่านปกป้องแค่คนรับใช้?"
  - **Cliffhanger:** เตชินสั่งให้มิลลี่ย้ายมาอยู่ห้องข้างห้องเขา
    "ตั้งแต่นี้ เธออยู่ภายใต้การดูแลของฉัน — โดยตรง"

• ตอนที่ 11: คืนที่ข้ามเส้น
  - มิลลี่อยู่ห้องข้างเตชิน — ทุกคืนได้ยินเสียงเขาทำงานดึก
  - คืนนั้น เตชินเมาหลังจัดการศัตรู — เลือดเปื้อนเสื้อ
  - มิลลี่เข้าไปช่วยเช็ดแผล — เตชินจับมือเธอ "อย่ากลัวฉัน"
  - "ฉันไม่เคยกลัวท่าน" — มิลลี่ตอบ สบตาไม่กะพริบ
  - ข้ามเส้นครั้งแรก — ความต้องการที่อั้นมาปะทุ

  NC (ตอนที่ 11):
   • Trigger: เตชินบาดเจ็บ + เมา + มิลลี่ดูแลแผลใกล้ชิด
   • ประเภท: [x] tender [x] passionate [ ] rough
   • ความเข้มข้น: 7/10 (ครั้งแรกที่ข้ามเส้น แต่ยังไม่ถึงที่สุด)
   • อารมณ์หลัก: ความเปราะบางของชายที่แข็งแกร่ง / การปลดปล่อย
   • รายละเอียด: เตชินจูบรอยแผลที่มิลลี่โดนคนในตระกูลทำ
     มิลลี่เป็นฝ่ายเอื้อมมือสัมผัสใบหน้าเขา — เตชินควบคุมตัวเองไม่อยู่
     **ตัดจบแบบ Cliffhanger** — ทำให้คนอ่านต้องซื้อตอนถัดไป

• ตอนที่ 12: เช้าหลังข้ามเส้น
  - เตชินตื่นก่อน เห็นมิลลี่นอนข้างๆ — ใบหน้าเย็นชาสั่นเล็กน้อย
  - มิลลี่ตื่น — ไม่แสดงอาการเสียใจ "ฉันทำเพราะอยาก ไม่ใช่เพราะท่านบังคับ"
  - เตชิน: "เธอรู้ไหมว่าเธอเข้ามาในดงอันตราย"
  - มิลลี่: "ฉันรู้ตั้งแต่แรกแล้ว...ท่านเจ้าลอร์ด"
  - ความสัมพันธ์เปลี่ยน — เตชินเริ่มปกป้องมิลลี่อย่างเปิดเผยมากขึ้น
  - แต่คนในตระกูลเริ่มไม่พอใจ — "ลูกสาวคนรับใช้คิดจะมาเป็นเจ้านาย?"

[ความสัมพันธ์ลึกซึ้ง + สืบความจริง — ตอนที่ 13-15]

• ตอนที่ 13: ครั้งแรกที่ยอม
  - เตชินพามิลลี่ไปห้องลับส่วนตัว — ที่ที่ไม่เคยพาใครไป
  - "ที่นี่ไม่มีเจ้าลอร์ด มีแค่ผม" — เตชินถอดหน้ากากเย็นชา
  - ครั้งแรกที่มิลลี่เห็นด้านอ่อนโยนของเขา — รอยแผลเป็นทั่วตัว
  - NC เต็มรูปแบบครั้งแรก ที่ทั้งสองยอมอย่างแท้จริง

  NC เต็มรูปแบบ (ตอนที่ 13):
   • Trigger: เตชินพามิลลี่ไปที่ปลอดภัย ถอดหน้ากาก ยอมเปิดใจ
   • ประเภท: [x] tender [x] passionate [ ] rough
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: ความไว้วางใจ / ยอมให้อีกฝ่ายเห็นด้านที่ซ่อน
   • รายละเอียด: เตชินอ่อนโยนผิดจากปกติ ค่อยๆ ถาม "พร้อมไหม"
     มิลลี่สัมผัสแผลเป็นเขาทีละรอย ก่อนดึงเขาเข้ามา
     จังหวะช้าแต่เร่าร้อน — "ร้องชื่อผม ไม่ใช่เจ้าลอร์ด"
     ครั้งแรกที่เตชินยิ้มจริงๆ หลังเสร็จสิ้น

• ตอนที่ 14: ความลับระหว่างสองคน
  - ตกลงซ่อนความสัมพันธ์ — ในที่สาธารณะยังเป็นเจ้านาย-คนรับใช้
  - มิลลี่ใช้ตำแหน่งใกล้ชิดเตชินสืบเรื่องคนทรยศ
  - ค้นพบว่า ธาริน มีการติดต่อลับกับตระกูลมาเฟียคู่แข่ง
  - แต่ยังไม่มีหลักฐานเรื่อง ผู้เฒ่าวรพล — เขามีส่วนด้วยหรือเปล่า?
  - มิลลี่ต้องระวัง: ถ้าเปิดเผยว่ารู้อนาคต จะอธิบายยังไง?

• ตอนที่ 15: ศัตรูจากภายนอก
  - ตระกูลมาเฟียคู่แข่ง (ตระกูล "สินธร") เริ่มรุกราน
  - ยิงถล่มรถขนส่งของตระกูลวรเดช — สงครามเริ่มก่อตัว
  - เตชินเริ่มยุ่งกับสงคราม ไม่มีเวลาสนใจเรื่องภายใน
  - มิลลี่รู้ว่า "นี่คือจุดเริ่มต้น" — ในชีวิตก่อน สงครามนี้ทำให้เตชินเสียเปรียบ
    จนถูกทรยศจากคนใกล้ชิด
  - เธอต้องเร่งหาคนทรยศก่อนที่ทุกอย่างจะสายเกินไป

[รักลับ + วางแผน — ตอนที่ 16-18]

• ตอนที่ 16: รักลับในห้องทำงาน
  - เตชินทำงานดึกติดต่อกันหลายวัน มิลลี่เป็นคนเดียวที่อยู่ข้างๆ
  - ความเครียดจากสงคราม + ความคิดถึงกัน ปะทุในห้องทำงาน

  NC (ตอนที่ 16):
   • Trigger: ความเครียดจากสงคราม + ใกล้ชิดในห้องทำงานดึก
   • ประเภท: [ ] tender [x] passionate [ ] rough
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: ปลดปล่อยความเครียด / ต้องการกันและกัน
   • รายละเอียด: เตชินดึงมิลลี่ขึ้นนั่งบนโต๊ะทำงาน เอกสารกระจาย
     "เธอคือสิ่งเดียวที่ทำให้ฉันยังมีสติ" — จังหวะเร่าร้อน ดุดัน
     มิลลี่กัดริมฝีปากเพื่อไม่ให้เสียงดัง — เตชิน: "ที่นี่ไม่มีใครได้ยิน"

• ตอนที่ 17: ปมของเตชิน
  - เตชินเริ่มเปิดเผยอดีต: พ่อแม่ถูกฆ่าตอนเขา 10 ขวบ
  - ผู้เฒ่าวรพล (ลุง) เลี้ยงเขามา — แต่สอนให้เป็น "อาวุธ" ไม่ใช่ "คน"
  - เตชินไม่เคยไว้ใจใคร — จนกระทั่งมิลลี่
  - มิลลี่ตกใจ: ผู้เฒ่าวรพลมีแรงจูงใจมากพอที่จะเป็นคนทรยศ
  - **จุดพลิก:** มิลลี่รู้ตัวว่าเธอไม่ได้แค่อยากเปลี่ยนชะตาอีกต่อไป
    — เธอรักเตชินจริงๆ

• ตอนที่ 18: สัญญาณอันตราย
  - มิลลี่ถูกลอบทำร้าย — มีคนผลักเธอตกบันได
  - เตชินโกรธจัด สั่งค้นหาคนทำ — แต่ไม่มีหลักฐาน
