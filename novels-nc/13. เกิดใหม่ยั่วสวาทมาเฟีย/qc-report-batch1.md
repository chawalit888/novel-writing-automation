## QC ขั้นที่ 5: Thai Language Polish

### คำ/วลีที่ใช้ซ้ำบ่อย
| คำ/วลี | จำนวนครั้ง (5 ตอน) | ระดับ |
|--------|-------------------|-------|
| "กลืนน้ำลาย" | 6 ครั้ง (ตอน2,3,3,4,5,5) | **ซ้ำมาก** — ควรหาคำอื่นสลับ |
| "ดวงตาสีดำสนิท" | 8+ ครั้ง | **ซ้ำมาก** — เป็น motif แต่ควรมีวิธีเรียกอื่นบ้าง |
| "เสียงเรียบ" / "เสียงเรียบนิ่ง" | 7+ ครั้ง | **ซ้ำมาก** — ควรหาคำอื่นบรรยายน้ำเสียงเตชิน |
| "สายตาที่ทะลุทุกสิ่ง" | 3 ครั้ง | ปานกลาง — ยอมรับได้ |
| "หัวใจเต้นแรง" | 5+ ครั้ง | **ซ้ำมาก** — ควรใช้ variation |
| "ก้มหัว" | 5+ ครั้ง | ปานกลาง — สอดคล้องกับสถานะคนรับใช้ |

### คำแนะนำลดคำซ้ำ
- "กลืนน้ำลาย" → สลับกับ: "คอแห้งผาก" / "ลมหายใจติดขัด" / "คอตีบ"
- "ดวงตาสีดำสนิท" → สลับกับ: "ดวงตาคมเฉือน" / "สายตามืดดำ" / "ตาที่เย็นเฉียบ"
- "เสียงเรียบ" → สลับกับ: "เสียงนิ่ง" / "น้ำเสียงราบเรียบ" / "เสียงไม่มีอารมณ์"
- "หัวใจเต้นแรง" → สลับกับ: "ชีพจรเร่ง" / "เลือดสูบฉีดแรง" / "อกสั่น"

### คุณภาพภาษาโดยรวม
- **ดีมาก:** ภาษาทันสมัย อ่านลื่น ผสมบรรยากาศมาเฟียได้ดี
- **ดีมาก:** Dialogue เป็นธรรมชาติ เสียงตัวละครชัดเจน
- **ดีมาก:** Internal monologue มิลลี่เข้าถึงง่าย relatable
- **ดีมาก:** Short paragraphs สร้าง tension ได้ดี
- **ดี:** การใช้ "—" em dash เพื่อเน้นข้อมูลสำคัญทำงานได้ดี
