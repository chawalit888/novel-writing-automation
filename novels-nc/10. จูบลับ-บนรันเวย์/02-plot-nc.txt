[จุดเริ่มต้น - ตอนที่ 1-3]
• ตอนที่ 1: มาดเย็นปะทะสดใส
  - ลลิดา (ลิน) นางแบบอันดับ 1 ของไทย มาดเย็น สง่า สูงเพรียว
  - เธอเป็นราชินีรันเวย์ — ทุกคนเกรงกลัว ไม่มีใครกล้าเข้าใกล้
  - วันนี้: Fashion Week กรุงเทพ เตรียมโชว์ใหญ่
  - ดีไซเนอร์แขกรับเชิญจากปารีส: กัญญ์ณัฏฐ์ (แนท) หญิงสาวสวยสดใส กล้า แรง
  - แนทเข้ามาเปลี่ยนโชว์ทั้งหมด: "คอนเซปต์เดิมน่าเบื่อ ฉันมีไอเดียใหม่"
  - ลินโกรธ: "เธอเพิ่งมาจะมาเปลี่ยนทุกอย่าง?"
  - แนทยิ้ม: "ถ้าไม่เปลี่ยน มันก็ไม่มีอะไรน่าจดจำ...เหมือนหน้าเธอบนรันเวย์"
  - ปะทะกันตั้งแต่วินาทีแรก

• ตอนที่ 2: บนเวทีกับหลังเวที
  - ลินต้องเดินแบบในชุดที่แนทออกแบบ — เกลียดแต่ต้องทำ
  - แนทมาจัดชุดให้ลินเอง ใกล้ชิด สัมผัสตัว จัดผ้า จัดซิป
  - ลินรู้สึกแปลก: มือผู้หญิงคนนี้ทำไมอุ่นขนาดนี้
  - บนรันเวย์: ลินเดินสวยสะกดจนแนทหยุดหายใจ
  - หลังเวที: แนทบอก "เธอเดินสวยมาก...ฉันยอมรับ"
  - ลิน (หัวใจเต้นแรง แต่ทำหน้าเฉย): "ก็ชุดเธอไม่เลว"

• ตอนที่ 3: คู่แข่งที่ดึงดูด
  - แนทถูกจ้างเป็นดีไซเนอร์หลักของ Fashion Week ทั้ง season
  - ลินต้องทำงานกับแนทอีก 3 เดือน — ไม่มีทางหนี
  - ทั้งคู่ขัดแย้งเรื่องคอนเซปต์ เรื่องชุด เรื่องทุกอย่าง
  - แต่ทุกครั้งที่ทะเลาะ สายตาทั้งคู่จ้องกันนาน...นานเกินไป
  - ข่าวลือ: "ลินกับแนทไม่ถูกกัน" กลายเป็นกระแส

[Tension Build-up - ตอนที่ 4-6]
• ตอนที่ 4: ชุดที่ออกแบบเพื่อเธอ
  - แนทออกแบบชุดพิเศษ "เฉพาะ" สำหรับลิน
  - ชุดที่สวยที่สุดที่เธอเคยเห็น — เข้ารูป เปิดหลัง มีรายละเอียดเล็กๆ ที่แนทบอกว่า
    "ออกแบบจากเส้นโค้งของเธอ"
  - ลินลองชุด แนทมาช่วยซิปด้านหลัง นิ้วลากตามแนวกระดูกสันหลัง
  - ลินสั่น แต่ไม่ผลักออก
  - แนท (กระซิบข้างหู): "สวยมาก ลิน...ชุดนี้มีแค่ตัวเดียวในโลก เหมือนเธอ"

• ตอนที่ 5: เบื้องหลังเวที 🔥(เบา)
  - หลังรันเวย์โชว์ ในห้องแต่งตัว ลินถอดชุดอยู่
  - แนทเข้ามาโดยไม่เคาะ เห็นลินในชุดชั้นใน — ทั้งคู่แข็งไป
  - แนทไม่หันออก: "ไม่ต้องปิด ฉันเป็นดีไซเนอร์ เห็นมาเยอะ"
  - ลิน: "เธอไม่ใช่ดีไซเนอร์คนอื่น" (ไม่ได้หมายอะไร...หรือเปล่า)
  - แนทเดินเข้าไปช่วยปลดเครื่องประดับ ใกล้มากจนรู้สึกลมหายใจ
  - ลินจ้องตาแนทในกระจก ทั้งคู่มองกัน...แต่ไม่มีใครเอ่ยอะไร
  - [Sexual tension: 4/10]

• ตอนที่ 6: เส้นบางระหว่างเกลียดกับ...
  - ทะเลาะหนักเรื่องคอนเซปต์โชว์ใหญ่ ลินร้องไห้ (ไม่เคยร้องต่อหน้าใคร)
  - แนทตกใจ เข้าไปปลอบ กอดโดยสัญชาตญาณ
  - ลินผลักออก: "อย่ามายุ่งกับฉัน"
  - แนท: "ฉันยุ่ง เพราะฉันแคร์เธอ ลิน"
  - ลิน: "ทำไมแคร์ เราเป็นแค่คนทำงานด้วยกัน"
  - แนทจ้องนาน: "เธอรู้ดีว่ามันไม่ใช่แค่นั้น"
  - ลินหนีไป — แต่หัวใจเต้นแรงจนหยุดไม่ได้

[ข้ามเส้น - ตอนที่ 7-9]
• ตอนที่ 7: คืนก่อนโชว์ใหญ่
  - เตรียมงานดึก มีแค่ลินกับแนทในสตูดิโอ
  - แนทช่วยลินลองชุดเดินแบบชุดสุดท้าย — ชุดแต่งงาน
  - ลินสวยจนแนทหยุดหายใจ
  - แนท: "ถ้าฉันเป็นผู้ชาย ฉันจะขอเธอแต่งงานเลย"
  - ลิน (เสียงสั่น): "...แล้วถ้าเธอไม่ใช่ผู้ชายล่ะ ทำไมต้องเป็น"
  - ทั้งคู่มองกัน บรรยากาศเปลี่ยน

🔥 ฉาก NC แรก (ตอนที่ 8): จูบแรกในห้องลองชุด
   • Trigger: คืนก่อนโชว์ + สตูดิโอว่าง + ลินในชุดแต่งงาน + อารมณ์สะสม
   • ประเภท: [x] tender [x] passionate [ ] rough
   • ความเข้มข้น: 7/10
   • อารมณ์หลัก: ค้นพบ + ตื่นเต้น + อ่อนโยน + กลัว
   • รายละเอียด:
     - แนทเอื้อมมือเก็บผมลินหลังหู นิ้วลากช้าๆ
     - ลินไม่หลบ หลับตา หัวใจเต้นแรง
     - จูบแรก — เบา อ่อนโยน ริมฝีปากนุ่มสัมผัสกัน
     - แนทดึงตัวเข้ามา ลินกอดตอบ จูบลึกขึ้น
     - มือแนทลงไปตามแนวชุดแต่งงาน สัมผัสผิวเปลือย
     - ลินผลักเบาๆ: "แนท...นี่มัน..."
     - แนท: "บอกฉันว่าหยุด แล้วฉันจะหยุด"
     - ลิน: "..." (ไม่บอกให้หยุด)
   • หมายเหตุ: ฉากนี้เป็นจุดดึงดูดก่อน paywall — อ่อนโยนแต่เข้มข้น

• ตอนที่ 9: เช้าหลังจูบ
  - ลินไม่นอนทั้งคืน คิดเรื่องที่เกิดขึ้น
  - เจอกันที่โชว์ ทำเป็นปกติ แต่สายตาหลบกัน
  - แนทส่งข้อความ: "ฉันไม่เสียใจ แต่ถ้าเธอต้องการเวลา ฉันให้ได้"
  - ลินอ่านแล้วร้องไห้คนเดียว — ไม่ใช่เพราะเสียใจ แต่เพราะสับสน

[PAYWALL HOOK - ตอนที่ 10-12]
• ตอนที่ 10: โชว์ที่เปลี่ยนทุกอย่าง (PAYWALL HOOK)
  - Fashion Week โชว์ใหญ่ ลินเดินปิดโชว์ในชุดพิเศษของแนท
  - สวยจนคนทั้งงานลุกขึ้นปรบมือ
  - หลังเวที: แนทรอลินอยู่ ตาเป็นประกาย
  - ลินเดินเข้ามาตรง ไม่พูดอะไร — จูบแนทต่อหน้าทีมงานหลังเวที
  - ทุกคนช็อก แนทช็อก
  - ลิน: "ฉันไม่อยากซ่อนอีกแล้ว...แต่ฉันก็ยังไม่พร้อมบอกทั้งโลก"
  - CLIFFHANGER: แนทกอดลิน "เราค่อยๆ ไปด้วยกัน"

• ตอนที่ 11: ความลับของเรา
  - ทีมงานหลังเวทีรู้ แต่สัญญาจะเก็บความลับ
  - ลินกับแนทเริ่มคบกันลับๆ — หลังงาน ที่คอนโดแนท ในสตูดิโอดึกๆ
  - ลินเรียนรู้ที่จะเปิดใจ ถอดเกราะมาดเย็นออก
  - แนทเห็นลินอีกด้านที่ไม่มีใครเคยเห็น — ขี้อ้อน ตัวอุ่น เสียงหวาน

🔥 ฉาก NC (ตอนที่ 12): คืนหลังโชว์สำเร็จ
   • Trigger: โชว์ได้รับกระแสดีมาก + ฉลองสองคนที่คอนโด + ไวน์ + ความสุข
   • ประเภท: [x] tender [x] passionate [ ] rough
   • ความเข้มข้น: 8/10
   • อารมณ์หลัก: มีความสุข + ไว้ใจ + สำรวจกัน + ค้นพบ
   • รายละเอียด:
     - ดื่มไวน์ ฟังเพลง แนทวาดรูปลินบนสเก็ตช์บุ๊ค
     - ลินเข้ามาดูรูป — แนทวาดเธอตอนนอน สวยงาม
     - "เธอมองฉันแบบนี้จริงเหรอ" ลินถาม
     - "สวยกว่านี้อีก" แนทตอบก่อนจะดึงเข้ามา
     - ค่อยๆ อ่อนโยน ดูแล ทุกสัมผัสเต็มไปด้วยความรัก
     - ลินเปิดเผยว่าเคยคบผู้ชายแต่ไม่เคยรู้สึกแบบนี้: "กับเธอ...ต่างกันทุกอย่าง"

• ตอนที่ 13: ศัตรูในวงการ
  - มนัสนันท์ (มิ้น) นางแบบคู่แข่ง สงสัยว่าลินกับแนทมีอะไรกัน
  - แอบสังเกต แอบถ่ายรูป
  - มิ้นเคยเป็นนางแบบอันดับ 1 ก่อนลิน — อิจฉา ต้องการทำลาย

• ตอนที่ 14: ทริปปารีส
  - แนทพาลินไปปารีสในฐานะ "นางแบบเอก" ของคอลเลกชั่นใหม่
  - ปารีส: ได้อยู่ด้วยกันอย่างเปิดเผย (ไม่มีใครรู้จัก)
  - เดินมือกัน จูบริมแม่น้ำแซน กินอาหารฝรั่งเศส — โรแมนติก
  - ลินยิ้มมากกว่าที่เคย แนทมีความสุขที่สุด

🔥 ฉาก NC (ตอนที่ 15): ปารีส คืนที่ไม่ต้องซ่อน
   • Trigger: ปารีส + อิสระ + ไม่ต้องกลัวใครรู้ + ห้องพักโรงแรมหรู
   • ประเภท: [x] tender [x] passionate [ ] rough
   • ความเข้มข้น: 8/10
   • รายละเอียด:
     - แนท: "สอนฉันโพสท่าแบบนางแบบจริงๆ สิ"
     - ลิน: "บนเตียงไม่ต้องโพสท่า แนท"
     - แนทวาดเส้นบนร่างลินด้วยนิ้ว: "ทุกเส้นโค้งของเธอคือแรงบันดาลใจ"
     - อ่อนโยน สวยงาม ช้า เต็มไปด้วยสุนทรียะ
     - หน้าต่างเปิด แสงเมืองปารีสส่องเข้ามา
     - "ฉันรักเธอ ลลิดา" — ครั้งแรกที่แนทบอกรัก

• ตอนที่ 16: กลับมาเผชิญความจริง
  - กลับเมืองไทย ต้องซ่อนอีกครั้ง ลินเศร้า
  - แนทปลอบ: "สักวันเราจะไม่ต้องซ่อน"
  - มิ้นเริ่มมีหลักฐาน — ภาพจากปารีสที่เพื่อนส่งมา

• ตอนที่ 17: คนที่สาม
  - ธีรวัฒน์ (ทัช) นายแบบหนุ่ม มาจีบลิน อย่างเปิดเผย
  - สื่อจับคู่ "ลินทัช" เป็นคู่จิ้นของวงการ
  - แนทหึง แต่ทำอะไรไม่ได้ เพราะเป็นความลับ
  - ลินต้องเดินเคียงทัชในงาน ขณะที่แนทยืนมองจากหลังเวที — เจ็บ

🔥 ฉาก NC (ตอนที่ 18): หลังทะเลาะเรื่องหึง
   • Trigger: แนทหึงทัช + ทะเลาะ + ลินมาขอโทษ + สตูดิโอดึกๆ
   • ประเภท: [x] passionate [ ] tender [ ] rough
   • ความเข้มข้น: 8/10
   • รายละเอียด:
     - แนทนั่งออกแบบอยู่คนเดียว ลินมาหา
     - "ฉันมาทำงาน" ลินโกหก
     - "ตอนตี 2?" แนทไม่เชื่อ
     - ลิน: "ฉันมาหาเธอ เพราะฉันทนไม่ได้ที่เราทะเลาะกัน"
     - ร้อนแรง บนโต๊ะทำงานที่เต็มไปด้วยผ้าและสเก็ตช์
     - "ฉันไม่ได้เป็นของทัช ฉันเป็นของเธอ แนท"

• ตอนที่ 19: ภาพหลุด
  - มิ้นปล่อยภาพลินกับแนทจูบกันที่ปารีส ลงโซเชียล
  - กระแสระเบิด: "ราชินีรันเวย์คบหญิง!"
  - ค่ายลิน เรียกประชุม กดดันให้ปฏิเสธข่าว

• ตอนที่ 20: เลือกระหว่างอาชีพกับรัก
  - ค่ายเสนอ: ปฏิเสธข่าว + คบทัชเป็นการ PR
  - ลินเจ็บ — ถ้ายอม เท่ากับทำลายแนท
  - แนทบอก: "ถ้าเธอต้องเลือกอาชีพ ฉันเข้าใจ"
  - ลิน: "ฉันเคยเลือกงานมาตลอด ครั้งนี้...ฉันจะเลือกเธอ"

🔥 ฉาก NC (ตอนที่ 21): หึง — หลังเห็นทัชจีบลิน
   • Trigger: ทัชกอดลินต่อหน้า + แนทเจ็บ + ลินตามมาขอโทษ
   • ประเภท: [ ] tender [x] passionate [x] rough (เบาๆ)
   • ความเข้มข้น: 9/10
   • รายละเอียด:
     - แนทหึง ไม่รับสาย ลินมาหาที่คอนโด
     - "ฉันไม่ได้ชอบเขา เธอรู้" ลินอ้อนวอน
     - แนท: "ก็มันเจ็บ ลิน เวลาเห็นคนอื่นแตะเธอ"
     - ลิน: "งั้นทำให้ฉันลืมมือคนอื่น...ด้วยมือเธอ"
     - เข้มข้น แนทนำ ลินยอม — ทุกสัมผัสเหมือนบอกว่า "เธอเป็นของฉัน"

• ตอนที่ 22: ยืนหยัดด้วยกัน
  - ลินปฏิเสธข้อเสนอค่าย ไม่ยอมคบทัชแบบ PR
  - แนทปกป้องลินในสื่อ ออกมาพูดว่า "ความรักไม่จำกัดเพศ"
  - กระแสเริ่มเปลี่ยน — คนรุ่นใหม่สนับสนุน

• ตอนที่ 23: มิ้นถูกจัดการ
  - พบหลักฐานว่ามิ้นเป็นคนปล่อยภาพ
  - ลินเผชิญหน้ามิ้น ไม่ได้แก้แค้น แค่พูดว่า: "ฉันสงสารเธอ ที่ต้องทำลายคนอื่นเพื่อรู้สึกดี"
  - มิ้นถูกแบนจากวงการ
