                         PLOT เรื่อง
              "รุ่นพี่วิศวะ รักนี้คำนวณไม่ได้"
══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│                  ภาค 1: การพบกัน                           │
│                   (ตอนที่ 01-12)                           │
└─────────────────────────────────────────────────────────────┘

[จุดเริ่มต้น]
• ใบบุญ นักศึกษาใหม่คณะอักษรฯ หลงทางวันแรก
• ภูมิ รุ่นพี่ปี 4 วิศวะเครื่องกล ชี้ทางให้โดยบังเอิญ
• ใบบุญถูกส่งเป็นตัวแทนประกวดดาวมหา'ลัย
• เจอกันอีกครั้งในค่ายรับน้อง ภูมิเป็นหัวหน้าสตาฟ

[ความขัดแย้งแรก]
• ภูมิคิดว่าใบบุญเป็นแค่เด็กสวยที่ถือตัว
• ใบบุญพิสูจน์ตัวเองด้วยการลุยกิจกรรมไม่กลัวเลอะ
• ภูมิเริ่มประทับใจ เปลี่ยนมุมมอง
• พาไปดูดาวบนดาดฟ้าคืนสุดท้ายของค่าย

[จุดพลิกผัน]
• ภูมิช่วยใบบุญตอนหลงทางกลางคืน
• แลกไลน์กัน เริ่มคุยกันบ่อยขึ้น
• ทั้งคู่เริ่มมีความรู้สึก แต่ยังไม่ยอมรับ
• ก้องเพื่อนภูมิเริ่มแซว สังเกตเห็นว่าภูมิเปลี่ยนไป

┌─────────────────────────────────────────────────────────────┐
│                   ภาค 2: ใกล้ชิด                           │
│                   (ตอนที่ 13-24)                           │
└─────────────────────────────────────────────────────────────┘

[ความสัมพันธ์พัฒนา]
• ใบบุญได้งานเขียนสารคดี ขอสัมภาษณ์ภูมิ
• ภูมิพาเดินชมตึกวิศวะ พาดูห้องแล็บ
• ทั้งคู่กินข้าวด้วยกัน คุยกันสนิทขึ้น
• ใบบุญสอบแคลฯ ไม่ผ่าน ภูมิอาสาติวให้

[ช่วงเวลาหวานๆ]
• ติวหนังสือที่ห้องสมุด ใกล้ชิดมากขึ้น
• ภูมิมาเชียร์ใบบุญในงานกีฬาสี
• พาไปดูดาวบนดาดฟ้าตึกวิศวะอีกครั้ง
• เพื่อนทั้งสองฝ่ายเริ่มสังเกตเห็น

[รู้ตัวว่ารัก]
• ใบบุญยอมรับกับตัวเองว่าชอบภูมิ
• ภูมิก็ยอมรับกับก้องว่าชอบใบบุญ
• แต่ทั้งคู่ยังไม่กล้าบอก

[ปัญหาเริ่มก่อตัว]
• พิม รุ่นพี่วิศวะไฟฟ้า แอบชอบภูมิมานาน
• เห็นภูมิสนิทกับใบบุญ เริ่มไม่พอใจ
• พิมเริ่มเข้าหาภูมิบ่อยขึ้น

┌─────────────────────────────────────────────────────────────┐
│                   ภาค 3: อุปสรรค                           │
│                   (ตอนที่ 25-36)                           │
└─────────────────────────────────────────────────────────────┘

[ตัวร้ายเริ่มลงมือ]
• พิมมาหาภูมิทุกวัน เอาข้าว กาแฟมาให้
• ข่าวลือแพร่ว่าภูมิกับพิมเป็นแฟนกัน
• ใบบุญได้ยินข่าวลือ หัวใจสลาย

[ความลับแตก / All is Lost]
• ใบบุญเข้าใจผิด เริ่มหลบหลีกภูมิ
• ไม่ตอบข้อความ ยกเลิกนัดติว หนีเวลาเจอ
• ภูมิไม่เข้าใจ พยายามหาคำตอบ
• ก้องสืบเรื่องราว รู้ว่ามีข่าวลือ

[จุดต่ำสุด]
• วันประกวดดาวมหา'ลัย ภูมิไปเชียร์ ใบบุญชนะ
• ภูมิพยายามแสดงความยินดี แต่ถูกขัดจังหวะ
• พิมกอดแขนภูมิตอนใบบุญเดินผ่านมา
• ใบบุญเห็น น้ำตาคลอ วิ่งหนี ตัดสินใจลืม
• บล็อกไลน์ภูมิ ไม่ยอมพบ

┌─────────────────────────────────────────────────────────────┐
│                  ภาค 4: คลี่คลาย                           │
│                   (ตอนที่ 37-48)                           │
└─────────────────────────────────────────────────────────────┘

[ตัวละครพิสูจน์ตัวเอง]
• ภูมิไปเผชิญหน้ากับพิม รู้ว่าเธอตั้งใจทำ
• บอกพิมชัดเจนว่าชอบใบบุญ ไม่มีความรู้สึกกับพิม
• ก้องหาหลักฐาน - คลิปกล้องวงจรปิด
• ในคลิปเห็นว่าพิมเป็นฝ่ายเข้ามากอด ภูมิผลักออก

[พยายามอธิบาย]
• ภูมิพยายามไปหาใบบุญ แต่เธอไม่ยอมฟัง
• เขียนจดหมายบอกความจริงทั้งหมด
• แนบ USB คลิปวิดีโอเป็นหลักฐาน
• ฝากไว้หน้าหอพักใบบุญ

[ความจริงถูกเปิดเผย]
• ใบบุญอ่านจดหมาย ดูคลิป รู้ความจริง
• เสียใจที่เข้าใจผิด รีบไปหาภูมิที่ห้องแล็บ
• ขอโทษ ภูมิกอดให้อภัย
• ทั้งคู่คืนดี สารภาพรักกัน

[Happy Ending]
• เป็นแฟนกันอย่างเป็นทางการ
• ประกาศความสัมพันธ์ในงานลอยกระทง
• ภูมิให้สร้อยจี้ดาว บอกรัก
• Epilogue: หนึ่งปีต่อมา ภูมิจบปริญญา
• ทั้งคู่ยังรักกัน สัญญาว่าจะรอ

1. ความรักไม่จำกัดสถานะ - พระเอกหัวกะทิ นางเอกดาวมหา'ลัย
   แต่ทั้งคู่มองข้ามเรื่องพวกนี้ มองที่ตัวตนจริงๆ

2. ความไว้ใจและการสื่อสาร - ปัญหาเกิดจากข่าวลือและความเข้าใจผิด
   แก้ไขด้วยการพูดคุยและหลักฐาน

3. ความกล้าหาญในความรัก - ทั้งคู่ต้องกล้าที่จะเผชิญหน้า
   กล้าบอกความรู้สึก และกล้าให้อภัย
