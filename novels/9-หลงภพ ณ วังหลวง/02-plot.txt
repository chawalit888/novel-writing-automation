                         PLOT เรื่อง
              "หลงภพ ณ วังหลวง"
══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│                   องก์ที่ 1: หลงภพ                           │
│                    (ตอนที่ 1-10)                            │
└─────────────────────────────────────────────────────────────┘

[จุดเริ่มต้น - ตอน 1-3]
• "ปุณณ์" นักวิจัยประวัติศาสตร์สาว ค้นพบวัตถุโบราณลึกลับในวังเก่า
• เกิดอุบัติเหตุ แสงสว่างจ้า ตื่นขึ้นมาในร่าง "จันทร์แจ่ม" นางข้าหลวง
• รู้ตัวว่าอยู่ในสมัยรัตนโกสินทร์ตอนต้น ต้องเรียนรู้วิถีชีวิตในวัง

[พบพระเอก - ตอน 4-6]
• "เจ้าพระยาภูวเนตร" ขุนนางผู้มีอำนาจ สายตาเย็นเฉียบ
• จันทร์แจ่มทำผิดพลาด ถูกลงโทษ แต่ภูวเนตรเห็นความแปลกในตัวเธอ
• เขาสังเกตว่าเธอพูดจา/กิริยาไม่เหมือนคนในยุค

[ความขัดแย้งแรก - ตอน 7-10]
• "ท่านหญิงศศิธร" ธิดาเสนาบดี หมายปองภูวเนตร เริ่มจับตาจันทร์แจ่ม
• "คุณหญิงนวล" พี่สาวบุญธรรมของร่างเดิม อิจฉา คอยกลั่นแกล้ง
• จันทร์แจ่มใช้ความรู้ประวัติศาสตร์รอดพ้นอันตรายครั้งแรก

┌─────────────────────────────────────────────────────────────┐
│                 องก์ที่ 2: ชิงรักหักสวาท                      │
│                    (ตอนที่ 11-25)                           │
└─────────────────────────────────────────────────────────────┘

[ความสัมพันธ์พัฒนา - ตอน 11-15]
• ภูวเนตรเริ่มสนใจจันทร์แจ่ม เพราะเธอกล้าโต้แย้ง ไม่ก้มหัวเหมือนคนอื่น
• เขาแอบปกป้องเธอจากเล่ห์เหลี่ยมในวัง แต่ทำเป็นเมินเฉย
• จันทร์แจ่มเริ่มเข้าใจว่าความเย็นชาคือ "เกราะป้องกันตัว" จากบาดแผลอดีต
• มีฉากหวานๆ ที่พระเอกแสดงความห่วงใยแบบซ่อนเร้น

[ท่านหญิงศศิธรลงมือ - ตอน 16-20]
• ท่านหญิงศศิธรวางแผนใส่ร้ายจันทร์แจ่มหลายครั้ง
  - กล่าวหาว่าขโมยเครื่องประดับ
  - วางยาพิษในอาหาร
  - จับขังในห้องมืด
• คุณหญิงนวลร่วมมือกับท่านหญิงศศิธร
• จันทร์แจ่มรอดพ้นทุกครั้งด้วยไหวพริบและความรู้

[จุดหักมุม - ตอน 21-25]
• ภูวเนตรช่วยจันทร์แจ่มจากการถูกลงโทษอย่างเปิดเผย
• เขาประกาศรับเธอเป็น "นางบำเรอ" เพื่อปกป้อง
• สร้างศัตรูทั่ววัง ท่านหญิงศศิธรโกรธแค้นถึงขีดสุด
• ความลับของจันทร์แจ่มเริ่มถูกสงสัย

┌─────────────────────────────────────────────────────────────┐
│                 องก์ที่ 3: ความลับแตก                         │
│                    (ตอนที่ 26-32)                           │
└─────────────────────────────────────────────────────────────┘

[ความจริงถูกเปิดเผย - ตอน 26-28]
• ภูวเนตรรู้ความจริงว่าจันทร์แจ่มมาจากอนาคต
• เขาถาม "หากมีทางกลับ... เจ้าจะไปหรือไม่?"
• จันทร์แจ่มค้นพบว่าร่างเดิมของตนตายไปแล้ว - ไม่มีทางกลับ

[ท่านหญิงศศิธรรู้ความลับ - ตอน 29-32]
• ท่านหญิงศศิธรค้นพบความลับเรื่องข้ามภพ
• เธอกล่าวหาว่าจันทร์แจ่มเป็น "แม่มดผีสิง" ต่อหน้าผู้ใหญ่ในวัง
• ภูวเนตรต้องเลือกระหว่างตำแหน่ง/ตระกูล กับการปกป้องจันทร์แจ่ม
• เขาเลือกจันทร์แจ่ม ยอมสละทุกอย่าง

┌─────────────────────────────────────────────────────────────┐
│               องก์ที่ 4: รักแท้ข้ามกาลเวลา                    │
│                    (ตอนที่ 33-40)                           │
└─────────────────────────────────────────────────────────────┘

[การต่อสู้ครั้งสุดท้าย - ตอน 33-36]
• ภูวเนตรรวบรวมหลักฐานพิสูจน์ความบริสุทธิ์ของจันทร์แจ่ม
• เปิดโปงแผนร้ายของท่านหญิงศศิธรและคุณหญิงนวล
• จันทร์แจ่มใช้ความรู้ประวัติศาสตร์ช่วยพลิกสถานการณ์

[ไคลแมกซ์ - ตอน 37-38]
• ท่านหญิงศศิธรถูกลงโทษ
• คุณหญิงนวลได้รับการอภัยแต่ถูกส่งไปอยู่ต่างเมือง
• ภูวเนตรและจันทร์แจ่มผ่านพ้นวิกฤต

[ตอนจบ - ตอน 39-40]
• ทั้งคู่ออกจากวังหลวง ใช้ชีวิตเรียบง่ายด้วยกัน
• ภูวเนตรพูดประโยคแรกที่แสดงความรู้สึกตรงๆ:
  "ข้าไม่เคยรู้ว่าหัวใจข้ายังเต้นได้... จนกระทั่งพบเจ้า"
• Happy Ending - รักข้ามกาลเวลา
