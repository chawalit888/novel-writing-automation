──────────────────────────────────────────────────────────────
              พระเอก: ภาคิน วิชญ์ญาณี (วิญญาณ)
──────────────────────────────────────────────────────────────

ชื่อเล่น        : ภาคิน
อายุ (ตอนตาย)  : 29 ปี (ตายเมื่อ พ.ศ. 2468 - กว่า 100 ปีก่อน)
อาชีพ (ตอนมีชีวิต) : ทายาทตระกูลวิชญ์ญาณี นักธุรกิจหนุ่ม
สถานะ         : วิญญาณที่ติดอยู่ในคฤหาสน์

[รูปลักษณ์]
• สูง 182 ซม. รูปร่างสง่างามแบบผู้ดีสมัยก่อน
• ใบหน้าคมเข้ม คิ้วเข้ม ดวงตาสีดำสนิทลึกราวกับดูดวิญญาณ
• ผมสีดำเงา หวีเสยไปด้านหลังแบบสุภาพบุรุษยุคเก่า
• มักปรากฏในชุดสูทสีดำโบราณ บางครั้งโปร่งแสง บางครั้งชัดเจน
• มีรอยแผลเป็นจางๆ ที่หน้าอก (รอยมีด - สาเหตุการตาย)

[นิสัย]
• สุขุม เยือกเย็น พูดน้อยแต่ได้ใจความ
• อ่อนโยนลึกๆ แม้ภายนอกจะดูเย็นชา
• ซื่อสัตย์ รักษาสัญญาเสมอ
• หึงหวงเงียบๆ ไม่แสดงออกตรงๆ
• เศร้าซ่อนลึก จากความเจ็บปวดในอดีต

[ภูมิหลัง]
• เกิดในตระกูลเศรษฐีใหญ่ เป็นน้องชายคนเล็ก
• มีพี่ชายชื่อ ภาณุ ที่ริษยาเขามาตลอด
• ตกหลุมรักหญิงสาวคนหนึ่งชื่อ "วารี" แต่พี่ชายก็หมายปองเธอเช่นกัน
• ถูกพี่ชายฆาตกรรมในคืนที่จะประกาศหมั้นกับวารี
• วิญญาณติดอยู่ในคฤหาสน์เพราะความทรงจำที่ขาดหาย ไม่รู้ว่าตัวเองตายอย่างไร

[ความลับ]
• เขาจำไม่ได้ว่าตัวเองถูกฆ่าตาย คิดว่าตัวเองฆ่าตัวตาย
• วารีคือบรรพบุรุษของธารา - นี่คือเหตุผลที่เขารู้สึกผูกพันกับเธอ

[จุดเปลี่ยน]
• พบธาราทำให้เขารู้สึก "มีชีวิต" อีกครั้งหลังจากเป็นวิญญาณมานาน
• ค้นพบความจริงเรื่องการถูกฆาตกรรม ต้องเผชิญหน้ากับพี่ชาย
• เรียนรู้ที่จะให้อภัยและปล่อยวาง เพื่อไปสู่ภพหน้า

──────────────────────────────────────────────────────────────
                   นางเอก: ธารา ธาราทิพย์
──────────────────────────────────────────────────────────────

ชื่อเล่น        : ธารา
อายุ           : 28 ปี
อาชีพ          : นักจิตวิทยาคลินิก
สถานะ         : โสด

[รูปลักษณ์]
• สูง 165 ซม. รูปร่างบอบบาง
• ใบหน้าสวยใส ดวงตากลมโต มีความลึกลับซ่อนอยู่
• ผมยาวสีดำ มักรวบมัดเรียบร้อย
• แต่งตัวเรียบง่าย สุภาพ เน้นความสบาย
• มีไฝเล็กๆ ใต้ตาซ้าย (เหมือนวารี บรรพบุรุษของเธอ)

[นิสัย]
• เข้มแข็ง กล้าหาญ ไม่ยอมแพ้ง่ายๆ
• ใจดี ชอบช่วยเหลือคนอื่น
• ฉลาด ช่างสังเกต วิเคราะห์เก่ง
• เก็บความรู้สึกไว้ข้างใน ไม่ค่อยเปิดใจให้ใคร
• กลัวการถูกมองว่าแปลก เพราะพลังพิเศษของเธอ

[ภูมิหลัง]
• พ่อแม่เสียตั้งแต่เด็ก เติบโตในสถานสงเคราะห์
• มีพลังเห็นวิญญาณตั้งแต่เด็ก ทำให้ถูกเด็กคนอื่นรังเกียจ
• พยายามซ่อนพลังพิเศษ เรียนจิตวิทยาเพื่อหาคำตอบให้ตัวเอง
• ทำงานช่วยผู้ป่วยที่มีประสบการณ์เหนือธรรมชาติ

[ความลับ]
• เธอเป็นทายาทสายเลือดตระกูลวิชญ์ญาณี
• พลังพิเศษของเธอสืบทอดมาจากตระกูล
• วารี (คนรักเก่าของภาคิน) คือบรรพบุรุษของเธอ

[จุดเปลี่ยน]
• พบภาคินทำให้เธอยอมรับพลังพิเศษของตัวเอง
• รักภาคินทำให้เธอเข้าใจว่าความรักไม่มีขอบเขต
• เรียนรู้ที่จะปล่อยวางคนที่รัก และเชื่อมั่นในความรักแท้

──────────────────────────────────────────────────────────────
                   นักล่าผี: วิชญ์ เมธาวี
──────────────────────────────────────────────────────────────

ชื่อเล่น        : วิชญ์
อายุ           : 32 ปี
อาชีพ          : นักล่าผี / นักวิจัยเรื่องเหนือธรรมชาติ

[รูปลักษณ์]
• สูง 180 ซม. รูปร่างกำยำ แข็งแรง
• ใบหน้าคมเข้มแบบลูกครึ่ง ดวงตาสีน้ำตาลอ่อน
• ผมสีน้ำตาลเข้ม สั้นเท่าต้นคอ ดูเป็นทางการ
• มักใส่เสื้อโค้ทยาวสีดำ มีอุปกรณ์ล่าผีติดตัวเสมอ

[นิสัย]
• จริงจัง มุ่งมั่น ทุ่มเทกับงาน
• มีเสน่ห์ พูดจาดี
• ซ่อนความอ่อนโยนไว้ใต้ท่าทีเข้มแข็ง
• เสียสละ พร้อมช่วยเหลือคนอื่นแม้ต้องเจ็บตัว

[บทบาทในเรื่อง]
• มาสืบสวนคฤหาสน์ สัมผัสถึงพลังวิญญาณ
• สนใจธาราและพลังพิเศษของเธอ เป็นมุมรักสามเหลี่ยม
• ต้องเลือกระหว่างหน้าที่ (ทำลายวิญญาณ) กับมิตรภาพ (ช่วยภาคิน)
• เสียสละพลังพิเศษของตัวเองเพื่อเปิดประตูมิติ

──────────────────────────────────────────────────────────────
                  ตัวร้าย: ภาณุ วิชญ์ญาณี (วิญญาณร้าย)
──────────────────────────────────────────────────────────────

ชื่อเล่น        : ภาณุ
อายุ (ตอนตาย)  : 33 ปี
สถานะ         : ปีศาจ / วิญญาณร้าย

[รูปลักษณ์]
• หน้าตาคล้ายภาคินแต่แก่กว่า ดูเข้มและดุดันกว่า
• ตาสีแดงเรืองแสง (เมื่อโกรธ)
• ร่างบิดเบี้ยวน่ากลัวเมื่อกลายเป็นปีศาจ

[นิสัย/แรงจูงใจ]
• ริษยาน้องชายที่ได้รับความรักจากพ่อมากกว่า
• หมกมุ่นกับวารี ผู้หญิงที่เขาหมายปองแต่เธอรักภาคิน
• ถูกความโกรธและความผิดครอบงำ กลายเป็นปีศาจหลังความตาย

[บทบาทในเรื่อง]
• ฆาตกรที่ฆ่าภาคินเมื่อ 100 ปีก่อน
• วิญญาณร้ายที่หลอกหลอนคฤหาสน์
• ตื่นขึ้นเมื่อมีคนมาขุดคุ้ยความจริง พยายามหยุดธาราและภาคิน
• ในที่สุดได้รับการอภัยจากภาคิน และไปสู่สุคติ

──────────────────────────────────────────────────────────────
                  ตัวละครสำคัญ: ลุงณัฐพงษ์ (เสียชีวิตแล้ว)
──────────────────────────────────────────────────────────────

อายุ (ตอนตาย)  : 75 ปี
อาชีพ          : เจ้าของคฤหาสน์คนก่อน

[บทบาทในเรื่อง]
• ยกมรดกคฤหาสน์ให้ธารา
• รู้เรื่องพลังพิเศษของธารา ตามดูเธอมาตลอด
• ทิ้งจดหมายและเบาะแสไว้ให้เธอ
• เป็นคนเชื่อมโยงธารากับตระกูลวิชญ์ญาณี

──────────────────────────────────────────────────────────────
              ร่างกลับชาติมาเกิด: ภูมิ (ตอนท้ายเรื่อง)
──────────────────────────────────────────────────────────────

ชื่อเล่น        : ภูมิ
อายุ           : 28 ปี
อาชีพ          : จิตรกร

[รูปลักษณ์]
• หน้าตาเหมือนภาคินทุกประการ
• แต่มีความสดใส มีชีวิตชีวามากกว่า

[บทบาทในเรื่อง]
• ร่างกลับชาติมาเกิดของภาคิน
• ไม่มีความทรงจำก่อนอายุ 20 ปี
• เมื่อพบธารา เริ่มฝันเห็นชีวิตในอดีต
• แต่งงานกับธารา ทำให้คำสัญญารักข้ามภพเป็นจริง
