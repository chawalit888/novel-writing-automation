# Character Creator Skill

## Overview
Skill สำหรับสร้างตัวละครที่มีความลึกและน่าสนใจ

## Usage
```
"สร้างตัวละครหลักสำหรับนิยายแฟนตาซี"
"สร้างตัวร้ายที่ซับซ้อน"
"สร้าง supporting characters 3 คน"
```

## Character Types

### 1. Protagonist (ตัวละครหลัก)
- มีเป้าหมายชัดเจน
- มีจุดอ่อนที่ต้องเอาชนะ
- มี character arc
- ผู้อ่านต้อง root for

### 2. Antagonist (ตัวร้าย)
- ไม่ใช่แค่ "ชั่ว"
- มีเหตุผลของตัวเอง
- เป็นภาพสะท้อนของ protagonist
- ท้าทาย protagonist

### 3. Supporting Characters
- มี role ชัดเจน
- ไม่ใช่แค่ filler
- ช่วยพัฒนา protagonist
- มีความเป็นตัวเอง

### 4. Love Interest
- Chemistry กับ protagonist
- มี agency ของตัวเอง
- ไม่ใช่แค่รางวัล
- ช่วย protagonist grow

## Character Components

### 1. Basic Info
- ชื่อ (มีความหมาย/เหตุผล)
- อายุ
- เพศ
- รูปลักษณ์ภายนอก
- อาชีพ/สถานะ

### 2. Personality
- บุคลิก 3-5 traits หลัก
- จุดแข็ง (Strengths)
- จุดอ่อน (Flaws)
- Quirks/นิสัยเฉพาะ

### 3. Background
- ครอบครัว/ต้นกำเนิด
- เหตุการณ์สำคัญในอดีต
- Trauma/แผลใจ
- ความลับ

### 4. Motivation
- Want (สิ่งที่คิดว่าต้องการ)
- Need (สิ่งที่ต้องการจริงๆ)
- Fear (ความกลัว)
- Lie (สิ่งที่เชื่อผิดๆ)

### 5. Relationships
- ครอบครัว
- เพื่อน
- ศัตรู
- ความรัก

### 6. Voice
- วิธีพูด (Speech pattern)
- คำที่ใช้บ่อย
- สิ่งที่ไม่พูด
- Body language

### 7. Arc (สำหรับ main characters)
- จุดเริ่มต้น
- Catalyst
- การเปลี่ยนแปลง
- จุดจบ

## Character Creation Process

### Step 1: Concept
- Role ในเรื่อง
- Theme ที่ represent
- ความสัมพันธ์กับตัวละครอื่น

### Step 2: Core
- Want vs Need
- Main flaw
- Central conflict

### Step 3: Details
- Background
- Personality traits
- Physical appearance

### Step 4: Voice
- Speech patterns
- Mannerisms
- Reactions

### Step 5: Validate
- ขัดแย้งกับตัวละครอื่นไหม?
- Support theme ไหม?
- มี agency ไหม?
- น่าสนใจไหม?

## Common Mistakes

### ❌ Mary Sue/Gary Stu
- เก่งเกินไป ไม่มีจุดอ่อนจริงๆ
**แก้:** ให้ flaw ที่มีผลกระทบต่อเรื่อง

### ❌ One-Dimensional
- มีแค่ trait เดียว
**แก้:** เพิ่ม contradictions และ complexity

### ❌ No Agency
- ถูกเรื่องพัดพาไป ไม่ตัดสินใจ
**แก้:** ให้ตัวละครเลือก และรับผลที่ตามมา

### ❌ Info Dump Background
- เล่าประวัติทั้งหมดตอนแนะนำ
**แก้:** Reveal ค่อยๆ ผ่าน action และ dialogue

### ❌ Inconsistent
- ทำตัวขัดกับบุคลิกที่ตั้งไว้
**แก้:** Reference character sheet เสมอ

## Genre-Specific Tips

### Fantasy
- Power level ต้องชัดเจน
- มี room to grow
- Background เชื่อมกับ world

### Horror
- มี vulnerability
- Relatable fears
- Smart but not genre-savvy

### Romance
- Chemistry ต้องมี
- Complementary traits
- Individual goals

### Mystery
- มี secrets
- Believable motives
- Distinct alibis

## Checklist

- [ ] ชื่อมีความหมาย/เหตุผล
- [ ] มี want และ need ที่ต่างกัน
- [ ] มี flaw ที่มีผลต่อเรื่อง
- [ ] มี background ที่ shape บุคลิก
- [ ] มี distinct voice
- [ ] มี arc (ถ้าเป็น main character)
- [ ] ไม่ซ้ำกับตัวละครอื่น
- [ ] Support theme ของเรื่อง
