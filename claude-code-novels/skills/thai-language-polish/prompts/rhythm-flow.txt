## หลักการจังหวะ

### 1. ความยาวประโยคสื่ออารมณ์

```
ประโยคสั้น = เร็ว, ตึงเครียด, ตกใจ
"หยุด"
"เงียบ"
"มันมา"

ประโยคยาว = ช้า, ผ่อนคลาย, บรรยาย
"แสงอาทิตย์ยามเย็นทอดยาวผ่านหน้าต่าง
สาดส่องลงบนใบหน้าของเธอ
ทำให้ผิวขาวนวลเหลืองอ่อนเหมือนน้ำผึ้ง"
```

### 2. รูปแบบจังหวะ

#### Fast Rhythm (เร็ว)
```
วิ่ง
หายใจ
เร็วกว่านี้
ไม่ทัน
เร็ว
วิ่ง
```

#### Slow Rhythm (ช้า)
```
เธอนั่งลงริมหน้าต่าง
มองฝนตกพรำๆ
หยดน้ำไหลบนกระจก ช้าๆ
เหมือนเวลาที่ไม่เคยเร่งรีบ
```

#### Building Rhythm (ค่อยๆ เพิ่ม)
```
เขาเดิน
เดินเร็วขึ้น
และเร็วขึ้น
จนกลายเป็นวิ่ง
วิ่งสุดแรง
วิ่งจนหายใจไม่ทัน
```

#### Falling Rhythm (ค่อยๆ ลด)
```
เสียงปืนดังก้อง
แล้วก็เงียบ
เงียบสนิท
มีเพียงลมหายใจ
และความเงียบ
```

## การใช้เครื่องหมาย

### จุด (.)
```
สร้างการหยุด ให้ผู้อ่านหายใจ
ใช้มาก = จังหวะช้า, หนักแน่น
```

### จุลภาค (,)
```
เชื่อมต่อ ไหลต่อเนื่อง
ใช้มาก = ไหลลื่น, เร็ว
```

### ไข่ปลา (...)
```
สร้างความค้างคา, ลังเล
"เธอ... ไม่รู้จะพูดอะไร"
```

### ขีดกลาง (—)
```
ตัดบท, เปลี่ยนทิศ
"ฉันจะบอก—"
"ไม่ต้อง"
```

## จังหวะตามประเภทฉาก

### Action Scene
```
สั้น เร็ว กระชับ

ล้ม
ลุก
วิ่ง
กระโดด
หลบ
—ไม่ทัน
```

### Romantic Scene
```
ยาวปานกลาง ไหลลื่น

มือเขาสัมผัสมือเธอ เบาๆ
ราวกับกลัวเธอจะหายไป
เธอไม่ขยับ ปล่อยให้สัมผัสนั้นอยู่
อุ่นๆ บนผิว
```

### Suspense Scene
```
สลับสั้นยาว สร้าง tension

เสียงนั้น
มันมาจากไหน?
เธอหันไปมอง ห้องว่างเปล่า ไม่มีอะไร
แต่เสียงนั้น...
อีกแล้ว
ใกล้ขึ้น
```

### Emotional Scene
```
จังหวะไม่สม่ำเสมอ เหมือนอารมณ์

"ทำไม"

เธอถาม
แต่เขาไม่ตอบ

"ทำไม?"

เสียงดังขึ้น สั่น

"ทำไม!"

น้ำตาไหล
```
