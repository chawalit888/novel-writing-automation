# Word Choice Prompt (การเลือกใช้คำ)

## Context Variables
- {{context}}: บริบทของฉาก
- {{emotion}}: อารมณ์ที่ต้องการสื่อ
- {{character}}: ตัวละครที่เกี่ยวข้อง
- {{intensity}}: ระดับความเข้มข้น (1-5)

---

## Prompt Template

คุณคือผู้เชี่ยวชาญด้านการเลือกใช้คำภาษาไทย หาคำที่แม่นยำและสื่อความหมายได้ดีที่สุด

### ข้อมูล:
- บริบท: {{context}}
- อารมณ์: {{emotion}}
- ตัวละคร: {{character}}
- ความเข้มข้น: {{intensity}}/5

## คลังคำตามอารมณ์

### ความโกรธ (Anger)

| ระดับ | คำ |
|------|-----|
| 1 | หงุดหงิด, รำคาญ, ขุ่นเคือง |
| 2 | โมโห, โกรธ, ไม่พอใจ |
| 3 | เดือดดาล, พิโรธ, โกรธจัด |
| 4 | คลั่ง, บ้าคลั่ง, เกรี้ยวกราด |
| 5 | เลือดเดือด, บ้าเลือด, อาละวาด |

**กริยาแสดงความโกรธ:**
```
กำหมัด, กัดฟัน, ตวาด, ตะคอก, กระชาก, ฟาด, เหวี่ยง
```

### ความเศร้า (Sadness)

| ระดับ | คำ |
|------|-----|
| 1 | เหงา, ว้าเหว่, หดหู่ |
| 2 | เศร้า, โศกเศร้า, เศร้าใจ |
| 3 | ทุกข์, ระทม, เจ็บปวด |
| 4 | ร้าวราน, แหลกสลาย, พังทลาย |
| 5 | สิ้นหวัง, ตรอมใจ, หัวใจแตกสลาย |

**กริยาแสดงความเศร้า:**
```
ร้องไห้, สะอื้น, ครวญ, ถอนหายใจ, ก้มหน้า, หลบตา
```

### ความกลัว (Fear)

| ระดับ | คำ |
|------|-----|
| 1 | กังวล, ไม่สบายใจ, ระแวง |
| 2 | กลัว, หวาดกลัว, ตกใจ |
| 3 | หวาดผวา, ขนลุก, ขวัญเสีย |
| 4 | ตัวสั่น, ขวัญหาย, หน้าซีด |
| 5 | หวาดกลัวจนแข็ง, ขวัญหนีดีฝ่อ |

**กริยาแสดงความกลัว:**
```
สะดุ้ง, หดตัว, ถอยหลัง, กลืนน้ำลาย, หายใจติดขัด
```

### ความรัก (Love)

| ระดับ | คำ |
|------|-----|
| 1 | ชอบ, สนใจ, ประทับใจ |
| 2 | หลงใหล, ชื่นชอบ, ติดใจ |
| 3 | รัก, หลงรัก, คลั่งไคล้ |
| 4 | รักสุดหัวใจ, รักหมดใจ |
| 5 | รักจนตาย, รักข้ามภพ |

**กริยาแสดงความรัก:**
```
จ้องมอง, สัมผัส, โอบกอด, กระซิบ, ลูบไล้
```

### ความสุข (Happiness)

| ระดับ | คำ |
|------|-----|
| 1 | พอใจ, สบายใจ, โล่งใจ |
| 2 | ดีใจ, ยินดี, เบิกบาน |
| 3 | มีความสุข, ปลื้มใจ, ปิติ |
| 4 | ร่าเริง, สุขล้น, ปีติยินดี |
