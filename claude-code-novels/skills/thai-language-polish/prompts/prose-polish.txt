# Prose Polish Prompt (ขัดเกลาร้อยแก้ว)

## Context Variables
- {{text}}: ข้อความที่ต้องการขัดเกลา
- {{style}}: สไตล์ที่ต้องการ (เรียบง่าย, วรรณศิลป์, ทันสมัย)
- {{tone}}: โทน (ซีเรียส, เบาสบาย, ดราม่า)
- {{pov}}: มุมมอง (first person, third person)

---

## Prompt Template

คุณคือบรรณาธิการภาษาไทยมืออาชีพ ขัดเกลาข้อความให้สละสลวยและอ่านลื่น

### ข้อมูล:
- สไตล์: {{style}}
- โทน: {{tone}}
- มุมมอง: {{pov}}

### ข้อความที่ต้องขัดเกลา:
{{text}}

---

## หลักการขัดเกลา

### 1. ตัดคำฟุ่มเฟือย

**ตัดออก:**
```
- "ที่จะ" → ตัดออก
- "ซึ่ง" → ตัดถ้าไม่จำเป็น
- "เป็นอย่างมาก" → "มาก"
- "มีความ" → ตัดออก
- "ในเรื่องของ" → ตัดออก
- "อย่างไรก็ตาม" → ตัดหรือใช้คำอื่น
```

### 2. รวมประโยค

```
ก่อน:
"เขาเดินไปที่ประตู เขาเปิดประตู เขาเห็นเธอยืนอยู่"

หลัง:
"เขาเปิดประตู เห็นเธอยืนอยู่"
```

### 3. Active Voice

```
ก่อน:
"ความจริงถูกเปิดเผยโดยเธอ"

หลัง:
"เธอเปิดเผยความจริง"
```

### 4. แสดง ไม่บอก

```
ก่อน:
"เธอรู้สึกโกรธมาก"

หลัง:
"เธอกำมือแน่น นิ้วขาวซีด"
```

---

## ระดับการขัดเกลา

### Level 1: Light Polish (เบา)
```
- แก้คำผิด
- ตัดคำซ้ำซ้อนชัดเจน
- แก้ประโยคที่อ่านไม่รู้เรื่อง
```

### Level 2: Medium Polish (ปานกลาง)
```
- Level 1 +
- ปรับโครงสร้างประโยค
- เลือกคำที่ดีกว่า
- ปรับจังหวะ
```

### Level 3: Deep Polish (ลึก)
```
- Level 2 +
- ปรับสไตล์ทั้งหมด
- เพิ่มความไพเราะ
- สร้าง rhythm
```

```
3. [การเปลี่ยนแปลง 3]

### Word Count
- ก่อน: [จำนวน]

- ลดลง: [เปอร์เซ็นต์]
```

## ตัวอย่าง

### Input:
```
เธอรู้สึกว่าตัวเองมีความรู้สึกกลัวเป็นอย่างมาก
เธอไม่สามารถที่จะขยับตัวได้เลย
เธอได้ยินเสียงฝีเท้าที่กำลังเดินเข้ามาใกล้มากขึ้นเรื่อยๆ
และเธอก็รู้ว่ามันกำลังจะมาถึงแล้ว
```

### Output:
```
กลัวจนขยับไม่ได้
เสียงฝีเท้าใกล้เข้ามาทุกที
มันมาแล้ว
```

### Changes:
1. "รู้สึกว่าตัวเองมีความรู้สึกกลัว" → "กลัว"
2. "ไม่สามารถที่จะขยับตัวได้เลย" → "ขยับไม่ได้"
3. รวม 4 ประโยค → 3 ประโยคสั้น สร้าง tension
4. ตัดคำเชื่อม "และเธอก็รู้ว่า"
