# Thai Language Polish Skill

## Overview
Skill สำหรับขัดเกลาภาษาไทย ให้สำนวนสละสลวย อ่านลื่นไหล เป็นธรรมชาติ

---

## ความสามารถหลัก

### 1. Prose Polish (ขัดเกลาร้อยแก้ว)
- ตัดคำซ้ำซ้อน
- ปรับประโยคให้กระชับ
- เพิ่มความไพเราะ

### 2. Word Choice (การเลือกใช้คำ)
- คำที่เหมาะกับบริบท
- คำที่สื่ออารมณ์
- หลีกเลี่ยงคำซ้ำ

### 3. Rhythm & Flow (จังหวะและลีลา)
- ความยาวประโยค
- จังหวะการอ่าน
- การเว้นวรรค

---

## ปัญหาภาษาที่พบบ่อย

### 1. คำซ้ำซ้อน
```
❌ แย่:
"เธอรู้สึกมีความสุขและดีใจมาก"
"เขาเดินเดินไปเดินมา"
"สิ่งที่สำคัญที่สุดก็คือความรัก"

✅ ดี:
"เธอดีใจมาก"
"เขาเดินวนไปมา"
"สิ่งสำคัญคือความรัก"
```

### 2. ประโยคยาวเกินไป
```
❌ แย่:
"เธอเดินไปที่ร้านกาแฟที่อยู่ตรงหัวมุมถนนซึ่งเป็นร้านที่เธอชอบมา
เพราะกาแฟที่นี่อร่อยและบรรยากาศดีและพนักงานก็น่ารัก"

✅ ดี:
"เธอเดินไปร้านกาแฟหัวมุมถนน ร้านที่เธอชอบ
กาแฟอร่อย บรรยากาศดี พนักงานน่ารัก"
```

### 3. ภาษาแข็งเกินไป
```
❌ แย่:
"เธอรู้สึกโกรธ เธอต้องการที่จะตะโกน"
"เขามีความรู้สึกรักเธอ"

✅ ดี:
"โกรธจนอยากตะโกน"
"เขารักเธอ"
```

### 4. คำเชื่อมมากเกินไป
```
❌ แย่:
"และแล้วเขาก็เดินไปและเปิดประตูและเห็นเธอ"

✅ ดี:
"เขาเดินไปเปิดประตู เห็นเธอยืนอยู่"
```

### 5. Passive Voice ที่ไม่จำเป็น
```
❌ แย่:
"หนังสือถูกอ่านโดยเธอ"
"ความจริงถูกเปิดเผย"

✅ ดี:
"เธออ่านหนังสือ"
"ความจริงปรากฏ"
```

## หลักการขัดเกลา

### 1. ตัดคำที่ไม่จำเป็น

| ตัดออกได้ | ตัวอย่าง |
|----------|---------|
| "ที่จะ" | รู้สึก~~ที่จะ~~โกรธ → รู้สึกโกรธ |
| "ซึ่ง" (ถ้าไม่จำเป็น) | ร้าน~~ซึ่ง~~อยู่ใกล้ → ร้านใกล้ๆ |
| "เป็นอย่างมาก" | ดีใจ~~เป็นอย่างมาก~~ → ดีใจมาก |
| "มีความ" | ~~มีความ~~รู้สึก → รู้สึก |
| "ในที่สุด" (ถ้าชัดอยู่แล้ว) | - |

### 2. เปลี่ยนคำให้แม่นยำ

| แทนที่ | ด้วย (ตามบริบท) |
|-------|----------------|
| ดี | วิเศษ, เยี่ยม, ยอด |
| ไม่ดี | แย่, ย่ำแย่, เลว |
| มาก | ล้น, ท่วม, เหลือเกิน |
| เดิน | ย่ำ, ก้าว, เยื้องย่าง |
| พูด | กระซิบ, ตะโกน, บ่น |
| มอง | จ้อง, เหลือบ, เพ่ง |

### 3. สร้างจังหวะ

```
ประโยคสั้นสลับยาว:

"เธอหยุด นิ่ง
ลมพัดผ่านใบหน้า พาความเย็นมาสัมผัสผิว
เธอรู้
รู้ว่าเขาอยู่ที่นี่"
```

## ระดับความเป็นทางการ

### 1. ภาษาทางการ (Formal)
```
ใช้: รายงาน, จดหมาย, สุนทรพจน์
- ข้าพเจ้า, ท่าน, กระผม
- ประโยคสมบูรณ์
- ไม่ใช้คำย่อ/แสลง
```

### 2. ภาษากึ่งทางการ (Semi-formal)
```
ใช้: นิยายทั่วไป, บทสนทนาปกติ
- ฉัน, เธอ, คุณ
- ประโยคปานกลาง
- ใช้คำย่อบ้าง
```

### 3. ภาษาไม่ทางการ (Informal)
```
ใช้: บทสนทนาเพื่อน, ความคิดในใจ
- กู, มึง, เรา
- ประโยคสั้น ไม่สมบูรณ์
- แสลง/คำย่อ
```

## สำนวนภาษาไทย

### การใช้สำนวนให้เหมาะสม

```
สถานการณ์: กลัวมาก
❌ "เขากลัวมากๆ"
✅ "หัวใจเต้นตุ้บๆ" / "ขนลุก" / "ตัวสั่น"

สถานการณ์: โกรธมาก
❌ "เธอโกรธมากจนอยากตบ"
✅ "เลือดขึ้นหน้า" / "กำมือแน่น" / "ฟันกัด"

สถานการณ์: เศร้า
❌ "เขาเศร้ามาก"
✅ "หัวใจหนักอึ้ง" / "ตาร้อน" / "คอตีบ"
```

## ตัวอย่างการขัดเกลา

### Before & After

**ก่อน:**
```
เธอรู้สึกว่าตัวเองมีความรักต่อเขาเป็นอย่างมาก
และเธอก็รู้สึกว่าเขาก็น่าจะมีความรักต่อเธอเหมือนกัน
เธอจึงตัดสินใจที่จะบอกความรู้สึกของเธอให้เขาฟัง
```

**หลัง:**
```
เธอรักเขา
และเชื่อว่าเขาก็รักเธอเช่นกัน
ถึงเวลาบอกแล้ว
```

---

**ก่อน:**
```
เขาเดินเข้าไปในห้องซึ่งมืดมากเป็นอย่างยิ่ง
เขาไม่สามารถมองเห็นอะไรได้เลย
เขารู้สึกกลัวมากเพราะไม่รู้ว่ามีอะไรอยู่ข้างหน้า
```

**หลัง:**
```
เขาก้าวเข้าไปในความมืด
มองไม่เห็นอะไร
หัวใจเต้นแรง ไม่รู้ว่าอะไรรอข้างหน้า
```

## Checklist

- [ ] ไม่มีคำซ้ำซ้อน
- [ ] ประโยคกระชับ
- [ ] คำที่ใช้แม่นยำ
- [ ] จังหวะเหมาะสม
- [ ] ระดับภาษาสม่ำเสมอ
- [ ] ไม่มี Passive Voice ที่ไม่จำเป็น
- [ ] อ่านออกเสียงแล้วลื่น
- [ ] สื่ออารมณ์ได้

---

## Tips

### DO:
- อ่านออกเสียง ถ้าสะดุด = ต้องแก้
- ใช้คำกริยาแทนคำคุณศัพท์ถ้าได้
- ตัดคำที่ไม่เพิ่มความหมาย
- ใช้ความเงียบ (จุด) แทนคำเชื่อม

### DON'T:
- ใช้คำหรูเกินไปจนไม่เป็นธรรมชาติ
- เปลี่ยนสไตล์กลางทาง
- ขัดจนสูญเสียเอกลักษณ์ของตัวละคร
- ลืมว่าบทสนทนาไม่ต้องสมบูรณ์แบบ
