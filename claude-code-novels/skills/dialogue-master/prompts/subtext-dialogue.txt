## ประเภท Subtext

### 1. Avoidance (หลีกเลี่ยง)
พูดเรื่องอื่นเพื่อหลีกเลี่ยงเรื่องจริง

```
Surface: พูดเรื่องอาหาร
Hidden: ไม่อยากพูดเรื่องแยกทาง

"ข้าวเย็นกินอะไรดี?"
"อะไรก็ได้"
"ร้านเดิมไหม?"
"...แล้วแต่เธอ"
"หรือจะลองร้านใหม่?"
ความเงียบ
"หรือเราจะ... ไม่กินด้วยกันก็ได้นะ"
```

### 2. Power Play (ชิงอำนาจ)
บทสนทนาที่แฝงการต่อสู้เพื่ออำนาจ

```
Surface: พูดเรื่องงาน
Hidden: ใครเป็นหัวหน้าตัวจริง

"รายงานเสร็จหรือยัง?"
"กำลังทำอยู่"
"เมื่อวานบอกว่าเสร็จแล้ว"
"มีการแก้ไขเพิ่มเติม"
"ใครสั่งแก้?"
"...ผมตัดสินใจเอง"
"อ้อ เธอตัดสินใจเองได้แล้วนี่"
```

### 3. Unspoken Love (รักที่ไม่ได้พูด)
พูดเรื่องธรรมดาแต่แฝงความรู้สึก

```
Surface: คืนหนังสือ
Hidden: ไม่อยากให้ไป

"หนังสือที่ยืมไป"
"อ่านเสร็จแล้วเหรอ?"
"ยัง... แต่เธอต้องใช้ไหม?"
"ไม่เร่ง เก็บไว้ก่อนก็ได้"
"จริงเหรอ?"
"ยังไงก็... ต้องเจอกันอยู่แล้วนี่"
"ใช่สินะ... ต้องเจอกัน"
```

### 4. Hidden Threat (ขู่แบบไม่ขู่)
ดูเหมือนพูดดีแต่แฝงภัยคุกคาม

```
Surface: ถามสารทุกข์สุกดิบ
Hidden: ฉันรู้ว่าเธอทำอะไร

"ครอบครัวเป็นไงบ้าง?"
"ดีครับ"
"ลูกสาวเรียนที่ไหนนะ?"
"..."
"โรงเรียนสตรี... ใกล้บ้านใช่ไหม?"
"ท่านรู้ได้ยังไง?"
"ฉันรู้หลายอย่าง"
```

### 5. Saying Goodbye (ลาโดยไม่ลา)
พูดเรื่องปกติแต่กำลังจะจากไป

```
Surface: พูดเรื่องของในบ้าน
Hidden: ลาก่อน

"เสื้อตัวนี้... เธอซื้อให้ใช่ไหม?"
"ใช่ ทำไมล่ะ?"
"ไม่มีอะไร แค่... ชอบมาก"
"เดี๋ยวซื้อให้ใหม่ก็ได้"
"ไม่ต้องหรอก..."
เธอยิ้มแต่ตาแดง
"ตัวนี้ตัวเดียวก็พอแล้ว"
```

## เทคนิคสร้าง Subtext

### 1. สิ่งที่ไม่ได้พูดออกมา
```
"ฉัน..." เขาหยุด
"อะไร?"
"ไม่มีอะไร"

(ผู้อ่านรู้ว่าเขาอยากบอกอะไร)
```

### 2. คำถามแทนคำตอบ
```
"เธอรักฉันไหม?"
"ทำไมถามตอนนี้?"

(คำตอบซ่อนอยู่ในคำถาม)
```

### 3. พูดตรงข้าม
```
"ไม่เป็นไร"
"แน่ใจนะ?"
"บอกว่าไม่เป็นไร" พูดเสียงแข็ง

(หมายความว่าเป็นไรมาก)
```

### 4. เปลี่ยนเรื่อง
```
"ทำไมเมื่อคืนกลับดึก?"
"วันนี้อากาศดีนะ"

(หลบคำถาม = มีอะไรซ่อน)
```
