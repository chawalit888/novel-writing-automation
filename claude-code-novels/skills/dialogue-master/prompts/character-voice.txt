## องค์ประกอบ Character Voice

### 1. Vocabulary (คำศัพท์)
```
- คำที่ใช้บ่อย
- คำที่ไม่เคยใช้
- ศัพท์เฉพาะทาง (ถ้ามี)
- คำแสลง/คำหยาบ (ถ้าเหมาะสม)
```

### 2. Sentence Structure (โครงสร้างประโยค)
```
- สั้น vs ยาว
- ซับซ้อน vs เรียบง่าย
- มีการพูดไม่จบประโยค?
- มีการพูดวกวน?
```

### 3. Speech Habits (ติดปาก)
```
- คำพูดที่พูดบ่อย
- คำอุทาน
- การลงท้ายประโยค
- การเริ่มประโยค
```

### 4. Tone (น้ำเสียง)
```
- ทางการ vs ไม่ทางการ
- สุภาพ vs หยาบ
- มั่นใจ vs ลังเล
- อบอุ่น vs เย็นชา
```

## ตัวอย่างตามบุคลิก

### คนมั่นใจ
```
ประโยคสั้น กระชับ
ไม่ใช้คำว่า "อาจจะ" "น่าจะ" "คงจะ"
ใช้ "ต้อง" "จะ" "เป็น"

"ทำอย่างนี้"
"ไม่ใช่"
"ฉันจัดการเอง"
```

### คนขี้กลัว/ไม่มั่นใจ
```
ประโยคไม่จบ
ใช้คำว่า "อาจจะ" "ไม่แน่ใจ" "คงจะ"
พูดเบา

"ผม... ไม่แน่ใจว่า..."
"อาจจะลองดู... ก็ได้นะ... ถ้าไม่—"
"ขอโทษ ผมหมายความว่า..."
```

### คนฉลาด/นักวิชาการ
```
ประโยคซับซ้อน
ศัพท์ยาก
อธิบายละเอียด

"ตามทฤษฎีแล้ว กรณีนี้น่าจะเข้าข่าย
ปรากฏการณ์ที่เรียกว่า cognitive dissonance
ซึ่งหมายความว่า..."
```

### เด็ก/วัยรุ่น
```
ประโยคสั้น
คำแสลง
อารมณ์รุนแรง

"โคตรเจ๋งเลย!"
"ไม่แฟร์!"
"ก็ได้วะ ช่างมัน"
```

### ผู้สูงอายุ
```
เล่าเรื่องยาว
อ้างอิงอดีต
ช้า รอบคอบ

"สมัยก่อนน่ะ... เราไม่ได้ทำกันแบบนี้หรอก
ตอนนั้นน่ะ..."
"ใจเย็นๆ ก่อน ค่อยๆ คิด"
```
