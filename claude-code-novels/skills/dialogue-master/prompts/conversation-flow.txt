# Conversation Flow Prompt (จังหวะบทสนทนา)

## Context Variables
- {{scene_type}}: ประเภทฉาก (ขัดแย้ง, โรแมนติก, ตลก, ลึกลับ)
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{tension_level}}: ระดับความตึงเครียด (1-5)
- {{goal}}: เป้าหมายของบทสนทนา
- {{setting}}: สถานที่

คุณคือผู้เชี่ยวชาญด้านจังหวะบทสนทนา สร้างการไหลของบทสนทนาที่น่าสนใจ

## เทคนิคจังหวะบทสนทนา

### 1. Rapid-Fire (รัวเร็ว)
```
ใช้เมื่อ: โต้เถียง, ตื่นเต้น, เร่งรีบ

"ทำไม?"
"ไม่รู้"
"โกหก"
"ไม่"
"ทำไมหลบตา?"
"ไม่ได้หลบ"
"หลบอยู่"
```

### 2. Slow Build (ค่อยๆ เพิ่ม)
```
ใช้เมื่อ: ซีเรียส, เปิดเผยความจริง, อารมณ์ลึก

"มีเรื่องจะบอก"

เธอวางแก้วน้ำลง "อะไรล่ะ"

เขาหายใจลึก

"เรื่องที่... ฉันไม่เคยบอกใคร"

เธอรอ

"เกี่ยวกับคืนนั้น"
```

### 3. Interruption (ขัดจังหวะ)
```
ใช้เมื่อ: โกรธ, เร่งด่วน, ไม่อยากฟัง

"ฉันจะอธิบาย—"
"ไม่ต้อง"
"แต่—"
"ฉันบอกว่าไม่ต้อง"
"ขอแค่—"
"พอได้แล้ว!"
```

### 4. Silence Beat (จังหวะเงียบ)
```
ใช้เมื่อ: ช็อก, ตึงเครียด, อารมณ์ท่วมท้น

"ฉันรู้หมดแล้ว"

ความเงียบ

"รู้อะไร?"

ความเงียบยาวขึ้น

"เธอรู้ว่าฉันรู้อะไร"
```

### 5. Overlapping (พูดทับ)
```
ใช้เมื่อ: ทะเลาะ, ตื่นเต้น, หลายคนพูดพร้อมกัน

"เราต้อง—" "ไม่เอา!" "ฟังก่อนสิ!" "—ทำอะไรสักอย่าง!"
```

## โครงสร้างจังหวะตามอารมณ์

### ฉากตึงเครียด
```
เริ่ม: ช้า ระมัดระวัง
กลาง: เร็วขึ้น สั้นลง
จุดพีค: รัวมาก หรือเงียบสนิท
หลังพีค: ช้าลง หายใจ
```

### ฉากโรแมนติก
```
เริ่ม: อ้อมค้อม หลบตา
กลาง: ใกล้ชิดขึ้น ลังเล
จุดพีค: พูดน้อย ใช้ความเงียบ
หลังพีค: อบอุ่น นุ่มนวล
```

### ฉากตลก
```
เริ่ม: ปกติ
กลาง: เพิ่มจังหวะ ไม่คาดคิด
จุดพีค: Punchline (ทิ้งช่วง)
หลังพีค: Reaction
```
