# Dialogue Master Skill

## Overview
Skill สำหรับเขียนบทสนทนาที่มีชีวิต ทำให้ตัวละครแต่ละตัวมี "เสียง" เฉพาะตัว

---

## ความสามารถหลัก

### 1. Character Voice (เสียงตัวละคร)
ทำให้ตัวละครแต่ละคนพูดไม่เหมือนกัน:
- คำศัพท์ที่ใช้
- ความยาวประโยค
- สำเนียง/ภาษาถิ่น
- ติดปาก/คำพูดประจำ
- ระดับภาษา (ทางการ/ไม่ทางการ)

### 2. Conversation Flow (จังหวะบทสนทนา)
- การตัดบท/แทรก
- ความเงียบที่มีความหมาย
- จังหวะเร็ว-ช้า
- การหลีกเลี่ยงคำตอบ

### 3. Subtext (ความหมายซ่อน)
- พูดอย่างหมายความอีกอย่าง
- สิ่งที่ไม่ได้พูดออกมา
- อารมณ์ที่ซ่อนอยู่

---

## ประเภทบทสนทนา

### 1. Exposition Dialogue
บทสนทนาที่ให้ข้อมูล แต่ไม่รู้สึกว่าเป็นการบรรยาย

```
❌ แย่:
"คุณรู้ไหมว่าเมืองนี้ก่อตั้งเมื่อ 500 ปีก่อน
โดยกษัตริย์องค์แรกของราชวงศ์"

✅ ดี:
"ป้ายนี้บอกว่าเมืองก่อตั้ง 500 ปีแล้วนะ"
"แค่นั้น? คุณยายฉันเล่าว่าตอนก่อตั้ง
มีคนตายเป็นร้อย แต่ไม่มีใครกล้าเขียนลงไป"
```

### 2. Conflict Dialogue
บทสนทนาที่มีความขัดแย้ง

```
"ฉันไม่ได้ทำอะไรผิด"
"แล้วเงินหายไปไหนล่ะ?"
"ฉันไม่รู้"
"ไม่รู้หรือไม่อยากบอก?"
ความเงียบ
"นั่นแหละคำตอบ"
```

### 3. Romantic Dialogue
บทสนทนาโรแมนติก

```
"ทำไมมองฉันอย่างนั้น?"
"อย่างไหน?"
"อย่าง... ไม่รู้สิ แปลกๆ"
"ก็แค่มอง"
"แต่มันทำให้ฉัน..."
"ทำให้เธออะไร?"
เธอหันหน้าหนี "ไม่มีอะไร"
```

### 4. Comedic Dialogue
บทสนทนาตลก

```
"ฉันมีข่าวดีกับข่าวร้าย"
"บอกข่าวดีก่อน"
"แฟนเก่าเธอแต่งงานแล้ว"
"นั่นข่าวดี?"
"เขาแต่งกับคนที่น่าเกลียดกว่าเธอ"
"..."
"ข่าวร้ายคือเธอต้องไปงานแต่ง"
```

## เทคนิคสำคัญ

### 1. Speech Pattern ตามบุคลิก

| บุคลิก | ลักษณะการพูด |
|--------|-------------|
| เจ้านาย/ผู้มีอำนาจ | ประโยคสั้น สั่ง ไม่ถาม |
| คนขี้กลัว | ประโยคไม่จบ ลังเล |
| คนฉลาด | ศัพท์ยาก พูดซับซ้อน |
| คนซื่อ | ตรงไปตรงมา ไม่อ้อมค้อม |
| คนโกหก | พูดมากเกินไป ซ้ำๆ |
| คนโกรธ | ประโยคสั้น ขาด รุนแรง |

### 2. การแสดง Action ระหว่างสนทนา

```
"ฉันไม่ได้โกหก" เขาพูดพลางเอามือล้วงกระเป๋า

"จริงเหรอ?" เธอยกคิ้ว

"จริง" เขาหลบตา

เธอถอนหายใจ "ตาเธอบอกอีกอย่างนะ"
```

### 3. การใช้ความเงียบ

```
"ทำไมถึงทำอย่างนั้น?"

ความเงียบยาวนาน

"ตอบฉันสิ"

"..."

"โอเค ไม่ต้องตอบก็ได้ ฉันรู้แล้ว"
```

### 4. การตัดบท

```
"ฉันจะบอกความจริงให้ฟัง จริงๆ แล้ว—"
"ช่างมันเถอะ"
"แต่—"
"ฉันบอกว่าช่างมัน"
```

## ตัวอย่าง Character Voice

### ตัวละคร A: CEO เย็นชา
```
"รายงาน"
"ยอดขายเดือนนี้—"
"ตัวเลข"
"ลดลง 12%"
"เหตุผล"
"ตลาดผันผวน—"
"แก้ไข"
"เราจะ—"
"ภายในสัปดาห์หน้า"
```

### ตัวละคร B: เด็กสาวร่าเริง
```
"เฮ้ เฮ้ เฮ้! รู้ไหมว่าวันนี้เกิดอะไรขึ้น!"
"อะไรล่ะ?"
"ฉันเจอแมวน่ารักมากกก ตัวส้มๆ ขนฟูๆ
มันมานั่งข้างๆ ฉันเลย! น่ารักสุดๆ!
แล้วก็แล้วก็ มันร้องเมี้ยวใส่ฉันด้วย!"
```

### ตัวละคร C: ตำรวจเก่า
```
"ฟังนะ..."
เขาจุดบุหรี่ สูดควันลึก
"เรื่องนี้... มันไม่ง่าย"
"หมายความว่า?"
"หมายความว่า..." เขาสูดควันอีกครั้ง
"บางเรื่องไม่ควรขุดคุ้ย"
```

## Tips

### DO:
- ให้ตัวละครขัดจังหวะกัน
- ใช้ความเงียบเป็นเครื่องมือ
- ให้ตัวละครหลบเลี่ยงคำถาม
- ใส่ Action ระหว่างบทพูด

### DON'T:
- ให้ทุกคนพูดยาวเท่ากัน
- ให้ทุกคนพูดสุภาพเหมือนกัน
- ใส่ชื่อในทุกประโยค ("เฮ้ สมชาย" "ใช่ สมหญิง")
- บอกอารมณ์ตรงๆ ("ฉันโกรธมาก!" "ฉันเศร้า!")
