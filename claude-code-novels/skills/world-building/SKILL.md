# World Building Skill

## Overview
Skill สำหรับสร้างโลกนิยาย ระบบเวทย์ ภูมิศาสตร์ วัฒนธรรม และประวัติศาสตร์

---

## ความสามารถหลัก

### 1. Magic/Power System (ระบบพลัง)
- กฎและข้อจำกัด
- ที่มาของพลัง
- ราคาที่ต้องจ่าย
- ระดับความแข็งแกร่ง

### 2. Geography & Environment (ภูมิศาสตร์)
- แผนที่โลก/ดินแดน
- ภูมิอากาศ
- ทรัพยากรธรรมชาติ
- สถานที่สำคัญ

### 3. Culture & Society (วัฒนธรรม)
- ประเพณี/พิธีกรรม
- ระบบชนชั้น
- ศาสนา/ความเชื่อ
- ภาษา/สำเนียง

### 4. History & Lore (ประวัติศาสตร์)
- เหตุการณ์สำคัญ
- ตำนาน/นิทาน
- สงคราม/ความขัดแย้ง
- บุคคลในตำนาน

---

## ประเภท World Building

### 1. High Fantasy World
โลกที่สร้างขึ้นใหม่ทั้งหมด

```
Elements:
- ระบบเวทย์มนตร์
- เผ่าพันธุ์ต่างๆ (เอลฟ์, คนแคระ, ฯลฯ)
- อาณาจักร/ดินแดน
- ภาษาเฉพาะ
- ศาสนา/เทพเจ้า

ตัวอย่าง: Lord of the Rings, Game of Thrones
```

### 2. Urban Fantasy
โลกจริงแต่มีองค์ประกอบเหนือธรรมชาติ

```
Elements:
- ประวัติศาสตร์ลับ
- สังคมลับของสิ่งเหนือธรรมชาติ
- กฎการซ่อนตัว
- พื้นที่ปลอดภัย/อันตราย

ตัวอย่าง: Harry Potter, Twilight
```

### 3. Sci-Fi World
โลกอนาคตหรือต่างดาว

```
Elements:
- เทคโนโลยี
- การเดินทางอวกาศ
- สังคมอนาคต
- เผ่าพันธุ์ต่างดาว

ตัวอย่าง: Star Wars, Dune
```

### 4. Alternate History
ประวัติศาสตร์ทางเลือก

```
Elements:
- จุดแยก (Point of Divergence)
- ผลกระทบต่อเนื่อง
- เทคโนโลยีที่เปลี่ยน
- สังคมที่เปลี่ยน

ตัวอย่าง: The Man in the High Castle
```

## ระบบเวทย์มนตร์ (Magic System)

### Hard Magic System
กฎชัดเจน ข้อจำกัดชัดเจน

```
ตัวอย่างโครงสร้าง:

1. แหล่งที่มา
   - มาจากไหน? (ภายใน/ภายนอก)
   - ใครใช้ได้? (เลือดสืบทอด/ฝึกฝน/ของขวัญ)

2. กลไก
   - ใช้ยังไง? (ท่องมนตร์/จินตนาการ/สัญลักษณ์)
   - ต้องใช้อะไร? (ไม้กายสิทธิ์/หินวิเศษ/การสัมผัส)

3. ข้อจำกัด
   - ราคาที่ต้องจ่าย (พลังงาน/สุขภาพ/อายุ)
   - ข้อห้าม (สร้างชีวิตไม่ได้/ย้อนเวลาไม่ได้)
   - ข้อจำกัดทางกายภาพ (ระยะ/ปริมาณ)

4. ระดับ
   - ระดับความแข็งแกร่ง
   - การพัฒนา/ฝึกฝน
```

### Soft Magic System
ลึกลับ ไม่อธิบายทั้งหมด

```
ตัวอย่าง:

- มีเวทย์มนตร์อยู่แต่ไม่รู้ที่มา
- บางคนมีพลังบางคนไม่มี
- ข้อจำกัดไม่ชัดเจน
- สร้างความลึกลับและ wonder

ข้อควรระวัง: อย่าใช้แก้ปัญหาง่ายเกินไป (Deus ex machina)
```

## การสร้างวัฒนธรรม

### Framework วัฒนธรรม

```
1. Basics (พื้นฐาน)
   - อาหาร
   - เสื้อผ้า
   - ที่อยู่อาศัย
   - ภาษา

2. Social Structure (โครงสร้างสังคม)
   - ชนชั้น
   - ครอบครัว
   - การแต่งงาน
   - มรดก

3. Beliefs (ความเชื่อ)
   - ศาสนา
   - ไสยศาสตร์
   - ค่านิยม
   - ข้อห้าม (Taboo)

4. Arts & Entertainment (ศิลปะ)
   - ดนตรี
   - การแสดง
   - กีฬา/เกม
   - เทศกาล

5. Economics (เศรษฐกิจ)
   - อาชีพ
   - การค้า
   - เงินตรา
   - ทรัพยากร
```

## ตัวอย่าง World Building

### ระบบเวทย์: "เลือดสีทอง"
```
แหล่งที่มา: สืบทอดทางสายเลือด
กลไก: เลือดเป็นสีทอง ยิ่งทองมากพลังยิ่งเยอะ
ข้อจำกัด: ใช้เลือดเป็นเชื้อเพลิง ใช้มากตาย
ระดับ:
- ทองจาง = พลังน้อย
- ทองเข้ม = พลังมาก
- ทองบริสุทธิ์ = ตำนาน (หายไปแล้ว?)
```

### วัฒนธรรม: "เมืองลอยฟ้า"
```
พื้นฐาน: เมืองลอยอยู่บนเมฆ
ชนชั้น: ยิ่งสูงยิ่งสูงศักดิ์
ความเชื่อ: บูชาลม ถือว่าพื้นดินสกปรก
ข้อห้าม: ห้ามแตะพื้นดิน ถ้าแตะจะถูกเนรเทศ
เศรษฐกิจ: ค้าขายกับพื้นดินผ่านลิฟต์
```

## Checklist

- [ ] ระบบเวทย์มีกฎและข้อจำกัดชัดเจน
- [ ] ภูมิศาสตร์สมเหตุสมผล
- [ ] วัฒนธรรมมีเอกลักษณ์
- [ ] ประวัติศาสตร์เชื่อมโยงกับปัจจุบัน
- [ ] รายละเอียดไม่ขัดแย้งกัน
- [ ] มี "ทำไม" อธิบายทุกอย่าง
- [ ] ไม่มากเกินจนเล่าไม่หมด

---

## Tips

### DO:
- สร้างกฎแล้วทำตาม
- ให้รายละเอียดที่เกี่ยวกับเรื่อง
- ให้มีข้อจำกัดที่สร้าง conflict
- เชื่อมโยง setting กับ plot

### DON'T:
- Info dump ทุกอย่างในบทแรก
- สร้างกฎแล้วแหก
- สร้างโลกที่ perfect ไม่มีปัญหา
- ลืมว่าโลกมีคนธรรมดาอยู่ด้วย
