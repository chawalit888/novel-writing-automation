# History & Lore Prompt (ประวัติศาสตร์และตำนาน)

## Context Variables
- {{world_name}}: ชื่อโลก/ดินแดน
- {{time_span}}: ระยะเวลาประวัติศาสตร์
- {{key_events}}: เหตุการณ์สำคัญที่ต้องมี
- {{legendary_figures}}: บุคคลในตำนาน
- {{current_situation}}: สถานการณ์ปัจจุบัน (ตอนเรื่องเริ่ม)

## Prompt Template

คุณคือนักประวัติศาสตร์และนักเล่านิทานแห่งโลกจินตนาการ สร้างประวัติศาสตร์ที่เชื่อมโยงกับปัจจุบัน

### ข้อมูล:
- โลก/ดินแดน: {{world_name}}
- ระยะเวลา: {{time_span}}
- เหตุการณ์สำคัญ: {{key_events}}
- บุคคลในตำนาน: {{legendary_figures}}
- สถานการณ์ปัจจุบัน: {{current_situation}}

## โครงสร้างประวัติศาสตร์

### 1. Creation Era (ยุคสร้างโลก)
```
- โลกเกิดขึ้นมาได้อย่างไร?
- เทพเจ้า/พลังที่สร้าง?
- เผ่าพันธุ์แรกคือใคร?
- ยุคแรกเป็นอย่างไร? (ทอง/เงิน/ทองแดง?)
```

### 2. Ancient Era (ยุคโบราณ)
```
- อาณาจักรแรกเริ่ม
- ผู้นำคนแรก/วีรบุรุษแรก
- การค้นพบสำคัญ (เวทย์มนตร์? เทคโนโลยี?)
- ความขัดแย้งแรก
```

### 3. Golden Age (ยุคทอง)
```
- ช่วงรุ่งเรืองสูงสุด
- ความสำเร็จที่ภูมิใจ
- อารยธรรมที่ยิ่งใหญ่
- สิ่งที่สูญหายไป (จะเป็น plot ได้)
```

### 4. Dark Age (ยุคมืด)
```
- หายนะครั้งใหญ่
- สงครามทำลายล้าง
- ความรู้ที่สูญหาย
- ผู้รอดชีวิต/ผู้สืบทอด
```

### 5. Recovery Era (ยุคฟื้นฟู)
```
- การสร้างใหม่
- อาณาจักรใหม่
- ความขัดแย้งใหม่
- บทเรียนที่ลืม/จำ
```

### 6. Current Era (ยุคปัจจุบัน)
```
- สถานการณ์ตอนเรื่องเริ่ม
- ปัญหาที่กำลังเกิด
- เชื่อมโยงกับอดีตอย่างไร
```

## ตำนานและนิทาน

### ประเภทตำนาน

#### 1. Creation Myth (ตำนานสร้างโลก)
```
"ในครั้งกาลก่อน เมื่อยังไม่มีสิ่งใด
มีเพียง [เทพ/พลัง] ที่ [ทำอะไร]
จึงเกิดเป็น [โลก/สิ่งมีชีวิต]
และมอบ [ของขวัญ/คำสาป] แก่ลูกหลาน"
```

#### 2. Hero Legend (ตำนานวีรบุรุษ)
```
"[ชื่อวีรบุรุษ] ผู้ที่ [ที่มา]
เผชิญกับ [ความท้าทาย]
ด้วย [คุณสมบัติ/อาวุธ]
จึง [สำเร็จ/ล้มเหลว]
และทิ้ง [มรดก] ไว้ให้คนรุ่นหลัง"
```

#### 3. Cautionary Tale (นิทานสอนใจ)
```
"มี [ตัวละคร] ที่ [ทำผิด]
ผลคือ [ลงโทษ/ผลร้าย]
ดังนั้น [บทเรียน]"
```

#### 4. Prophecy (คำทำนาย)
```
"เมื่อ [สัญญาณ] ปรากฏ
[สิ่งที่จะเกิด]
เฉพาะ [ผู้ที่ถูกเลือก]
จะสามารถ [ภารกิจ]"
```

## Checklist

- [ ] มี Timeline ที่ชัดเจน
- [ ] เหตุการณ์เชื่อมโยงกัน (Cause → Effect)
- [ ] ตำนานสอดคล้องกับประวัติศาสตร์
- [ ] บุคคลในตำนานมีบทบาทชัดเจน
- [ ] เชื่อมโยงกับ plot ปัจจุบัน
- [ ] มีความลับ/ความจริงที่ซ่อนอยู่ (สำหรับ reveal)
- [ ] ไม่มากเกินจนจำไม่ได้
