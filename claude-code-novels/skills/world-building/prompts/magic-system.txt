# Magic System Prompt (ระบบเวทย์/พลัง)

## Context Variables
- {{genre}}: แนวนิยาย (Fantasy, Sci-fi, Urban Fantasy)
- {{magic_type}}: ประเภท (Hard/Soft)
- {{tone}}: โทนเรื่อง (Dark, Light, Epic)
- {{key_abilities}}: พลังที่ต้องมีในเรื่อง
- {{protagonist_power}}: พลังของตัวเอก

---

## Prompt Template

คุณคือผู้เชี่ยวชาญด้านการออกแบบระบบเวทย์มนตร์ สร้างระบบที่มีความสมบูรณ์และน่าสนใจ

### ข้อมูล:
- แนว: {{genre}}
- ประเภทระบบ: {{magic_type}}
- โทน: {{tone}}
- พลังสำคัญ: {{key_abilities}}
- พลังตัวเอก: {{protagonist_power}}

---

## โครงสร้างระบบเวทย์

### 1. แหล่งที่มา (Source)
```
คำถามที่ต้องตอบ:
- พลังมาจากไหน?
  □ ภายใน (พลังชีวิต, จิตใจ, วิญญาณ)
  □ ภายนอก (ธรรมชาติ, เทพเจ้า, มิติอื่น)
  □ สิ่งของ (หิน, อาวุธ, สิ่งประดิษฐ์)

- ใครมีพลังได้?
  □ ทุกคน (ต้องฝึก)
  □ บางคน (เลือดสืบทอด)
  □ ถูกเลือก (ของขวัญ/คำสาป)
```

### 2. กลไก (Mechanism)
```
คำถามที่ต้องตอบ:
- ใช้ยังไง?
  □ ท่องมนตร์/คาถา
  □ จินตนาการ/ตั้งใจ
  □ ท่าทาง/การเคลื่อนไหว
  □ เครื่องมือ/ตัวกลาง

- ต้องใช้อะไรประกอบ?
  □ ไม่ต้องใช้อะไร
  □ อุปกรณ์เฉพาะ (ไม้กายสิทธิ์, แหวน)
  □ วัตถุดิบ (ส่วนผสม, คริสตัล)
  □ เงื่อนไขพิเศษ (แสงจันทร์, เลือด)
```

### 3. ข้อจำกัด (Limitations)
```
⭐ ส่วนสำคัญที่สุด!

- ราคาที่ต้องจ่าย:
  □ พลังงาน/ความเหนื่อย
  □ สุขภาพ/อายุขัย
  □ ความเจ็บปวด
  □ ความทรงจำ
  □ อารมณ์/จิตใจ

- ข้อห้าม:
  □ สิ่งที่ทำไม่ได้เด็ดขาด
  □ สิ่งที่ต้องห้าม (ทำได้แต่ไม่ควร)

- ข้อจำกัดทางกายภาพ:
  □ ระยะทาง
  □ ปริมาณ/ขนาด
  □ ระยะเวลา
  □ จำนวนครั้ง
```

### 4. ระดับพลัง (Power Levels)
```
ตัวอย่างโครงสร้าง:

ระดับ 1: เริ่มต้น
- ทำอะไรได้: [รายการ]
- ข้อจำกัด: [รายการ]

ระดับ 2: กลาง
- ทำอะไรได้: [รายการ]
- ข้อจำกัด: [รายการ]

ระดับ 3: สูง
- ทำอะไรได้: [รายการ]
- ข้อจำกัด: [รายการ]

ระดับ 4: ตำนาน
- ทำอะไรได้: [รายการ]
- ข้อจำกัด: [รายการ]
```

## ตัวอย่างระบบเวทย์

### ตัวอย่าง 1: "Ink Magic"
```
แหล่งที่มา: หมึกพิเศษที่ผลิตจากน้ำตาปีศาจ
กลไก: เขียนสิ่งที่ต้องการให้เกิด
ราคา: หมึกดูดพลังชีวิตตอนเขียน
ข้อจำกัด:
- เขียนผิดไม่ได้ ผิดแล้วตาย
- ต้องเขียนให้จบในลมหายใจเดียว
- สิ่งที่เขียนมีผลแค่ 1 วัน
```

### ตัวอย่าง 2: "Echo Power"
```
แหล่งที่มา: สะท้อนพลังจากความตายของคนใกล้ชิด
กลไก: นึกถึงคนที่ตาย ใช้พลังของเขา
ราคา: ยิ่งใช้ยิ่งลืมความทรงจำกับคนนั้น
ข้อจำกัด:
- พลังอ่อนลงเรื่อยๆ
- ลืมหมดแล้วใช้ไม่ได้อีก
- ต้องมีความผูกพันแท้จริง
```

## Checklist

- [ ] มีแหล่งที่มาชัดเจน
- [ ] มีกลไกที่เข้าใจได้
- [ ] มีข้อจำกัดที่สร้าง conflict
- [ ] ระดับพลังสมดุล
- [ ] ไม่แก้ปัญหาได้หมดทุกอย่าง
- [ ] สอดคล้องกับ tone ของเรื่อง
- [ ] น่าสนใจและเป็นเอกลักษณ์
