## การสร้างภูมิศาสตร์

### 1. Terrain (ภูมิประเทศ)
```
□ ภูเขา - แหล่งเหมือง, กำแพงธรรมชาติ
□ แม่น้ำ - เส้นทางการค้า, แหล่งน้ำ
□ ป่า - ทรัพยากร, อันตราย
□ ทะเลทราย - กันดาร, ซ่อนตัว
□ ชายฝั่ง - ท่าเรือ, ประมง
□ ที่ราบ - เกษตร, สนามรบ
□ หนองบึง - กันดาร, ลึกลับ
□ ธารน้ำแข็ง - ความหนาว, โดดเดี่ยว
```

### 2. Climate (ภูมิอากาศ)
```
□ ร้อนชื้น - เสื้อผ้าบาง, โรคระบาด
□ หนาวเย็น - เสื้อผ้าหนา, อาหารถนอม
□ ทะเลทราย - กลางวันร้อน กลางคืนหนาว
□ อบอุ่น - เกษตรอุดมสมบูรณ์
□ เขตร้อน - ฝนชุก, ป่าทึบ
□ ขั้วโลก - มืดนาน, สว่างนาน
```

### 3. Resources (ทรัพยากร)
```
□ แร่ธาตุ - อะไร? หายากแค่ไหน?
□ อาหาร - ปลูกอะไร? เลี้ยงอะไร?
□ น้ำ - อุดมสมบูรณ์? ขาดแคลน?
□ ไม้ - มาก? น้อย?
□ สิ่งวิเศษ - (Fantasy) แร่เวทย์? สมุนไพร?
```

## การสร้างวัฒนธรรม

### 1. Basics (ชีวิตประจำวัน)
```
อาหาร:
- กินอะไรเป็นหลัก?
- อาหารพิเศษ/ต้องห้าม?
- มารยาทบนโต๊ะอาหาร?

เสื้อผ้า:
- วัสดุอะไร?
- สี/ลวดลายมีความหมาย?
- เสื้อผ้าตามชนชั้น?

ที่อยู่อาศัย:
- บ้านทำจากอะไร?
- รูปแบบบ้าน?
- ครอบครัวใหญ่/เดี่ยว?
```

### 2. Social Structure (โครงสร้างสังคม)
```
ชนชั้น:
□ กษัตริย์/ขุนนาง/สามัญ
□ นักรบ/พ่อค้า/ชาวนา
□ ผู้มีพลัง/ไม่มีพลัง
□ ไม่มีชนชั้น (เท่าเทียม)

ครอบครัว:
□ พ่อเป็นใหญ่ (Patriarchy)
□ แม่เป็นใหญ่ (Matriarchy)
□ เท่าเทียม

การแต่งงาน:
□ คลุมถุงชน
□ เลือกเอง
□ ห้ามข้ามชนชั้น
□ พหุภรรยา/สามี
```

### 3. Beliefs (ความเชื่อ)
```
ศาสนา:
□ เทพเดียว (Monotheism)
□ หลายเทพ (Polytheism)
□ บูชาธรรมชาติ (Animism)
□ ไม่มีศาสนา

พิธีกรรม:
- เกิด: [พิธีอะไร]
- โต: [พิธีบรรลุนิติภาวะ]
- แต่งงาน: [พิธีอะไร]
- ตาย: [พิธีอะไร]

ข้อห้าม (Taboo):
- ห้ามทำอะไร?
- ผลของการฝ่าฝืน?
```

### 4. Economics (เศรษฐกิจ)
```
อาชีพหลัก:
□ เกษตรกรรม
□ ค้าขาย
□ หัตถกรรม
□ เหมืองแร่
□ ประมง

ระบบเงินตรา:
□ เหรียญ (ทอง/เงิน/ทองแดง)
□ แลกเปลี่ยนสินค้า
□ ของมีค่าอื่น (เปลือกหอย, คริสตัล)
```
