# Chapter Pacing Prompt (วิเคราะห์จังหวะรายบท)

## Context Variables
- {{chapter_number}}: หมายเลขบท
- {{chapter_content}}: เนื้อหาบท (หรือสรุป)
- {{chapter_purpose}}: จุดประสงค์ของบท
- {{previous_chapter}}: บทก่อนหน้า (สรุป)
- {{next_chapter}}: บทถัดไป (สรุป)

---

## Prompt Template

คุณคือนักวิเคราะห์ Pacing มืออาชีพ วิเคราะห์จังหวะของบทนิยาย

### ข้อมูล:
- บทที่: {{chapter_number}}
- เนื้อหา: {{chapter_content}}
- จุดประสงค์: {{chapter_purpose}}
- บทก่อนหน้า: {{previous_chapter}}
- บทถัดไป: {{next_chapter}}

---

## Criteria การวิเคราะห์

### 1. Story Movement (การดำเนินเรื่อง)
```
□ เรื่องเดินหน้า?
□ มีเหตุการณ์สำคัญ?
□ ตัวละครเข้าใกล้/ไกลเป้าหมาย?

Score: ___/3
```

### 2. Conflict & Tension (ความขัดแย้ง)
```
□ มี conflict (ภายใน/ภายนอก)?
□ Stakes ชัดเจน?
□ มี tension ตลอด?

Score: ___/3
```

### 3. Character Development (พัฒนาตัวละคร)
```
□ ตัวละครเปลี่ยนแปลง?
□ เรียนรู้อะไรใหม่?
□ ความสัมพันธ์เปลี่ยน?

Score: ___/3
```

### 4. Information & Revelation (ข้อมูล)
```
□ มีข้อมูลใหม่?
□ มีการเปิดเผยความลับ?
□ ผู้อ่านรู้มากขึ้น?

Score: ___/3
```

### 5. Emotional Impact (อารมณ์)
```
□ มีจุดที่กระทบอารมณ์?
□ ผู้อ่านจะรู้สึกอย่างไร?
□ มี emotional payoff?

Score: ___/3
```

### 6. Hook & Ending (การจบ)
```
□ จบด้วย hook?
□ ทำให้อยากอ่านต่อ?
□ ไม่จบแบบลอยๆ?

Score: ___/3
```

---

## การประเมินความเร็ว

### เร็วเกินไป (Too Fast)
```
สัญญาณ:
- หลายเหตุการณ์ในบทเดียว
- ไม่มีเวลาให้ moment สำคัญ
- Emotional scene ผ่านไปเร็ว
- บทสนทนาสั้นเกินไป

แนะนำ:
- แยกเป็น 2 บท
- เพิ่ม internal monologue
- ให้เวลา reaction
```

### ช้าเกินไป (Too Slow)
```
สัญญาณ:
- ไม่มีอะไรเกิดขึ้น
- บรรยายมากเกินไป
- ตัวละครคิดวนไปมา
- ไม่มี forward movement

แนะนำ:
- รวมกับบทอื่น
- ตัด filler
- เพิ่ม conflict
```

### พอดี (Just Right)
```
สัญญาณ:
- มี 1-2 เหตุการณ์สำคัญ
- มีเวลาให้ moment
- มี conflict + development
- จบด้วย hook
```

---

## Scoring Guide

| Total Score | Assessment |
|-------------|------------|
| 15-18 | ยอดเยี่ยม - บทนี้ทำงานได้ดี |
| 11-14 | ดี - มีจุดเล็กน้อยที่ปรับได้ |
| 7-10 | ปานกลาง - ต้องปรับปรุง |
| 4-6 | อ่อน - พิจารณาเขียนใหม่ |
| 0-3 | ตัดออก - ไม่ contribute อะไร |

- [ ] ระบุปัญหาชัดเจน
- [ ] มีคำแนะนำที่ทำได้จริง
- [ ] เปรียบเทียบกับบทก่อน/หลัง
