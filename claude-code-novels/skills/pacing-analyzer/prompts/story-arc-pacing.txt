## Three-Act Structure Analysis

### Act 1: Setup (25%)
```
เป้าหมาย:
- แนะนำตัวละคร
- สร้าง ordinary world
- Inciting Incident
- ตัดสินใจเข้าสู่ adventure

ตรวจสอบ:
□ ใช้กี่บท? ____ (ควรเป็น 25% ของทั้งหมด)
□ มี Inciting Incident ชัดเจน?
□ ผู้อ่านเข้าใจ stakes?
□ จบ Act 1 ด้วย turning point?
```

### Act 2: Confrontation (50%)
```
เป้าหมาย:
- Rising Action
- Midpoint (twist)
- ความยากลำบากเพิ่มขึ้น
- All is Lost moment

ตรวจสอบ:
□ ใช้กี่บท? ____ (ควรเป็น 50%)
□ มี Midpoint twist?
□ Tension เพิ่มขึ้นเรื่อยๆ?
□ ไม่มี Sagging Middle?
□ มี subplot ช่วยพยุง?
```

### Act 3: Resolution (25%)
```
เป้าหมาย:
- Climax
- Final Battle/Confrontation
- Resolution
- New Normal

ตรวจสอบ:
□ ใช้กี่บท? ____ (ควรเป็น 25%)
□ Climax ได้เวลาเพียงพอ?
□ ปมทุกปมถูกคลี่คลาย?
□ Ending satisfying?
```

## Common Pacing Problems

### 1. Front-Heavy (หนักหน้า)
```
ปัญหา: Act 1 ยาวเกินไป

Pattern:
████████████████░░░░░░░░░░░░░░
        ↑
     ยาวเกิน

แก้ไข:
- ตัด backstory
- เริ่มที่ action
- World-build ระหว่างทาง
```

### 2. Sagging Middle (กลางหย่อน)
```
ปัญหา: Act 2 ไม่มีอะไร

Pattern:
████░░░░░░░░░░░░░░░░░░████████
         ↑
      flat

แก้ไข:
- เพิ่ม mini-conflicts
- เพิ่ม subplot
- เพิ่ม revelations
```

### 3. Rushed Ending (จบรีบ)
```
ปัญหา: Act 3 สั้นเกินไป

Pattern:
████████████████████████████░░
                            ↑
                         รีบเกิน

แก้ไข:
- ให้เวลา climax
- ให้เวลา emotional moments
- Resolution ที่สมบูรณ์
```
