# Scene Transition Prompt (วิเคราะห์การเชื่อมฉาก)

## Context Variables
- {{scene_a}}: ฉากก่อนหน้า
- {{scene_b}}: ฉากถัดไป
- {{time_gap}}: ช่วงเวลาที่ข้าม
- {{location_change}}: เปลี่ยนสถานที่?
- {{pov_change}}: เปลี่ยน POV?

---

## Prompt Template

คุณคือผู้เชี่ยวชาญด้านการเชื่อมฉาก สร้างการเปลี่ยนฉากที่ลื่นไหล

### ข้อมูล:
- ฉาก A: {{scene_a}}
- ฉาก B: {{scene_b}}
- ข้ามเวลา: {{time_gap}}
- เปลี่ยนที่: {{location_change}}
- เปลี่ยน POV: {{pov_change}}

## ประเภทการเชื่อมฉาก

### 1. Hard Cut (ตัดแบบแข็ง)
```
ใช้เมื่อ: ต้องการความแตกต่างชัดเจน

ตัวอย่าง:
"เขาหลับตา..."

---

"สองปีให้หลัง"
```

### 2. Soft Transition (เชื่อมแบบนุ่ม)
```
ใช้เมื่อ: เวลาผ่านไปไม่นาน, อารมณ์ต่อเนื่อง

ตัวอย่าง:
"เธอยิ้มลา แล้วเดินหายไปในฝูงชน"

"สิบนาทีผ่านไป เขายังยืนมองตรงนั้น"
```

### 3. Bridge Transition (สะพานเชื่อม)
```
ใช้เมื่อ: เชื่อมด้วยสิ่งที่เหมือนกัน

ตัวอย่าง:
"ฝนตกหนักจนมองไม่เห็นทาง"

"ฝนตกหนักในอีกมุมเมือง
เธอมองออกไปนอกหน้าต่าง"
(เชื่อมด้วย "ฝน")
```

### 4. Emotional Bridge (สะพานอารมณ์)
```
ใช้เมื่อ: เชื่อมด้วยอารมณ์ที่คล้ายกัน

ตัวอย่าง:
"เขารู้สึกว่างเปล่า..."

"ความว่างเปล่าเดียวกันนั้น
ทำให้เธอตื่นขึ้นกลางดึก"
```

### 5. Contrast Cut (ตัดแบบขัดแย้ง)
```
ใช้เมื่อ: ต้องการ irony หรือ contrast

ตัวอย่าง:
"เธอยิ้มอย่างมีความสุข"

---

"ขณะนั้นเอง
ในอีกมุมเมือง
เขาร้องไห้คนเดียว"
```

## เทคนิคบอกการข้ามเวลา

### 1. Direct Statement (บอกตรงๆ)
```
"สามวันต่อมา"
"หนึ่งสัปดาห์ผ่านไป"
"เช้าวันรุ่งขึ้น"
```

### 2. Environmental Change (สิ่งแวดล้อมเปลี่ยน)
```
"ใบไม้ที่เคยเขียวขจี
ตอนนี้เหลืองอำพัน"
(= ผ่านไปหลายเดือน)
```

### 3. Character Change (ตัวละครเปลี่ยน)
```
"ผมที่เคยสั้น
ตอนนี้ยาวถึงบ่า"
```

### 4. Action Implication (implied)
```
"เธอปิดประตูห้อง..."

"...และเปิดมันอีกครั้ง
ด้วยกระเป๋าพร้อมเดินทาง"
```

---

## ปัญหาการเชื่อมฉากที่พบบ่อย

### 1. Jarring Jump (กระโดดแรง)
```
ปัญหา: เปลี่ยนฉากกะทันหันเกินไป

สัญญาณ:
- ผู้อ่านสับสน
- ไม่รู้ว่าอยู่ไหน/เมื่อไหร่
- อารมณ์ขาดตอน

แก้ไข:
- เพิ่ม transition line
- บอก time/place
- ใช้ bridge technique
```

### 2. Over-Explanation (อธิบายมาก)
```
ปัญหา: บอกทุกอย่างมากเกินไป

สัญญาณ:
- "สามวันต่อมา ที่บ้านของเธอ
  ในขณะที่ฝนตก เธอนั่งอยู่บนโซฟา..."

แก้ไข:
- ให้ข้อมูลเท่าที่จำเป็น
- Trust the reader
```

### 3. Lost Thread (หายไป)
```
ปัญหา: ทิ้งสิ่งที่เกิดขึ้นในฉากก่อน

สัญญาณ:
- เหตุการณ์สำคัญถูกลืม
- อารมณ์ไม่ต่อเนื่อง

แก้ไข:
- Reference กลับไป
- Show consequence
```

## ตัวอย่าง

### Situation
```
ฉาก A: ตัวเอกโดนไล่ออกจากงาน (โกรธ, ผิดหวัง)
ฉาก B: ตัวเอกที่บ้าน 3 วันต่อมา
```

### Bad Transition
```
"เขาเดินออกจากออฟฟิศ"

สามวันต่อมา

"เขานั่งอยู่ที่บ้าน"
```

### Good Transition
```
"เขาเดินออกจากออฟฟิศ
ไม่หันกลับไปมอง"

สามวันผ่านไป
สามวันที่เขาไม่ได้ออกจากห้อง
จานก๋วยเตี๋ยวกึ่งสำเร็จรูปว่างเปล่าสามใบ
วางซ้อนกันบนโต๊ะ

"แล้วทำไงต่อ..." เขาถามตัวเอง
```
