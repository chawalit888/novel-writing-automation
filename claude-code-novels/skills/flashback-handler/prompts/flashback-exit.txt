# Flashback Exit Prompt (ออกจาก Flashback)

## Context Variables
- {{flashback_ending}}: ส่วนจบของ flashback
- {{present_scene}}: ฉากปัจจุบันที่จะกลับไป
- {{emotional_state}}: สภาพอารมณ์หลัง flashback
- {{exit_type}}: วิธีออก (snap, fade, interrupt)
- {{impact}}: ผลกระทบต่อตัวละคร

คุณคือผู้เชี่ยวชาญด้านการเขียน Flashback สร้างการออกจากความทรงจำอย่างราบรื่น

## เทคนิค Exit

### 1. Snap Back (ดึงกลับทันที)
```
ใช้เมื่อ: ถูกขัดจังหวะ, ต้องกลับเร็ว

[Flashback จบ]
"...และนั่นคือครั้งสุดท้ายที่เขาเห็นเธอ"

"เฮ้!"

เขาสะดุ้ง กลับมาสู่ปัจจุบัน
"อะไร?"
"ฟังอยู่ไหม?"
"อ้อ... ขอโทษ"
```

### 2. Fade Out (ค่อยๆ จาง)
```
ใช้เมื่อ: อารมณ์ลึก, ต้องการความนุ่มนวล

[Flashback จบ]
"...เธอยิ้ม แล้วทุกอย่างก็จางหายไป"

ภาพนั้นค่อยๆ เลือนลาง
เหลือเพียงความรู้สึกอบอุ่น
ที่ยังคงค้างอยู่ในอก

เธอลืมตา
อยู่ในห้องมืด คนเดียว
```

### 3. Parallel Exit (จบคู่ขนาน)
```
ใช้เมื่อ: ต้องการเชื่อมอดีต-ปัจจุบัน

[Flashback]
"ฉันจะรออยู่ตรงนี้" เธอพูด

[ปัจจุบัน]
และเธอก็ยังรออยู่
สิบปีผ่านไป
ยังคงรอ
```

### 4. Physical Return (กลับทางกาย)
```
ใช้เมื่อ: เน้น physical presence

[Flashback จบ]
"...แล้วทุกอย่างก็มืดลง"

เขาลืมตา
แสงไฟจ้าบาดตา
กลิ่นยาฆ่าเชื้อ
โรงพยาบาล
เขาอยู่ที่นี่ ปัจจุบัน
```

### 5. Emotional Bridge (สะพานอารมณ์)
```
ใช้เมื่อ: อารมณ์ต่อเนื่อง

[Flashback]
น้ำตาไหลริน...

[ปัจจุบัน]
...และน้ำตายังคงไหล
เหมือนไม่เคยหยุด
ตั้งแต่วันนั้น
```

## Landing Techniques

### Emotional Landing
```
หลังออกจาก flashback:

1. Physical State
   "เธอพบว่ามือตัวเองสั่น"

2. Emotional Realization
   "ความเจ็บปวดนั้นยังไม่หายไป"

3. Present Action
   "เธอลุกขึ้น พร้อมเผชิญหน้าอีกครั้ง"
```

### Contrast Landing
```
[Flashback: ความสุข]
"วันนั้นเราหัวเราะด้วยกัน"

[ปัจจุบัน: ความเศร้า]
แต่วันนี้
มีเพียงเสียงร้องไห้
```

### Resolution Landing
```
[Flashback: ความสับสน]
"เธอไม่เข้าใจว่าทำไมเขาถึงจากไป"

[ปัจจุบัน: เข้าใจ]
ตอนนี้เธอเข้าใจแล้ว
เขาไม่มีทางเลือก
```

## การเชื่อมโยงอดีต-ปัจจุบัน

### Theme Connection
```
[Flashback theme: การสูญเสีย]
"เขาสูญเสียพ่อไปในวันนั้น"

[ปัจจุบัน: กลัวสูญเสียอีก]
นั่นเป็นเหตุผลที่เขากลัว
กลัวที่จะรักใครอีก
```

### Object Connection
```
[Flashback: ได้รับแหวน]
"เก็บมันไว้ให้ดี" แม่พูด

[ปัจจุบัน: ยังมีแหวน]
เธอกำแหวนไว้แน่น
สิ่งเดียวที่เหลือจากแม่
```

### Character Growth
```
[Flashback: อ่อนแอ]
"เธอหนีไป ไม่กล้าเผชิญหน้า"

[ปัจจุบัน: แข็งแกร่งขึ้น]
แต่ตอนนี้ต่างออกไป
เธอไม่หนีอีกแล้ว
```
