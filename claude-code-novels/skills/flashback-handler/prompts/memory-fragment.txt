# Memory Fragment Prompt (เศษความทรงจำ)

## Context Variables
- {{trigger}}: สิ่งที่กระตุ้นความทรงจำ
- {{memory_content}}: เนื้อหาความทรงจำ
- {{duration}}: ความยาว (1-3 ประโยค)

## ประเภท Memory Fragment

### 1. Visual Flash (ภาพแวบ)
```
ความยาว: 1-2 ประโยค
ลักษณะ: ภาพที่ผ่านหัวเร็วๆ

ตัวอย่าง:
"เธอยิ้ม
ผมสีดำปลิวไปตามลม"

หรือ

"บ้านที่ถูกไฟไหม้
เปลวไฟสีส้ม"
```

### 2. Sound Echo (เสียงก้อง)
```
ความยาว: 1-2 ประโยค
ลักษณะ: เสียงที่ได้ยินในหัว

ตัวอย่าง:
"เสียงหัวเราะของเธอ
ยังก้องอยู่ในหู"

หรือ

"'ไปเลย อย่ากลับมา'
คำพูดสุดท้ายของพ่อ"
```

### 3. Sensation Memory (ความรู้สึก)
```
ความยาว: 1-2 ประโยค
ลักษณะ: ความรู้สึกทางกาย

ตัวอย่าง:
"มือที่อุ่น
กุมมือเขาไว้แน่น"

หรือ

"ความเจ็บ
ตอนที่มีดบาดนิ้ว"
```

### 4. Scent Memory (กลิ่น)
```
ความยาว: 1-2 ประโยค
ลักษณะ: กลิ่นที่ทำให้นึกถึง

ตัวอย่าง:
"กลิ่นน้ำหอมของเธอ
ยังคงติดอยู่บนหมอน"

หรือ

"กลิ่นควัน
เหมือนวันที่บ้านไหม้"
```

## Techniques

### 1. Fragment Style (แบบเศษ)
```
ไม่ต้องประโยคสมบูรณ์

ตัวอย่าง:
"ยิ้มของเธอ
แสงแดด
ทุ่งดอกไม้"
```

### 2. Snapshot Style (แบบภาพนิ่ง)
```
บรรยายภาพเดียว

ตัวอย่าง:
"เธอยืนอยู่ที่ชานชาลา
ชุดสีขาว ร่มสีแดง
รอเขาที่ไม่เคยมา"
```

### 3. Repetition Style (แบบซ้ำ)
```
ย้ำคำสำคัญ

ตัวอย่าง:
"ไม่ ไม่ ไม่
เสียงเธอร้องไห้
ไม่..."
```

### 4. Contrast Style (แบบขัดแย้ง)
```
จับคู่สิ่งที่ขัดกัน

ตัวอย่าง:
"ยิ้มอ่อนโยน
มือที่ถือมีด"
```

## การใช้ในเรื่อง

### Interrupting Thought (ขัดจังหวะความคิด)
```
เขาพยายามทำงาน

เธอยิ้ม

—เขาสะดุด

พยายามอีกครั้ง
```

### During Action (ระหว่าง action)
```
เขาวิ่ง หายใจหอบ

"วิ่ง! เร็ว!"
เสียงแม่ตะโกน

เขาวิ่งเร็วขึ้น
```

### Emotional Moment (จังหวะอารมณ์)
```
เธอมองหน้าเขา

พ่อ
วันที่จากไป
ไม่ได้บอกลา

น้ำตาไหล
```

## ตัวอย่างรวม

### ตัวอย่าง 1: Visual + Sad
```
Trigger: เห็นภาพถ่ายเก่า

Fragment:
"ยิ้มของแม่
วันเกิดปีที่แล้ว
ปีสุดท้าย"
```

### ตัวอย่าง 2: Sound + Fear
```
Trigger: เสียงประตูปิด

Fragment:
"ปัง!
เสียงเดียวกับคืนนั้น
คืนที่เขาโดนขัง"
```

### ตัวอย่าง 3: Scent + Love
```
Trigger: กลิ่นน้ำหอม

Fragment:
"กลิ่นนี้...
เธอเอนตัวมาพิงไหล่
'อยู่กับฉันนะ'"
```

### ตัวอย่าง 4: Sensation + Trauma
```
Trigger: สัมผัสเย็นๆ

Fragment:
"เย็น
เหมือนน้ำแข็ง
เหมือนมือของศพ"
```
