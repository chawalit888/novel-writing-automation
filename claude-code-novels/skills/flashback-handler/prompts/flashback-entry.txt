# Flashback Entry Prompt (เข้าสู่ Flashback)

## Context Variables
- {{current_scene}}: ฉากปัจจุบัน
- {{flashback_content}}: เนื้อหา Flashback
- {{trigger_type}}: ประเภท trigger (object, sound, sensation, dialogue)
- {{time_ago}}: นานแค่ไหน
- {{emotional_tone}}: อารมณ์ของ flashback

---

## Prompt Template

คุณคือผู้เชี่ยวชาญด้านการเขียน Flashback สร้างการเข้าสู่ความทรงจำอย่างราบรื่น

### ข้อมูล:
- ฉากปัจจุบัน: {{current_scene}}
- เนื้อหา Flashback: {{flashback_content}}
- Trigger: {{trigger_type}}
- เวลาผ่านมา: {{time_ago}}
- อารมณ์: {{emotional_tone}}

---

## เทคนิค Entry ตาม Trigger Type

### Object Trigger (วัตถุ)
```
Pattern:
1. เห็น/สัมผัส วัตถุ
2. อธิบายความสำคัญ
3. เข้า flashback

ตัวอย่าง:
"เธอหยิบแหวนวงนั้นขึ้นมา
แหวนที่เขาสวมให้เธอในวันแต่งงาน

วันนั้น ท้องฟ้าใส
และเขายิ้มอย่างมีความสุขที่สุดในชีวิต..."
```

### Sound Trigger (เสียง)
```
Pattern:
1. ได้ยินเสียง
2. จำเสียงนี้ได้
3. เข้า flashback

ตัวอย่าง:
"เสียงนกหวีดดังมาแต่ไกล
เสียงเดียวกับที่พ่อเคยเป่า

พ่อเป่านกหวีดทุกเย็น
เรียกเขากลับบ้านหลังเล่นจนมืด..."
```

### Sensation Trigger (ความรู้สึก)
```
Pattern:
1. รู้สึกบางอย่าง
2. เหมือนครั้งก่อน
3. เข้า flashback

ตัวอย่าง:
"ลมหนาวพัดผ่านใบหน้า
เย็นเหมือนคืนนั้น
คืนที่เธอยืนรอเขาที่สถานี

สถานีร้างยามดึก
หิมะเริ่มตก..."
```

### Dialogue Trigger (คำพูด)
```
Pattern:
1. มีคนพูดบางอย่าง
2. คำนั้นกระทบความทรงจำ
3. เข้า flashback

ตัวอย่าง:
"'ไม่เป็นไร' เขาพูด

ไม่เป็นไร...
คำเดียวกับที่เธอเคยพูดกับแม่
ครั้งสุดท้ายที่เห็นกัน

'ไม่เป็นไรค่ะแม่ หนูไปเองได้'
ถ้ารู้ว่านั่นคือครั้งสุดท้าย..."
```

## Transition Techniques

### 1. Soft Fade (ค่อยๆ จาง)
```
"เธอหลับตา..."

ภาพนั้นค่อยๆ ปรากฏ
ราวกับหมอกที่จางลง

[Flashback]
```

### 2. Hard Cut (ตัดทันที)
```
"เธอจำได้ทุกอย่าง"

---

สิบปีก่อน

[Flashback]
```

### 3. Sensory Zoom (ซูมเข้าหาประสาทสัมผัส)
```
"กลิ่นดอกไม้ลอยมา"

กลิ่นมะลิ
เหมือนวันนั้น
วันที่...

[Flashback]
```

### 4. Emotional Bridge (สะพานอารมณ์)
```
"หัวใจเธอเจ็บปวด
เจ็บเหมือนครั้งนั้น
ครั้งที่เขาจากไป

วันนั้นฝนตกหนัก..."

[Flashback]
```

## Time Markers

### Direct (บอกตรงๆ)
```
"สิบปีก่อน"
"เมื่อตอนยังเป็นเด็ก"
"ฤดูร้อนปีที่แล้ว"
"คืนนั้น"
```

### Implied (บอกอ้อม)
```
"ตอนที่ผมเธอยังยาว..." (= หลายปีก่อน)
"ก่อนที่ทุกอย่างจะเปลี่ยน..."
"สมัยที่แม่ยังอยู่..."
```
