# Flashback Handler Skill

## Overview
Skill สำหรับเขียน Flashback และความทรงจำให้ราบรื่น ไม่สะดุด

---

## ความสามารถหลัก

### 1. Flashback Entry (เข้าสู่ Flashback)
- เทคนิคการเปิด flashback
- Trigger ที่ทำให้นึกถึงอดีต
- การส่งสัญญาณให้ผู้อ่าน

### 2. Flashback Exit (ออกจาก Flashback)
- การกลับสู่ปัจจุบัน
- การเชื่อมโยงอดีต-ปัจจุบัน
- Emotional landing

### 3. Memory Fragment (เศษความทรงจำ)
- ความทรงจำสั้นๆ
- ภาพแฟลชที่ผ่านมา
- การใช้ sensory trigger

---

## ประเภท Flashback

### 1. Full Scene Flashback (ฉากเต็ม)
```
ใช้เมื่อ: ต้องการเล่าเหตุการณ์สำคัญในอดีต

ความยาว: หลายย่อหน้าถึงหลายหน้า
Tense: มักใช้ Past (อดีต)

ตัวอย่าง:
[ปัจจุบัน]
เธอมองรูปถ่ายเก่า ภาพวันที่พวกเขายังมีความสุข

[Flashback เริ่ม]
สิบปีก่อน...

"มาเร็ว! พระอาทิตย์กำลังตก!" เขาตะโกน
เธอวิ่งไปหาเขาที่ชายหาด
[ฉากดำเนินต่อ...]

[กลับปัจจุบัน]
เธอวางรูปลง น้ำตาไหล
```

### 2. Memory Flash (ความทรงจำวาบ)
```
ใช้เมื่อ: ต้องการภาพสั้นๆ ผ่านหัว

ความยาว: 1-3 ประโยค
Tense: Past หรือ Fragment

ตัวอย่าง:
เสียงหัวเราะของเธอ
ผมสีดำที่ปลิวไปตามลม
วันนั้นที่ชายหาด

เขาสะดุ้งกลับมาสู่ปัจจุบัน
```

### 3. Sensory Flashback (ประสาทสัมผัสดึงความทรงจำ)
```
ใช้เมื่อ: กลิ่น/เสียง/รส ทำให้นึกถึงอดีต

ตัวอย่าง:
กลิ่นกาแฟลอยมา
และทันใดนั้น เขาก็นึกถึงร้านเล็กๆ
ที่เขาเคยนั่งกับพ่อทุกวันอาทิตย์
ก่อนที่พ่อจะ...
```

### 4. Dream/Nightmare Flashback
```
ใช้เมื่อ: แสดงความทรงจำที่หลอกหลอน

ตัวอย่าง:
เขาหลับตา

และก็อยู่ที่นั่นอีกครั้ง
วันที่ไฟไหม้บ้าน
เสียงกรีดร้อง
ควันที่ทำให้หายใจไม่ออก

"ลุก!"

เขาตื่นขึ้น เหงื่อโชก
```

## เทคนิคเข้าสู่ Flashback

### 1. Trigger Object (วัตถุกระตุ้น)
```
"เธอหยิบพวงกุญแจขึ้นมา
พวงกุญแจที่มีจี้รูปหัวใจ
เขาให้มันในวันนั้น..."

[เข้า Flashback]
```

### 2. Trigger Sound (เสียงกระตุ้น)
```
"เสียงเพลงนั้นดังขึ้นจากวิทยุ
เพลงที่เขาเคยร้องให้ฟัง..."

[เข้า Flashback]
```

### 3. Trigger Sensation (ความรู้สึกกระตุ้น)
```
"ลมเย็นพัดมา
เหมือนคืนนั้นที่เขาจากไป..."

[เข้า Flashback]
```

### 4. Direct Statement (บอกตรงๆ)
```
"เธอนึกถึงวันนั้น"

หรือ

"สิบปีก่อน เมื่อเธอยังเป็นเด็ก..."
```

### 5. Dialogue Trigger (บทสนทนากระตุ้น)
```
"เธอเหมือนแม่มาก" เขาพูด

แม่...
เธอนึกถึงวันสุดท้ายที่เห็นแม่

[เข้า Flashback]
```

## เทคนิคออกจาก Flashback

### 1. Snap Back (ดึงกลับ)
```
[จบ Flashback]

"เฮ้ ฟังอยู่ไหม?"

เธอกระพริบตา กลับมาสู่ปัจจุบัน
"อะไรนะ?"
```

### 2. Fade Out (ค่อยๆ จาง)
```
[จบ Flashback]

ภาพนั้นค่อยๆ จางลง
เหลือเพียงความว่างเปล่า

และเธอก็อยู่ที่นี่ ในห้องมืด คนเดียว
```

### 3. Parallel Ending (จบคู่ขนาน)
```
[Flashback]
"ฉันจะไม่ลืมเธอ" เขาพูด

[ปัจจุบัน]
"ฉันลืมไม่ได้" เธอกระซิบกับตัวเอง
```

### 4. Emotional Bridge (สะพานอารมณ์)
```
[Flashback]
วันนั้นเธอร้องไห้...

[ปัจจุบัน]
...และวันนี้ก็ยังร้องไห้เหมือนกัน
```

## โครงสร้าง Flashback ที่ดี

```
┌─────────────────────────────────────────────────────────────┐
│  1. Present Anchor (ยึดปัจจุบัน)                           │
│     - สถานการณ์ปัจจุบันชัดเจน                              │
│     - ผู้อ่านรู้ว่าอยู่ไหน                                  │
│                                                             │
│  2. Trigger (ตัวกระตุ้น)                                   │
│     - อะไรทำให้นึกถึงอดีต                                  │
│     - เชื่อมโยงกับฉากปัจจุบัน                              │
│                                                             │
│  3. Transition In (เข้า Flashback)                        │
│     - ส่งสัญญาณชัดเจน                                      │
│     - ผู้อ่านรู้ว่ากำลังเข้าอดีต                            │
│                                                             │
│  4. Flashback Content (เนื้อหา)                           │
│     - เล่าสิ่งที่ต้องรู้                                    │
│     - ไม่ยาวเกินไป                                         │
│                                                             │
│  5. Transition Out (ออก Flashback)                        │
│     - ส่งสัญญาณชัดเจน                                      │
│     - เชื่อมกลับปัจจุบัน                                   │
│                                                             │
│  6. Present Impact (ผลกระทบ)                              │
│     - Flashback ส่งผลอย่างไร                              │
│     - ตัวละครเปลี่ยนไหม                                    │
└─────────────────────────────────────────────────────────────┘
```

## ปัญหาที่พบบ่อย

### 1. ไม่รู้ว่าอยู่ไหน
```
ปัญหา: ผู้อ่านสับสนว่าอยู่ปัจจุบันหรืออดีต

แก้ไข:
- ใช้ visual break (---)
- บอก time marker ชัดเจน
- ใช้ tense ต่างกัน
```

### 2. Flashback ยาวเกินไป
```
ปัญหา: ลืมเรื่องปัจจุบันไปแล้ว

แก้ไข:
- ตัดให้สั้น
- เล่าเฉพาะที่จำเป็น
- แบ่งเป็น flashback หลายครั้ง
```

### 3. ไม่มี Purpose
```
ปัญหา: Flashback ไม่เพิ่มอะไรให้เรื่อง

แก้ไข:
- ถามว่า "ต้องรู้เรื่องนี้จริงๆ ไหม?"
- ถ้าตัดออกแล้วเรื่องยังสมบูรณ์ = ไม่จำเป็น
```

### 4. Trigger ไม่สมเหตุสมผล
```
ปัญหา: อยู่ดีๆ นึกถึงอดีตไม่มีที่มา

แก้ไข:
- หา trigger ที่เหมาะสม
- เชื่อมโยงกับสถานการณ์ปัจจุบัน
```

## Checklist

- [ ] มี Trigger ที่สมเหตุสมผล
- [ ] ส่งสัญญาณเข้า Flashback ชัดเจน
- [ ] ผู้อ่านไม่สับสนว่าอยู่ไหน
- [ ] Flashback มี purpose
- [ ] ไม่ยาวเกินไป
- [ ] ส่งสัญญาณออก Flashback ชัดเจน
- [ ] มี Impact ต่อปัจจุบัน

---

## Tips

### DO:
- ให้ Flashback มี purpose ชัดเจน
- ใช้ sensory details
- เชื่อมโยงกับอารมณ์ปัจจุบัน
- ส่งสัญญาณให้ผู้อ่าน

### DON'T:
- Flashback ยาวเกินไป
- ใช้ Flashback บอก backstory ทั้งหมด
- หลาย Flashback ต่อกัน
- Flashback ที่ไม่ส่งผลต่อปัจจุบัน
