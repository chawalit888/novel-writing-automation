# Countdown Prompt (นับถอยหลัง/หมดเวลา)

## Context Variables
- {{genre}}: แนวนิยาย
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{deadline}}: เส้นตาย (เวลา/เหตุการณ์)
- {{stakes}}: อะไรจะเสียถ้าไม่ทัน
- {{obstacles}}: อุปสรรคที่ขวางทาง
- {{time_remaining}}: เวลาที่เหลือ
- {{outcome}}: ผลลัพธ์ (ทัน/ไม่ทัน/เกือบทัน)

---

## Prompt Template

คุณคือนักเขียนฉาก Countdown มืออาชีพ เชี่ยวชาญในการสร้างความลุ้นจากเวลาที่กำลังหมด

### ข้อมูลฉาก:
- แนว: {{genre}}
- ตัวละคร: {{characters}}
- เส้นตาย: {{deadline}}
- Stakes: {{stakes}}
- อุปสรรค: {{obstacles}}
- เวลาที่เหลือ: {{time_remaining}}
- ผลลัพธ์: {{outcome}}

## ประเภท Countdown

### 1. Literal Countdown (นับจริง)
```
แสดงเวลาที่เหลือชัดเจน

"นาฬิกาบนระเบิดแสดง 00:02:47
สองนาทีสี่สิบเจ็ดวินาที

เธอมองสายไฟ สามสี แดง น้ำเงิน เขียว

00:02:35

ตัดสายไหน?"
```

### 2. Event Deadline (เส้นตายเหตุการณ์)
```
ต้องทำให้ทันก่อนเหตุการณ์บางอย่าง

"งานแต่งงานเริ่มบ่ายสามโมง
ตอนนี้บ่ายสองโมงครึ่ง

และเธอยังอยู่อีกฝั่งเมือง
รถติดยาวเหยียด"
```

### 3. Race Against Others (แข่งกับคนอื่น)
```
ต้องถึงก่อนคนอื่น

"เขารู้ว่าฆาตกรกำลังเดินทางไปหาเธอ
และเขาต้องไปถึงก่อน

แต่ฆาตกรออกไปก่อนสิบนาที"
```

### 4. Physical Limitation (ข้อจำกัดทางกาย)
```
เวลาจำกัดจากสภาพร่างกาย

"ออกซิเจนในถังเหลือ 15 นาที
และเขาอยู่ลึกลงไปสามสิบเมตร"
```

## เทคนิคการเขียน Countdown

### 1. Time Markers (บอกเวลาที่เหลือ)
```
"เหลืออีก 10 นาที"
...
"เหลืออีก 5 นาที"
...
"เหลืออีก 1 นาที"
...
"30 วินาที"
...
"10... 9... 8..."
```

### 2. Increasing Pacing (เร่งจังหวะ)
```
ตอนเหลือเวลามาก: ประโยคยาว บรรยายละเอียด
ตอนใกล้หมดเวลา: ประโยคสั้นลงเรื่อยๆ

"เหลืออีก 10 นาที เธอยังพอมีเวลาคิด หาทางออก"

"5 นาที เธอต้องเร็วกว่านี้"

"2 นาที"

"หนึ่งนาที"

"ไม่ทัน"

"ต้องทัน"

"30"

"20"

"10"

"วิ่ง!"
```

### 3. Obstacles (อุปสรรค)
```
เพิ่มอุปสรรคเมื่อใกล้ถึงเวลา

"เหลืออีก 3 นาที เกือบถึงแล้ว

แล้วเครื่องบินก็ขึ้น

เธอเห็นมันค่อยๆ ลอยสูงขึ้น
พาเขาไป... ตลอดกาล"
```

### 4. False Hope/Despair (หลอกอารมณ์)
```
ทำให้คิดว่าจะทัน แล้วเกิดอุปสรรค
หรือทำให้คิดว่าจะไม่ทัน แล้วปาฏิหาริย์เกิดขึ้น

"ไม่ทัน... รถไฟออกไปแล้ว

เธอยืนหอบ น้ำตาไหล

แล้วรถไฟก็หยุด

ประกาศดังขึ้น 'ขออภัย มีปัญหาทางเทคนิค
รถไฟจะจอดอีก 5 นาที'

ห้านาที!"
```
