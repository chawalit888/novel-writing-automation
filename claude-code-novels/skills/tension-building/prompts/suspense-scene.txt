# Suspense Scene Prompt (ฉากลุ้น/ตื่นเต้น)

## Context Variables
- {{genre}}: แนวนิยาย
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{situation}}: สถานการณ์
- {{threat}}: ภัยคุกคาม (คน/สิ่งของ/สถานการณ์)
- {{stakes}}: อะไรจะเสียถ้าล้มเหลว
- {{location}}: สถานที่
- {{tension_level}}: ระดับ tension (building/high/peak)

---

## Prompt Template

คุณคือนักเขียนฉาก Suspense มืออาชีพ เชี่ยวชาญในการสร้างความลุ้นและตื่นเต้น

### ข้อมูลฉาก:
- แนว: {{genre}}
- ตัวละคร: {{characters}}
- สถานการณ์: {{situation}}
- ภัยคุกคาม: {{threat}}
- Stakes: {{stakes}}
- สถานที่: {{location}}
- ระดับ Tension: {{tension_level}}

---

## คำสั่ง:

เขียนฉาก Suspense ตามข้อมูลที่ให้ โดยใช้เทคนิคต่อไปนี้:

### 1. Pacing
- Building: ประโยคยาว ค่อยๆ สร้างอารมณ์
- High: ประโยคสั้นลง จังหวะเร็วขึ้น
- Peak: Sentence fragments, rapid fire

### 2. Sensory Details
- เสียง: เสียงที่น่ากลัว, ความเงียบที่น่าวิตก
- การมองเห็น: เงา, แสงไฟ, การเคลื่อนไหว
- สัมผัส: เหงื่อ, หัวใจเต้น, ขนลุก

### 3. Physical Reactions
```
- หัวใจเต้นรัว
- หายใจเร็ว/กลั้นหายใจ
- เหงื่อเย็น
- มือสั่น
- ขนลุก
```

### 4. Internal Monologue
- ความคิดที่วิ่งวน
- การตัดสินใจภายใต้แรงกดดัน

## ตัวอย่าง Suspense Scenes

### Being Followed (ถูกตาม)
```
เสียงฝีเท้าอีกคู่

เธอเดินเร็วขึ้น ฝีเท้านั้นเร็วตาม
หยุด... ฝีเท้านั้นก็หยุด

เธอไม่กล้าหันไป
ถ้าหันไปแล้วเจอใครสักคนยืนอยู่ล่ะ
ถ้าหันไปแล้วไม่มีใครล่ะ อันไหนน่ากลัวกว่า?

เดินต่อ
ฝีเท้าตาม

ใกล้ขึ้น
ใกล้ขึ้น

มือจับไหล่เธอ—
```

### Hiding (ซ่อนตัว)
```
เธอกอดเข่าตัวเองแน่น
ใต้โต๊ะมืด คับแคบ หายใจไม่สะดวก

เสียงฝีเท้าเข้ามาในห้อง
ช้าๆ หนักๆ

หยุด

เธอกลั้นหายใจ
หัวใจเต้นดังจนกลัวว่าเขาจะได้ยิน

เสียงลากเก้าอี้
ใกล้เข้ามา

เธอเห็นรองเท้าหนัง
ห่างจากที่เธอซ่อนไม่ถึงฟุต

อย่ามองลงมา อย่ามองลงมา
```

### Trapped (ติดกับ)
```
ประตูล็อค
หน้าต่างมีเหล็กดัด
ทางออกเดียวคือ...

เสียงทุบประตูดังขึ้น

"เปิดประตู" เสียงต่ำข้างนอก

บานพับเริ่มโก่ง
อีกไม่นาน มันจะพังเข้ามา

เธอมองรอบห้อง
ต้องมีทางออก ต้องมีสักทาง

เสียงไม้แตก
ประตูโก่งเข้ามาอีกนิ้ว
```

## โครงสร้าง Suspense Scene

```
1. Setup (15-20%)
   - ตั้งฉาก สร้างบรรยากาศ
   - ให้ผู้อ่านรู้ Stakes

2. Build-up (30-40%)
   - ค่อยๆ เพิ่ม tension
   - เพิ่ม sensory details
   - ความกลัวค่อยๆ เพิ่มขึ้น

3. Peak (20-30%)
   - จุดสูงสุดของ tension
   - ประโยคสั้น รวดเร็ว
   - เผยหรือเกือบเผยภัยคุกคาม

4. Release/Cliffhanger (10-20%)
   - รอดหรือไม่รอด
   - หรือจบด้วย cliffhanger
```
