# Mystery Reveal Prompt (ค่อยๆ เปิดเผยความลับ)

## Context Variables
- {{genre}}: แนวนิยาย
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{mystery}}: ปริศนา/ความลับที่จะเปิดเผย
- {{clues_given}}: เบาะแสที่ให้ไปแล้ว
- {{reveal_level}}: ระดับการเปิดเผย (hint/partial/full)
- {{reader_knowledge}}: ผู้อ่านรู้อะไรแล้วบ้าง

---

## Prompt Template

คุณคือนักเขียนฉาก Mystery Reveal มืออาชีพ เชี่ยวชาญในการเปิดเผยความลับอย่างมีศิลปะ

### ข้อมูลฉาก:
- แนว: {{genre}}
- ตัวละคร: {{characters}}
- ปริศนา/ความลับ: {{mystery}}
- เบาะแสที่ให้ไปแล้ว: {{clues_given}}
- ระดับการเปิดเผย: {{reveal_level}}
- ผู้อ่านรู้อะไรแล้ว: {{reader_knowledge}}

---

## ระดับการเปิดเผย

### Hint (บอกใบ้)
```
ให้ clue เล็กๆ ที่ผู้อ่านอาจสังเกตหรือไม่สังเกต
ตัวละครไม่รู้ แต่ผู้อ่านที่ช่างสังเกตอาจจับได้

ตัวอย่าง:
"เธอสังเกตว่าเขาไม่เคยพูดถึงพ่อของเขาเลย"
```

### Partial (เปิดเผยบางส่วน)
```
เปิดเผยส่วนหนึ่งของความจริง
สร้างคำถามใหม่ที่มากกว่าคำตอบ

ตัวอย่าง:
"เธอพบภาพถ่ายในลิ้นชักโต๊ะเขา
ภาพผู้หญิงคนหนึ่งที่เธอไม่เคยเห็น
แต่ผู้หญิงคนนั้นมีตาเหมือนเขาทุกประการ"
```

### Full (เปิดเผยทั้งหมด)
```
เปิดเผยความจริงทั้งหมด
แต่ยังอาจมีผลกระทบที่ต้องติดตาม

ตัวอย่าง:
"'เธอไม่ใช่ลูกแท้ๆ ของแม่'
แม่พูด เสียงสั่นเครือ
'แม่รับเธอมาเลี้ยงตอนเธออายุหนึ่งขวบ
พ่อแม่จริงๆ ของเธอ... พวกเขาตายในอุบัติเหตุ'"
```

---

## เทคนิคการเปิดเผย

### 1. The Puzzle Piece (ชิ้นส่วนปริศนา)
```
เปิดเผยทีละชิ้น ให้ผู้อ่านประกอบเอง

"เธอพบ:
- จดหมายที่ไม่มีชื่อผู้ส่ง
- ภาพถ่ายที่ถูกฉีกครึ่ง
- แหวนที่แม่ไม่เคยใส่

ทุกอย่างต้องเชื่อมโยงกันอย่างใดอย่างหนึ่ง"
```

### 2. The Realization (ตระหนักรู้)
```
ตัวละครประกอบปริศนาเข้าด้วยกัน

"แล้วทุกอย่างก็ประกอบเข้าด้วยกัน
การหายไปของพ่อ
ผู้หญิงที่มาหาแม่ทุกเดือน
เงินที่มาจากไหนก็ไม่รู้

เขาไม่ได้ตาย เขาแค่..."
```

### 3. The Confession (การสารภาพ)
```
ตัวละครอื่นเล่าความจริง

"'ฟังให้ดี เพราะฉันจะพูดครั้งเดียว'
ลุงวิชัยนั่งลง เหนื่อยล้า

'พ่อของเธอไม่ได้ตายในอุบัติเหตุ
เขาถูกฆ่า และฉันรู้ว่าใครฆ่า'"
```

### 4. The Discovery (การค้นพบ)
```
ตัวละครค้นพบหลักฐานด้วยตัวเอง

"ห้องลับหลังตู้หนังสือ
เธอไม่เคยรู้มาก่อนว่ามันมีอยู่

ข้างในเต็มไปด้วยเอกสาร ภาพถ่าย และ...
รายชื่อ

ชื่อของเธออยู่ในรายชื่อนั้น
ถูกขีดฆ่าด้วยหมึกแดง"
```

---

## โครงสร้าง Mystery Reveal

```
1. Setup (10-15%)
   - สถานการณ์ก่อนการเปิดเผย
   - ตัวละครกำลังทำอะไร

2. Discovery/Trigger (15-20%)
   - สิ่งที่ทำให้เริ่มรู้ความจริง
   - เบาะแสที่นำไปสู่การค้นพบ

3. Reveal (40-50%)
   - เปิดเผยตามระดับที่กำหนด
   - ให้เวลาผู้อ่านซึมซับ

4. Reaction/Implication (20-30%)
   - ปฏิกิริยาของตัวละคร
   - ผลกระทบ/คำถามใหม่
```

## ตัวอย่าง Mystery Reveal

### Identity Reveal
```
"ภาพถ่ายเก่าร่วงจากหนังสือ

เธอก้มลงเก็บ แล้วหยุดนิ่ง

ในภาพคือหญิงสาวสวมชุดเจ้าสาว
ยิ้มให้กล้องอย่างมีความสุข
เคียงข้างชายหนุ่มที่เธอไม่รู้จัก

แต่ใบหน้าหญิงสาวคนนั้น...
เหมือนกับใบหน้าที่เธอเห็นในกระจกทุกวัน

เธอพลิกดูหลังภาพ
มีข้อความเขียนด้วยลายมือ

'งานแต่งงานของเรา 2520
รักเธอตลอดไป - ว.'

แม่เกิดปี 2530
แม่จะแต่งงานตอนอายุติดลบได้อย่างไร?

แล้วหญิงสาวในภาพคือใคร?"
```

### Truth About Past
```
"'ลุงรู้มาตลอดใช่ไหม'

ลุงวิชัยถอนหายใจ นั่งลง

'พ่อของเธอไม่ได้ตายในอุบัติเหตุ'

เธอรู้สึกเหมือนโลกหยุดหมุน

'แล้วเขา...'

'เขาหนีไป' ลุงพูด เสียงเหนื่อยล้า
'หนีจากอะไรบางอย่างที่น่ากลัวกว่าความตาย'

'อะไร?'

ลุงส่ายหน้า
'เรื่องนั้น ลุงสัญญาว่าจะไม่พูด'

'แต่ลุง—'

'แต่ลุงบอกได้อย่างเดียว'
เขามองเข้าไปในตาเธอ
'อย่าเชื่อใครในบ้านนี้ โดยเฉพาะแม่ของเธอ'"
```

## Checklist

- [ ] ระดับการเปิดเผยเหมาะสม
- [ ] ไม่เปิดเผยมากเกินไปในครั้งเดียว
- [ ] มี Clue ที่ fair (ผู้อ่านสามารถเดาได้ถ้าสังเกตดี)
- [ ] มีปฏิกิริยาของตัวละครที่สมจริง
- [ ] สร้างคำถามใหม่ (ถ้าไม่ใช่ full reveal)
- [ ] มี Emotional impact
