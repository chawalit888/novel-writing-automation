# Paywall Hook Generator (ตอนที่ 10)

## ความสำคัญ
ตอนที่ 10 คือจุดสำคัญที่สุดของ Freemium Model
- ผู้อ่านอ่านฟรีมา 10 ตอน
- ถ้าจะอ่านต่อต้องจ่ายเงิน
- Cliffhanger ต้องแรงพอที่จะทำให้ "ยอมจ่าย"

---

## Context Variables
- {{genre}}: แนวนิยาย
- {{novel_title}}: ชื่อเรื่อง
- {{characters}}: ตัวละครหลัก
- {{plot_so_far}}: สรุปเรื่องตอน 1-9
- {{main_conflict}}: ความขัดแย้งหลัก
- {{secrets}}: ความลับที่ยังไม่เปิดเผย
- {{romantic_status}}: สถานะความรักปัจจุบัน

---

## Prompt Template

คุณคือผู้เชี่ยวชาญในการสร้าง PAYWALL HOOK - Cliffhanger ที่แรงที่สุดเพื่อให้ผู้อ่านยอมจ่ายเงินซื้อตอนต่อไป

### ข้อมูลเรื่อง:
- ชื่อเรื่อง: {{novel_title}}
- แนว: {{genre}}
- ตัวละครหลัก: {{characters}}
- ความขัดแย้งหลัก: {{main_conflict}}
- สถานะความรัก: {{romantic_status}}

### สรุปตอน 1-9:
{{plot_so_far}}

### ความลับที่ยังไม่เปิดเผย:
{{secrets}}

---

## คำสั่ง:

สร้าง PAYWALL HOOK สำหรับจบตอนที่ 10 โดยใช้สูตร:

```
[Revelation] + [Danger/Decision] + [Emotional Peak] + [Unanswered Question]
```

### ต้องมี:
1. **Revelation** - เปิดเผยความลับสำคัญ (แต่ยังไม่หมด)
2. **Crisis** - ตัวละครตกอยู่ในจุดวิกฤต
3. **Emotional Peak** - อารมณ์ถึงจุดสูงสุด
4. **Unanswered Question** - ยังมีคำถามค้างคา

### ห้าม:
- จบแบบลงตัว
- ไม่มีความตึงเครียด
- เปิดเผยทุกอย่างหมด

---

## รูปแบบ Output:

```
## PAYWALL HOOK สำหรับ "{{novel_title}}" ตอนที่ 10

### Setup (ก่อนถึง Cliffhanger):
[2-3 ประโยค อธิบายสถานการณ์ที่นำไปสู่ cliffhanger]

### The Hook:
[เนื้อหา cliffhanger 5-10 ประโยค]

### องค์ประกอบ:
- Revelation: [ความลับที่เปิดเผย]
- Crisis: [วิกฤตที่เกิดขึ้น]
- Emotional Peak: [อารมณ์ที่ถึงจุดสูงสุด]
- Unanswered Question: [คำถามที่ผู้อ่านต้องรู้คำตอบ]

### ทำไมต้องอ่านต่อ:
[3 เหตุผลว่าทำไมผู้อ่านจะยอมจ่ายเงิน]

### ความแรง: ★★★★★
```

## ตัวอย่าง Paywall Hook ตามแนว

### Romantic Comedy:
```
เธอยืนนิ่ง หัวใจหยุดเต้น

ภาพบนจอโปรเจคเตอร์คือรูปเธอกับธนกร กำลังกอดกัน
ต่อหน้าคนทั้งบริษัท

"ผมขอแนะนำ" ธนกรพูด น้ำเสียงเย็นชา
"ผู้หญิงที่จะมาทำลายบริษัทนี้"

มินตราหันไปมองเขา ตาเปล่งประกายน้ำตา
นี่คือคนที่บอกว่ารักเธอเหรอ?

"ธนกร..." เธอเอ่ย

"พี่ธนกรครับ" เสียงใครบางคนแทรกขึ้น
มินตราหันไป และเห็นหญิงสาวในชุดแต่งงานสีขาว
"ลืมไปแล้วเหรอคะ ว่าวันนี้เราแต่งงานกัน"

มินตราเห็นแหวนเพชรบนนิ้วเขา
แหวนที่เธอไม่เคยเห็นมาก่อน

แล้วเธอก็รู้...
เธอไม่เคยรู้จักเขาเลยจริงๆ
```

### BL/GL:
```
"ผมต้องไปแล้ว"

ภูมิหันหลังให้ ไม่กล้ามองตาพี่เวฟ

"ไปไหน" พี่เวฟจับแขนเขา "เราเพิ่ง..."

"มีคนมารับผมแล้ว" ภูมิพูด เสียงสั่น
"คนที่ผมหมั้นหมายไว้"

พี่เวฟแข็งค้าง มือที่จับแขนภูมิหลุดออก

ประตูเปิด หญิงสาวสวยเดินเข้ามา
เธอยิ้มอ่อนโยน ก่อนจูบแก้มภูมิ

"คิดถึงจังเลยค่ะ ที่รัก"

ภูมิไม่ได้หลบ
แต่ตาของเขาจ้องมองพี่เวฟ
เต็มไปด้วยความเจ็บปวด

"ผมขอโทษ" ภูมิกระซิบ
ก่อนเดินออกไปพร้อมหญิงสาว

พี่เวฟยืนนิ่ง
และเห็นกระดาษร่วงจากกระเป๋าภูมิ

เขาก้มเก็บขึ้นมา
มันคือใบตรวจโรค

และชื่อโรคที่เขียนอยู่...
ทำให้ขาเขาอ่อน
```

### Mystery/Thriller:
```
"ฆาตกรคือ..."

นักสืบหยุดพูดกลางคัน
เธอเห็นบางอย่างในกล้องวงจรปิด

ภาพย้อนหลังแสดงให้เห็นคืนเกิดเหตุ
ชายในเสื้อฮู้ดเดินเข้าห้องเหยื่อ

เขาถอดฮู้ดออก
และหน้าที่ปรากฏ...

เป็นหน้าของเธอเอง

"เป็นไปไม่ได้" เธอกระซิบ

แต่วิดีโอไม่โกหก
เธอเห็นตัวเองยกมีดขึ้น
เห็นตัวเองแทงเหยื่อซ้ำแล้วซ้ำเล่า
เห็นตัวเองยิ้ม...

"คุณไม่จำเหรอ?" เสียงกระซิบดังข้างหู
"คุณไม่จำว่าคุณทำอะไร?"

เธอหันไป
ไม่มีใคร

แต่เธอเห็นเงาสะท้อนในกระจก
เงาที่ยิ้มให้เธอ
ทั้งที่เธอไม่ได้ยิ้ม
```

## Checklist ก่อนส่ง Paywall Hook

- [ ] มี Revelation สำคัญ
- [ ] ตัวละครอยู่ในจุดวิกฤต
- [ ] อารมณ์ถึงจุดสูงสุด
- [ ] ยังมีคำถามค้างคา (ไม่เปิดเผยหมด)
- [ ] ไม่จบแบบลงตัว
- [ ] ผู้อ่านจะรู้สึก "ต้องอ่านต่อ"
- [ ] ความแรง ★★★★★

---

## คำถามทดสอบ

ถ้าผู้อ่านอ่านจบตอน 10 แล้ว:
1. เขาจะพูดว่า "ฉันต้องรู้ว่าเกิดอะไรขึ้น!" ไหม?
2. เขายอมจ่าย 10-20 บาทเพื่ออ่านต่อไหม?
3. เขาจะบอกเพื่อนว่า "ต้องอ่านเรื่องนี้!" ไหม?

ถ้าตอบ "ใช่" ทั้งหมด = Paywall Hook สมบูรณ์!
