# Hook & Cliffhanger Generator Prompt

## Context Variables
- {{genre}}: แนวนิยาย (romantic-comedy, bl-gl, fantasy, horror, mystery)
- {{chapter_number}}: ตอนที่
- {{chapter_content}}: เนื้อหาบท (ถ้ามี)
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{situation}}: สถานการณ์ปัจจุบัน
- {{hook_type}}: ประเภท hook ที่ต้องการ (opening/cliffhanger)
- {{intensity}}: ความแรง (normal/strong/maximum)

---

## Prompt Template

คุณคือผู้เชี่ยวชาญในการสร้าง Hook และ Cliffhanger สำหรับนิยายไทย

### ข้อมูลบท:
- แนว: {{genre}}
- ตอนที่: {{chapter_number}}
- ตัวละคร: {{characters}}
- สถานการณ์: {{situation}}

### ต้องการ:
- ประเภท: {{hook_type}}
- ความแรง: {{intensity}}

### คำสั่ง:

{% if hook_type == "opening" %}

สร้าง Opening Hook 3 แบบสำหรับเปิดตอน:

1. **Action Hook** - เริ่มด้วยการกระทำ
2. **Dialogue Hook** - เริ่มด้วยบทสนทนา
3. **Shock Hook** - เริ่มด้วยความช็อค

กฎ:
- ดึงดูดตั้งแต่ประโยคแรก
- ไม่มีอารัมภบทยืดยาว
- สร้างคำถามในใจผู้อ่าน
- เหมาะกับแนว {{genre}}

{% elif hook_type == "cliffhanger" %}

สร้าง Cliffhanger 3 แบบสำหรับจบตอน:

1. **Revelation Cliffhanger** - เปิดเผยความลับ
2. **Danger Cliffhanger** - อันตราย
3. **Romantic/Emotional Cliffhanger** - อารมณ์

กฎ:
- ต้องสร้างแรงจูงใจให้อ่านต่อ
- ไม่จบแบบลงตัว
- มีคำถามค้างคาในใจผู้อ่าน
- ความแรงตาม intensity ที่กำหนด

{% endif %}

### รูปแบบ Output:

```
## Hook/Cliffhanger ตัวเลือกที่ 1: [ชื่อ]
[เนื้อหา 2-5 ประโยค]

ความแรง: ★★★★☆
เหมาะกับ: [สถานการณ์]

---

## Hook/Cliffhanger ตัวเลือกที่ 2: [ชื่อ]
...

## Hook/Cliffhanger ตัวเลือกที่ 3: [ชื่อ]
...

## แนะนำ:
[ตัวเลือกไหนเหมาะที่สุดและทำไม]
```

---

## ความแรงตาม Chapter Number (Freemium Model)

| ตอนที่ | ความแรงขั้นต่ำ | หมายเหตุ |
|--------|---------------|---------|
| 1 | ★★★★★ | Opening Hook ต้องแรงมาก |
| 2-9 | ★★★★☆ | Strong Hook ทุกตอน |
| 10 | ★★★★★ | PAYWALL HOOK แรงที่สุด! |
| 11+ | ★★★☆☆ | ปกติ แต่ยังต้องมี |

---

## ตัวอย่างการใช้งาน

### Input:
```
genre: romantic-comedy
chapter_number: 5
characters: มินตรา (นางเอก แม่ค้าส้มตำ), ธนกร (พระเอก CEO)
situation: มินตราเพิ่งรู้ว่าธนกรเป็น CEO ไม่ใช่พนักงานธรรมดา
hook_type: cliffhanger
intensity: strong
```

### Output:
```
## Cliffhanger ตัวเลือกที่ 1: The Revelation
"คุณ... คุณคือประธานบริษัท?"

มินตราถอยหลัง มือสั่นจนถุงส้มตำร่วงลงพื้น
ชายที่เธอด่าทุกวัน ชายที่มานั่งกินส้มตำจานละ 40 บาท
คือมหาเศรษฐีพันล้าน

"ฉันอธิบายได้" เขาก้าวเข้ามา
"อย่าเข้ามา!" เธอตะโกน ตาแดงก่ำ

แล้วเธอก็วิ่งหนีไป โดยไม่รู้ว่า...
เบื้องหลังต้นไม้ มีแฟลชกล้องส่องมาที่เธอ

ความแรง: ★★★★★
เหมาะกับ: จุดเปลี่ยนความสัมพันธ์ + ปมใหม่

---

## แนะนำ:
ตัวเลือกที่ 1 เพราะ:
1. มี emotional peak (ความช็อค + เสียใจ)
2. มี mystery เพิ่ม (ใครถ่ายรูป?)
3. สร้างความสงสัยว่าเธอจะให้อภัยไหม
```
