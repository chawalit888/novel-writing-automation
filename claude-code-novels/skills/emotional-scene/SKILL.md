# Emotional Scene Skill

## Overview
Skill สำหรับเขียนฉากอารมณ์ที่ทรงพลัง ทำให้ผู้อ่านรู้สึกอิน ซึ้ง หรือจิ้นไปกับตัวละคร

---

## 1. ประเภทอารมณ์หลัก (6 แบบ)

### 1.1 ความรัก/โรแมนติก (Romance)
ฉากที่ทำให้ผู้อ่านจิ้น หัวใจเต้นแรง

**สถานการณ์:**
- First meeting chemistry
- Accidental touch
- Almost kiss / First kiss
- Confession
- ดูแลเวลาเจ็บป่วย
- หึงหวง

**Physical Cues:**
```
- หัวใจเต้นแรง
- แก้มร้อนผ่าว
- หลบตา / สบตาค้าง
- มือสั่น
- กลืนน้ำลาย
- หายใจไม่ทั่วท้อง
```

---

### 1.2 ความเศร้า/สูญเสีย (Sadness)
ฉากที่ทำให้ผู้อ่านน้ำตาซึม

**สถานการณ์:**
- การจากลา
- ความตาย
- ผิดหวังในความรัก
- สูญเสียสิ่งสำคัญ
- รู้ความจริงที่เจ็บปวด

**Physical Cues:**
```
- น้ำตาไหล
- เสียงสั่นเครือ
- แน่นหน้าอก
- ขาอ่อน
- หายใจไม่ออก
- มือกำแน่น
```

---

### 1.3 ความโกรธ/แค้น (Anger)
ฉากที่แสดงอารมณ์โกรธอย่างทรงพลัง

**สถานการณ์:**
- ถูกทรยศ
- ถูกดูถูก
- ปกป้องคนรัก
- รู้ความจริงที่ทำให้โกรธ
- การเผชิญหน้า

**Physical Cues:**
```
- กำปั้นบีบแน่น
- ขากรรไกรเกร็ง
- ตาเป็นประกาย
- เสียงดังขึ้น
- หน้าแดง
- ตัวสั่น
```

---

### 1.4 ความกลัว/หวาดผวา (Fear)
ฉากที่สร้างความหวาดกลัว

**สถานการณ์:**
- ตกอยู่ในอันตราย
- เจอสิ่งเหนือธรรมชาติ
- รอคอยข่าวร้าย
- ซ่อนตัว
- ไล่ล่า

**Physical Cues:**
```
- หัวใจเต้นรัว
- เหงื่อเย็น
- ขาแข็ง
- หายใจเร็ว
- กลืนน้ำลายไม่ลง
- ขนลุก
```

---

### 1.5 ความสุข/ดีใจ (Joy)
ฉากแห่งความสุขและความยินดี

**สถานการณ์:**
- ได้รับข่าวดี
- กลับมาเจอกัน
- ความสำเร็จ
- ถูกรัก
- ความฝันเป็นจริง

**Physical Cues:**
```
- ยิ้มกว้าง
- น้ำตาแห่งความสุข
- กระโดดโลดเต้น
- กอดแน่น
- หัวเราะ
- หัวใจเต้นแรง
```

---

### 1.6 ความผิดหวัง/หมดหวัง (Despair)
ฉากที่แสดงความสิ้นหวัง

**สถานการณ์:**
- ล้มเหลว
- ถูกปฏิเสธ
- ทำผิดพลาดใหญ่
- สูญเสียทุกอย่าง
- ไม่มีทางออก

**Physical Cues:**
```
- ตาว่างเปล่า
- ไม่มีแรง
- นั่งทรุด
- ไม่ร้องไห้ (เจ็บเกินกว่าจะร้อง)
- เสียงแหบ
- หัวใจว่างเปล่า
```

## 2. เทคนิค Show Don't Tell

### ❌ Tell (บอกตรงๆ)
```
"เธอรู้สึกเศร้ามาก"
"เขาโกรธมาก"
"เธอรักเขา"
```

### ✅ Show (แสดงผ่านการกระทำ)
```
"น้ำตาไหลลงมาโดยไม่รู้ตัว เธอไม่แม้แต่จะยกมือขึ้นเช็ด"

"กำปั้นของเขาบีบแน่นจนข้อนิ้วขาวซีด
ขากรรไกรเกร็งจนกล้ามเนื้อสั่น"

"หัวใจเธอเต้นแรงขึ้นทุกครั้งที่เขาเดินผ่าน
แม้จะพยายามไม่มอง แต่สายตาก็แอบตามเขาไปเอง"
```

---

## 3. Internal Monologue (ความคิดในใจ)

### วิธีใช้
แสดงความคิด/ความรู้สึกภายในของตัวละคร

### ตัวอย่าง

**ความรัก:**
```
'ทำไมต้องเป็นเขา' เธอคิด 'ทำไมหัวใจต้องเต้นแรงขนาดนี้
แค่เขายิ้ม ทั้งที่รู้ว่าไม่ควร'
```

**ความเศร้า:**
```
'มันจบแล้วจริงๆ เหรอ' ความคิดนั้นวนเวียนซ้ำแล้วซ้ำเล่า
'เขาจะไม่กลับมาอีกแล้ว ไม่มีวันกลับมา'
```

**ความโกรธ:**
```
'ปล่อยให้มันรอดไม่ได้' เขาคิด เลือดในกายเดือดพล่าน
'ไม่ว่าจะต้องทำอะไรก็ตาม'
```

## 4. Physical Reactions ตามระดับอารมณ์

### ระดับ 1: อ่อน (Subtle)
```
- หลบตา
- กัดริมฝีปาก
- นิ้วสั่นเล็กน้อย
- กลืนน้ำลาย
```

### ระดับ 2: ปานกลาง (Moderate)
```
- หัวใจเต้นแรง
- แก้มแดง / หน้าซีด
- เสียงสั่น
- หายใจเร็ว
```

### ระดับ 3: เข้มข้น (Intense)
```
- น้ำตาไหล
- ตัวสั่นทั้งตัว
- ขาอ่อน/ทรุดลง
- หายใจไม่ออก
```

### ระดับ 4: สุดขีด (Overwhelming)
```
- ร้องไห้โฮ / กรีดร้อง
- ล้มทั้งยืน
- สลบ
- หมดสติ
```

## 5. ตัวอย่างฉากตามแนวนิยาย

### Romantic Comedy - ฉากจิ้น
```
"อยู่เฉยๆ" เขาพูด เสียงต่ำ

มินตราแข็งค้าง เมื่อมือใหญ่ยื่นมาปัดกลีบดอกไม้ที่ติดผมเธอ

ใกล้มาก... ใกล้เกินไป

เธอได้กลิ่นน้ำหอมของเขา ผสมกับกลิ่นกาแฟ
หัวใจเต้นรัวจนกลัวว่าเขาจะได้ยิน

"เสร็จแล้ว" เขาถอยออก "ระวังด้วย"

เขาเดินจากไป
แต่หัวใจเธอยังเต้นแรงอยู่อีกนานกว่าจะเป็นปกติ
```

### BL/GL - ฉากซึ้ง
```
"ทำไมต้องเป็นอย่างนี้" ภูมิกระซิบ

น้ำตาไหลลงมาโดยไม่รู้ตัว
เขาพยายามหยุดมัน แต่มันไม่ยอมหยุด

พี่เวฟเอื้อมมือมาเช็ดน้ำตาให้
สัมผัสนั้นอ่อนโยน เหมือนจะบอกว่าทุกอย่างจะดีเอง

"ผมไม่ไปไหน" พี่เวฟพูด เสียงเรียบแต่หนักแน่น
"ผมจะอยู่ตรงนี้ ไม่ว่าจะเกิดอะไรขึ้น"

ภูมิก้มหน้า ไหล่สั่น
แล้วพี่เวฟก็ดึงเขาเข้ามาในอ้อมกอด
กอดแน่น เหมือนกลัวว่าจะหายไป
```

### Drama - ฉากเผชิญหน้า
```
"เธอรู้มาตลอดใช่ไหม"

คำถามนั้นทำให้อรณีหยุดชะงัก
เธอหันมา เจอกับดวงตาของแม่ที่เต็มไปด้วยความเจ็บปวด

"แม่..."

"ตอบมา!" แม่ตะโกน น้ำเสียงแตกพร่า
"ลูกรู้ว่าพ่อมีครอบครัวอื่น รู้มาตลอด แต่ไม่เคยบอกแม่!"

อรณียืนนิ่ง
ความจริงที่เก็บไว้สิบปีกำลังแตกสลาย
พร้อมกับหัวใจของเธอ
```

## 6. Emotional Arc ในแต่ละฉาก

### โครงสร้าง
```
1. Setup (10-20%)
   - ตั้งฉาก สร้างบรรยากาศ
   - ให้ผู้อ่านรู้ว่าใครอยู่ที่ไหน

2. Build-up (30-40%)
   - ค่อยๆ สร้างอารมณ์
   - เพิ่ม tension ทีละนิด

3. Peak (20-30%)
   - จุดสูงสุดของอารมณ์
   - ปล่อยทุกอย่างออกมา

4. Resolution/Aftermath (10-20%)
   - หลังจากจุดพีค
   - แสดงผลกระทบ
```

## 9. Common Mistakes

### ❌ Too Much Tell
```
"เธอรู้สึกเศร้ามาก เสียใจมาก และโกรธมากด้วย"
```
**ปัญหา:** บอกอารมณ์ตรงๆ ไม่แสดงให้เห็น

### ❌ Over-dramatic
```
"น้ำตาไหลเป็นสายเหมือนน้ำตก หัวใจแตกสลายเป็นล้านชิ้น
เธอร้องไห้จนแทบหมดลมหายใจ กรีดร้องออกมาดังลั่น"
```
**ปัญหา:** เยอะเกินไป ไม่สมจริง

### ❌ Inconsistent Character
```
[ตัวละครที่กำหนดว่าเย็นชา เก็บอารมณ์]
"เขาร้องไห้โฮ กระโดดโลดเต้นดีใจ"
```
**ปัญหา:** ไม่ตรงกับบุคลิกที่กำหนด

### ❌ No Build-up
```
"เขาเห็นเธอ แล้วก็รักเลย"
```
**ปัญหา:** ไม่มีการ build อารมณ์

## 10. Tips ตามแนวนิยาย

### Romantic Comedy
- ฉากจิ้นควรมีความน่ารัก ไม่หนักเกินไป
- ใส่ความตลกขบขันปนอารมณ์ได้
- Chemistry ต้องชัดเจน

### BL/GL
- เน้น emotional depth
- ความสัมพันธ์ที่ค่อยๆ พัฒนา
- ฉากสองคนต้องมี tension

### Drama
- อารมณ์เข้มข้น แต่สมจริง
- ความขัดแย้งภายในตัวละคร
- ผลกระทบของการตัดสินใจ

### Horror
- ความกลัวที่ค่อยๆ เพิ่มขึ้น
- ใช้ประสาทสัมผัสสร้างบรรยากาศ
- ความเงียบก่อนความกลัว
