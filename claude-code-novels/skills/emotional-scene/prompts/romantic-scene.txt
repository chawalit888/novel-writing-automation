# Romantic Scene Prompt

## Context Variables
- {{genre}}: แนวนิยาย
- {{characters}}: ตัวละครที่เกี่ยวข้อง (ชื่อ, บุคลิก, ความสัมพันธ์ปัจจุบัน)
- {{location}}: สถานที่
- {{situation}}: สถานการณ์ก่อนหน้า
- {{romantic_stage}}: ระยะความสัมพันธ์ (แรกพบ, ตกหลุมรัก, สารภาพ, คืนดี)
- {{intensity}}: ความเข้มข้น (light/medium/intense)

---

## Prompt Template

คุณคือนักเขียนฉากโรแมนติกมืออาชีพ เชี่ยวชาญในการเขียนฉากจิ้นที่ทำให้ผู้อ่านหัวใจเต้นแรง

### ข้อมูลฉาก:
- แนว: {{genre}}
- ตัวละคร: {{characters}}
- สถานที่: {{location}}
- สถานการณ์: {{situation}}
- ระยะความสัมพันธ์: {{romantic_stage}}
- ความเข้มข้น: {{intensity}}

---

## คำสั่ง:

เขียนฉากโรแมนติกตามข้อมูลที่ให้ โดยใช้เทคนิคต่อไปนี้:

### 1. Physical Proximity (ความใกล้ชิด)
- การสัมผัสที่ไม่ได้ตั้งใจ
- ระยะห่างที่ใกล้กว่าปกติ
- สบตา

### 2. Internal Monologue
- แสดงความคิดของตัวละครที่กำลังรู้สึก
- ความสับสน/ตื่นเต้น

### 3. Physical Reactions
- หัวใจเต้นแรง
- แก้มร้อน
- หายใจไม่ทั่วท้อง
- มือสั่น

### 4. Sensory Details
- กลิ่น (น้ำหอม, กลิ่นตัว)
- สัมผัส (ผิวหนัง, เสื้อผ้า)
- เสียง (เสียงหัวใจ, เสียงหายใจ)

## ตัวอย่าง Romantic Scenes

### First Meeting Chemistry
```
เธอชนเข้ากับคนแปลกหน้า กาแฟหกเต็มเสื้อ
แต่แทนที่จะโกรธ เธอกลับแข็งค้าง
เมื่อเงยหน้าขึ้น สบตากับดวงตาสีเข้มที่จ้องมองเธอ
```

### Almost Kiss
```
ใบหน้าเขาอยู่ใกล้เกินไป ใกล้จนเห็นขนตาทุกเส้น
ลมหายใจอุ่นปะทะริมฝีปากเธอ
เธอหลับตาลง... แต่แล้ว...
```

### Confession
```
"ผมรู้ว่าไม่ควรพูด" เขาว่า เสียงสั่น
"แต่ผมทนไม่ไหวแล้ว"
เขาสบตาเธอ ในแววตานั้นมีทุกอย่าง
ความรัก ความกลัว และความหวัง
"ผมรักคุณ"
```

### Jealousy to Romance
```
"ทำไมต้องสนใจว่าฉันคุยกับใคร" เธอถาม
เขานิ่งไป มือกำแน่น
แล้วก็พูดออกมาด้วยเสียงต่ำ
"เพราะฉันหึง"
คำนั้นทำให้เธอหยุดหายใจ
```
