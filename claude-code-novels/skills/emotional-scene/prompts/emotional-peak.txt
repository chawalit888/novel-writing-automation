# Emotional Peak Prompt (จุดพีคอารมณ์)

## Context Variables
- {{genre}}: แนวนิยาย
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{situation}}: สถานการณ์ที่นำไปสู่จุดพีค
- {{peak_type}}: ประเภทจุดพีค (breakdown, realization, confrontation, release)
- {{emotion}}: อารมณ์หลัก (รัก, เศร้า, โกรธ, กลัว, ดีใจ)
- {{aftermath}}: สิ่งที่ควรเกิดหลังจุดพีค

---

## Prompt Template

คุณคือนักเขียนฉากจุดพีคอารมณ์มืออาชีพ เชี่ยวชาญในการเขียนฉากที่อารมณ์ระเบิดออกมาอย่างทรงพลัง

### ข้อมูลฉาก:
- แนว: {{genre}}
- ตัวละคร: {{characters}}
- สถานการณ์: {{situation}}
- ประเภทจุดพีค: {{peak_type}}
- อารมณ์หลัก: {{emotion}}
- หลังจุดพีค: {{aftermath}}

## ประเภท Emotional Peak

### 1. Breakdown (อารมณ์แตก)
ตัวละครที่เก็บอารมณ์มานาน ไม่ไหวแล้ว

```
เธอยิ้มมาตลอด บอกทุกคนว่าไม่เป็นไร
แต่วันนี้ เมื่อเขาถามว่า "เป็นไงบ้าง"
รอยยิ้มนั้นก็แตกสลาย

น้ำตาทะลักออกมา หยุดไม่ได้
เสียงร้องไห้ที่เก็บไว้หลายเดือนก็พรั่งพรูออกมา
เธอทรุดลงกับพื้น ไหล่สั่น

"ฉันไม่ไหวแล้ว" เธอกระซิบ ระหว่างเสียงสะอื้น
"ฉันไม่ไหวแล้วจริงๆ"
```

### 2. Realization (ตระหนักรู้)
ตัวละครรู้ความจริงบางอย่างที่เปลี่ยนทุกอย่าง

```
ภาพทั้งหมดประกอบเข้าด้วยกัน
ทุกคำพูด ทุกการกระทำ ทุกรอยยิ้ม

มันไม่เคยเป็นความบังเอิญ
ทุกอย่างถูกวางแผนมาตั้งแต่แรก

"เธอ..." เสียงเขาสั่น
"เธอรู้มาตั้งแต่แรกว่าฉันเป็นใคร"

เธอไม่ตอบ แต่ความเงียบนั้นบอกทุกอย่าง

โลกของเขาพังทลาย
```

### 3. Confrontation (เผชิญหน้า)
การปะทะทางอารมณ์ที่รุนแรง

```
"พอได้แล้ว!" เธอตะโกน เสียงดังสะท้อนทั้งห้อง

ทุกคนหยุดนิ่ง หันมามองเธอ

"ฉันทนฟังเธอพูดแบบนี้มาตลอดชีวิต" เธอพูด เสียงสั่นด้วยความโกรธ
"ว่าฉันไม่เก่งพอ ว่าฉันเป็นแค่ลูกหลานคนรับใช้
แต่วันนี้ฉันจะบอกเธอว่า..."

เธอก้าวเข้าไปใกล้ ตาจ้องมองไม่กระพริบ
"ฉันไม่กลัวเธออีกต่อไปแล้ว"
```

### 4. Release (ปลดปล่อย)
การปล่อยวางอารมณ์ที่กดทับไว้

```
เขายืนอยู่ริมหน้าผา ลมพัดแรง

ทุกความเจ็บปวด ทุกความผิดหวัง ทุกความโกรธ
มันหนักเกินไป แบกไว้นานเกินไป

แล้วเขาก็กรีดร้องออกมา
เสียงตะโกนที่ดังไปทั่วหุบเขา
ร้องจนหมดแรง จนคอแหบ

เมื่อเสียงหยุด มีแค่ลมและความเงียบ
แต่หัวใจเขาเบาลง
เบากว่าที่เคยเป็นมานาน
```

## โครงสร้าง Emotional Peak

```
┌─────────────────────────────────────────────────────────────┐
│  1. Pre-Peak Setup (10-15%)                                │
│     - สถานการณ์ที่นำไปสู่จุดพีค                             │
│     - Tension ที่ค่อยๆ เพิ่มขึ้น                           │
│                                                             │
│  2. Trigger (5-10%)                                        │
│     - สิ่งที่ทำให้อารมณ์ระเบิด                              │
│     - อาจเป็นคำพูด/การกระทำ/ความจริง                       │
│                                                             │
│  3. Peak (40-50%)                                          │
│     - อารมณ์ระเบิดออกมา                                    │
│     - Physical + Internal + External reactions             │
│     - นี่คือส่วนที่ต้องเขียนอย่างละเอียด                    │
│                                                             │
│  4. Aftermath (20-30%)                                     │
│     - หลังจากอารมณ์ผ่านไป                                  │
│     - ผลกระทบ/การเปลี่ยนแปลง                               │
│     - สภาพของตัวละคร                                       │
└─────────────────────────────────────────────────────────────┘
```

---

## Tips สำหรับ Emotional Peak

### DO:
- ให้เวลา build up อารมณ์
- ใช้ sensory details เพิ่มความสมจริง
- แสดง physical reactions ที่เหมาะสม
- มี aftermath ที่สมเหตุสมผล

### DON'T:
- พุ่งเข้าสู่จุดพีคทันที
- ใช้คำว่า "มาก" หรือ "ที่สุด" ซ้ำๆ
- เยอะเกินไปจนไม่สมจริง
- ลืมผลกระทบหลังจุดพีค

---

## Checklist

- [ ] มี Build-up ก่อนถึงจุดพีค
- [ ] มี Trigger ที่ชัดเจน
- [ ] Peak มีความเข้มข้นเหมาะสม
- [ ] มี Physical Reactions
- [ ] มี Internal Monologue (ถ้าเหมาะสม)
- [ ] มี Aftermath ที่สมเหตุสมผล
- [ ] เหมาะกับบุคลิกตัวละคร
- [ ] ไม่ Over-dramatic
