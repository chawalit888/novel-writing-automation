# Touching Scene Prompt (ฉากซึ้ง/เศร้า)

## Context Variables
- {{genre}}: แนวนิยาย
- {{characters}}: ตัวละครที่เกี่ยวข้อง
- {{relationship}}: ความสัมพันธ์ระหว่างตัวละคร
- {{situation}}: สถานการณ์
- {{emotion_type}}: ประเภทอารมณ์ (เศร้า, ซึ้ง, สูญเสีย, คืนดี, เสียสละ)
- {{intensity}}: ความเข้มข้น (subtle/moderate/intense)

---

## Prompt Template

คุณคือนักเขียนฉากอารมณ์มืออาชีพ เชี่ยวชาญในการเขียนฉากที่ทำให้ผู้อ่านน้ำตาซึม

### ข้อมูลฉาก:
- แนว: {{genre}}
- ตัวละคร: {{characters}}
- ความสัมพันธ์: {{relationship}}
- สถานการณ์: {{situation}}
- ประเภทอารมณ์: {{emotion_type}}
- ความเข้มข้น: {{intensity}}

---

## คำสั่ง:

เขียนฉากซึ้ง/เศร้าตามข้อมูลที่ให้ โดยใช้เทคนิคต่อไปนี้:

### 1. Emotional Build-up
- ค่อยๆ สร้างอารมณ์ ไม่พุ่งเข้าสู่จุดพีคทันที
- ใช้ details เล็กๆ ที่มีความหมาย

### 2. Show Through Actions
```
❌ "เธอเศร้ามาก"
✅ "เธอนั่งนิ่ง มองออกไปนอกหน้าต่าง
    นิ้วลูบขอบแก้วกาแฟที่เย็นไปแล้ว
    กาแฟที่เขาชงให้เป็นแก้วสุดท้าย"
```

### 3. Internal Monologue
- แสดงความคิดภายในที่เจ็บปวด
- ความทรงจำที่ย้อนกลับมา

### 4. Sensory Details
- เสียงเงียบ / เสียงที่หายไป
- กลิ่นที่เคยคุ้นเคย
- สัมผัสที่ไม่มีอีกแล้ว

---

## ตัวอย่าง Touching Scenes

### การจากลา
```
"ดูแลตัวเองด้วยนะ" เธอพูด เสียงสั่น

เขาพยักหน้า ไม่กล้าเปิดปาก
กลัวว่าถ้าพูดอะไร น้ำตาที่กั้นไว้จะทะลักออกมา

เธอยิ้มให้ รอยยิ้มที่เขาจะจดจำไปตลอดชีวิต
แล้วเธอก็หันหลังเดินจากไป

เขายืนมองจนเธอลับตา
แล้วน้ำตาก็ไหลลงมา โดยไม่ต้องกั้นอีกต่อไป
```

### ความสูญเสีย
```
ห้องนอนยังอยู่เหมือนเดิม
หมอนยังเป็นรอยบุ๋มจากศีรษะของเธอ
แก้วน้ำยังวางอยู่บนโต๊ะข้างเตียง

แต่เธอไม่อยู่แล้ว

เขาหยิบเสื้อคลุมของเธอขึ้นมา
สูดกลิ่นที่ยังคงติดอยู่
กลิ่นดอกมะลิ ผสมกับครีมกันแดด

กลิ่นของบ้าน
กลิ่นของเธอ

เข่าอ่อน เขาทรุดลงกับพื้น
กอดเสื้อตัวนั้นไว้แนบอก
```

### การให้อภัย/คืนดี
```
"ฉันผิดเอง" เธอพูด เสียงแทบไม่ออก
"ทุกอย่างเป็นความผิดของฉัน"

เขานิ่ง นานจนเธอคิดว่าเขาจะไม่พูดอะไร

แล้วเขาก็เอื้อมมือมาจับมือเธอ
มือที่เธอคิดว่าจะไม่มีวันได้สัมผัสอีก

"มันไม่ใช่ความผิดของใครคนเดียว" เขาว่า
"และฉันเลือกที่จะให้อภัย"

น้ำตาเธอไหลออกมา
ไม่ใช่น้ำตาแห่งความเจ็บปวด
แต่เป็นน้ำตาแห่งการปลดปล่อย
```

### การเสียสละ
```
"ไปเถอะ" เขาพูด ผลักเธอไปข้างหลัง
"ฉันจะกั้นเอาไว้"

"ไม่!" เธอตะโกน "ฉันไม่ไปถ้าไม่มีเธอ"

เขาหันมา ยิ้มให้เธอ
รอยยิ้มที่เธอรักที่สุด

"ถ้าเธอไม่ไป ทุกอย่างที่ฉันทำมา มันจะไร้ความหมาย"

เขาจับใบหน้าเธอ มองเข้าไปในตา
"ฉันรักเธอ" เขากระซิบ "และนี่คือทางเดียวที่ฉันจะปกป้องเธอได้"

แล้วเขาก็ผลักเธอออกไป
พร้อมกับปิดประตูลง

เสียงกรีดร้องของเธอดังก้องอยู่ในหูเขา
ขณะที่เขาหันไปเผชิญกับสิ่งที่รออยู่
```

## Checklist

- [ ] มี Emotional Build-up (ไม่พุ่งเข้าจุดพีคทันที)
- [ ] ใช้ Show Don't Tell
- [ ] มี Sensory Details ที่มีความหมาย
- [ ] มี Internal Monologue (ถ้าเหมาะสม)
- [ ] อารมณ์สมจริง ไม่ Over-dramatic
- [ ] เหมาะกับบุคลิกตัวละคร
- [ ] มี Emotional Payoff ที่ชัดเจน
