# Historical Research Skill

## Overview
Skill สำหรับ research และตรวจสอบความถูกต้องทางประวัติศาสตร์ในนิยาย
เพื่อให้เนื้อเรื่องมีความสมจริงและไม่ผิดยุคสมัย

## Usage
```
"research ยุคสมัยรัตนโกสินทร์ตอนต้นสำหรับนิยาย"
"ตรวจความถูกต้องทางประวัติศาสตร์ของบทที่ 3"
"หาข้อมูลวัฒนธรรม/ขนบธรรมเนียมยุค [ชื่อยุค]"
"สร้าง historical reference sheet สำหรับ [ชื่อเรื่อง]"
```

## หมวดที่ต้อง Research

### 1. บริบทยุคสมัย (Period Context)
| หัวข้อ | รายละเอียดที่ต้องหา |
|--------|-------------------|
| ช่วงเวลา | ปี พ.ศ./ค.ศ. ที่แน่นอน, รัชกาลที่เท่าไหร่ |
| การเมือง | ระบบการปกครอง, ผู้มีอำนาจ, กฎหมายสำคัญ |
| เศรษฐกิจ | สกุลเงิน, การค้า, อาชีพหลัก, ราคาสินค้า |
| สังคม | ชนชั้น, ค่านิยม, บทบาทหญิง-ชาย |
| ศาสนา | ความเชื่อ, พิธีกรรม, วัดวาอาราม |

### 2. ชีวิตประจำวัน (Daily Life)
| หัวข้อ | รายละเอียดที่ต้องหา |
|--------|-------------------|
| อาหาร | เมนูที่มีในยุคนั้น, วัตถุดิบ, วิธีปรุง |
| เสื้อผ้า | เครื่องแต่งกายตามชนชั้น/โอกาส |
| ที่อยู่อาศัย | สถาปัตยกรรม, ห้อง, เฟอร์นิเจอร์ |
| การเดินทาง | ยานพาหนะ, ระยะเวลา, เส้นทาง |
| สุขอนามัย | การแพทย์, สมุนไพร, โรคภัย |
| บันเทิง | ดนตรี, การแสดง, กีฬา, งานเทศกาล |

### 3. ภาษาและการสื่อสาร (Language & Communication)
| หัวข้อ | รายละเอียดที่ต้องหา |
|--------|-------------------|
| คำเรียกขาน | ระดับภาษา (ราชาศัพท์, สุภาพ, ปกติ) |
| สรรพนาม | คำเรียกตัวเอง/ผู้อื่นตามชนชั้น |
| สำนวน | คำพูดที่ใช้ในยุคนั้น (ไม่ใช้คำสมัยใหม่) |
| การเขียน | เครื่องมือเขียน, ภาษาทางการ |

### 4. ตระกูลและครอบครัว (Family & Lineage)
| หัวข้อ | รายละเอียดที่ต้องหา |
|--------|-------------------|
| โครงสร้างตระกูล | ลำดับชั้น, การสืบทอด, อำนาจ |
| การแต่งงาน | พิธี, สินสอด, ธรรมเนียม, ข้อห้าม |
| มรดก | กฎหมายมรดก, สิทธิ์ลูกชาย/ลูกสาว |
| นามสกุล | ต้นกำเนิด, ความหมาย (พ.ร.บ.นามสกุล พ.ศ. 2456) |

## Workflow การ Research

### Step 1: กำหนดขอบเขต
```
1. ระบุยุคสมัยที่แน่นอน (ปี/รัชกาล)
2. ระบุสถานที่ (กรุงเทพ/ต่างจังหวัด/ต่างประเทศ)
3. ระบุชนชั้นของตัวละคร (ขุนนาง/สามัญ/ค้าขาย)
4. ระบุหัวข้อที่ต้อง research (เครื่องแต่งกาย/อาหาร/ภาษา/ฯลฯ)
```

### Step 2: รวบรวมข้อมูล
```
1. ใช้ Brave Search MCP ค้นหาข้อมูลจาก:
   - Wikipedia ไทย/อังกฤษ
   - ฐานข้อมูลประวัติศาสตร์ (ศิลปากร, หอจดหมายเหตุ)
   - บทความวิชาการ
   - หนังสือประวัติศาสตร์ (ถ้าหาได้ออนไลน์)
2. Cross-reference อย่างน้อย 2-3 แหล่ง
3. จดบันทึกแหล่งที่มา
```

### Step 3: สร้าง Historical Reference Sheet
```
บันทึกที่: knowledge-base/historical/[ชื่อยุค]-[หัวข้อ].md

รูปแบบ:
# [ยุคสมัย] - [หัวข้อ]
## แหล่งอ้างอิง
## ข้อเท็จจริงที่ยืนยันแล้ว
## สิ่งที่ยังไม่แน่ใจ (ต้องตรวจสอบเพิ่ม)
## ข้อควรระวังสำหรับนิยาย
```

### Step 4: ตรวจสอบเนื้อเรื่อง
```
1. อ่านบทนิยาย
2. ตรวจทุก element ที่เกี่ยวกับยุคสมัย
3. จัดทำ Historical Accuracy Report
```

## ยุคสมัยสำคัญในนิยายไทย

### สมัยรัตนโกสินทร์ตอนต้น (พ.ศ. 2325-2394)
```
รัชกาลที่ 1-3
- สังคมศักดินา, ระบบไพร่-ทาส
- เครื่องแต่งกาย: ห่มสไบ, นุ่งโจงกระเบน
- คำเรียกขาน: ข้า, เจ้าคุณ, ท่าน, หม่อม
- การค้า: ผูกขาดโดยพระคลัง, ค้ากับจีน
- ที่อยู่: เรือนไทย, คฤหาสน์, ตึกแบบจีน
- การเดินทาง: เรือ (หลัก), ช้าง, ม้า, เกวียน
```

### สมัยรัชกาลที่ 4-5 (พ.ศ. 2394-2453)
```
ยุคปฏิรูป/เปิดประเทศ
- ผสมตะวันตก: เริ่มมีถนน, รถราง, โทรเลข
- เครื่องแต่งกาย: เริ่มผสมตะวันตก
- ภาษา: เริ่มมีคำยืมอังกฤษ
- การศึกษา: เริ่มมีโรงเรียนสมัยใหม่
- เลิกทาส พ.ศ. 2448
```

### สมัยรัชกาลที่ 6-7 (พ.ศ. 2453-2475)
```
ก่อนเปลี่ยนแปลงการปกครอง
- มี พ.ร.บ.นามสกุล
- เริ่มมีรถยนต์, ไฟฟ้า
- สังคมกำลังเปลี่ยนผ่าน
- ชนชั้นกลางเริ่มเติบโต
```

### ยุคหลังเปลี่ยนแปลงการปกครอง (พ.ศ. 2475+)
```
สมัยใหม่
- ระบอบประชาธิปไตย
- สงครามโลกครั้งที่ 2 (พ.ศ. 2484-2488)
- การพัฒนาเศรษฐกิจ
```

## คำที่ต้องระวัง (Anachronism Alert)

### ห้ามใช้ในยุคก่อนสมัยที่ระบุ
| คำ/สิ่งของ | มีตั้งแต่เมื่อ | หมายเหตุ |
|-----------|-------------|---------|
| นามสกุล | พ.ศ. 2456 (ร.6) | ก่อนนี้ไม่มีนามสกุล |
| โทรศัพท์ | พ.ศ. 2424 (ร.5) | เฉพาะในวัง/ราชการ |
| รถยนต์ | พ.ศ. 2445 (ร.5) | คันแรกในสยาม |
| ไฟฟ้า | พ.ศ. 2427 (ร.5) | เฉพาะพระราชวัง |
| รถไฟ | พ.ศ. 2439 (ร.5) | สายแรก กรุงเทพ-อยุธยา |
| โทรเลข | พ.ศ. 2418 (ร.5) | สายแรก กรุงเทพ-สมุทรปราการ |
| หนังสือพิมพ์ | พ.ศ. 2387 (ร.3) | Bangkok Recorder |
| ธนบัตร | พ.ศ. 2445 (ร.5) | ก่อนนี้ใช้เงินพดด้วง/เหรียญ |
| ถนนลาดยาง | พ.ศ. 2406 (ร.4) | ถนนเจริญกรุง |
| โอเค/OK | สมัยใหม่ | ห้ามใช้ในยุคเก่า |
| มือถือ/SMS | พ.ศ. 2529+ | ห้ามใช้ในยุคก่อนนั้น |

### คำเรียกขานตามยุค
```
ยุคเก่า (ก่อน ร.5):
- ข้า/ข้าพเจ้า (พูดกับผู้ใหญ่)
- กระผม/ดิฉัน (สุภาพ)
- กู/มึง (สนิท/ดูถูก)
- เจ้าคุณ, ท่าน, คุณหญิง, คุณนาย

ยุคกลาง (ร.5-ร.7):
- ผม/ดิฉัน เริ่มใช้มากขึ้น
- คุณ + ชื่อ เริ่มนิยม

ยุคปัจจุบัน:
- ผม/หนู/เรา ตามสถานการณ์
- พี่/น้อง ตามอายุ
```

## Historical Accuracy Checklist

### ก่อนเขียน
- [ ] ระบุปี/รัชกาลที่แน่นอนแล้ว
- [ ] Research เครื่องแต่งกายของยุคนั้นแล้ว
- [ ] Research อาหาร/ของใช้ของยุคนั้นแล้ว
- [ ] Research ภาษา/คำเรียกขานแล้ว
- [ ] สร้าง Historical Reference Sheet แล้ว

### ขณะเขียน
- [ ] ไม่มี anachronism (ของที่ยังไม่มีในยุคนั้น)
- [ ] ภาษาสอดคล้องกับยุคสมัย
- [ ] เครื่องแต่งกายถูกต้อง
- [ ] อาหาร/สิ่งของถูกยุค
- [ ] ระบบสังคม/ชนชั้นถูกต้อง
- [ ] การเดินทาง/ระยะเวลาสมเหตุสมผล

### หลังเขียน
- [ ] ตรวจ anachronism ทั้งหมด
- [ ] ตรวจคำพูดที่ไม่เข้ายุค
- [ ] ตรวจเทคโนโลยีที่อ้างถึง
- [ ] สร้าง Historical Accuracy Report
